## 📌 关注领域

### AI与大模型
- MCP协议生态进展（Server/Client工具链完善）
- Agent框架演进（LangChain4J、Spring AI、Agent-Flex）
- LLM推理优化（量化、蒸馏、推理引擎）
- 向量数据库与RAG架构实践

### 云原生与容器
- Kubernetes版本更新与新特性
- Service Mesh（Istio、Linkerd）演进
- Serverless与FaaS平台
- eBPF与可观测性

### 微服务与分布式
- Spring Cloud Alibaba版本迭代
- Dubbo 3.x生态演进
- 服务网格与API网关
- 分布式事务方案（Seata、Saga）

### 数据库与存储
- MySQL版本更新与性能优化
- Redis新特性与集群方案
- PostgreSQL生态进展
- 向量数据库（Milvus、Pinecone、Weaviate）

### Java生态
- JDK新特性（LTS版本、Project Loom、Project Panama）
- 框架升级（Spring Boot/Cloud、MyBatis、Quarkus）
- GraalVM原生镜像进展
- 反应式编程生态

### 安全与DevOps
- 供应链安全（SBOM、Sigstore）
- GitOps与平台工程
- 可观测性标准（OpenTelemetry）
- 零信任架构
