## 📊 数据库与存储

### PostgreSQL 18正式发布

| 特性 | 说明 |
|------|------|
| 增量物化视图 | 数据变更时自动增量刷新，无需全量重建 |
| 异步IO | 支持`io_uring`，查询性能提升30% |
| 逻辑复制增强 | 支持DDL操作复制（CREATE/ALTER/DROP） |
| SQL/JSON增强 | JSON路径表达式性能优化 |
| 统计信息改进 | 扩展统计支持多列相关性 |

```sql
-- PostgreSQL 18 增量物化视图
CREATE MATERIALIZED VIEW mv_order_stats
REFRESH FAST ON COMMIT
AS 
SELECT user_id, COUNT(*) AS order_count, SUM(amount) AS total
FROM orders
GROUP BY user_id;

-- 数据变更时自动增量刷新，无需 REFRESH MATERIALIZED VIEW
```

```java
// Spring Boot 4.0 RC1 虚拟线程 + HTTP Interface
@HttpClientInterface
public interface OrderApiClient {
    
    @GetExchange("/orders/{id}")
    Order getOrder(@PathVariable Long id);
    
    @GetExchange(value = "/orders/stream", accept = "text/event-stream")
    Flux<Order> streamOrders();  // SSE Streaming
}

// 配置（虚拟线程自动生效）
@Configuration
public class AppConfig {
    @Bean
    RestClient restClient(RestClient.Builder builder) {
        return builder.build();
    }
}
```

