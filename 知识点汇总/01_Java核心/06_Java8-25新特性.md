## 2. Java 8核心特性

### Lambda表达式

```java
// 旧写法
Runnable r = new Runnable() {
    @Override
    public void run() {
        System.out.println("Hello");
    }
};

// Lambda
Runnable r = () -> System.out.println("Hello");

// 集合排序
List<String> names = Arrays.asList("Charlie", "Alice", "Bob");
names.sort((a, b) -> a.compareTo(b));
names.sort(String::compareTo);  // 方法引用
```

### Stream API

```java
List<User> users = ...;

// 过滤+映射+收集
List<String> adultNames = users.stream()
    .filter(u -> u.getAge() >= 18)
    .map(User::getName)
    .sorted()
    .collect(Collectors.toList());

// 分组
Map<String, List<User>> byCity = users.stream()
    .collect(Collectors.groupingBy(User::getCity));

// 统计
IntSummaryStatistics stats = users.stream()
    .mapToInt(User::getAge)
    .summaryStatistics();
// stats.getCount(), getAverage(), getMax(), getMin(), getSum()

// 并行流
long count = users.parallelStream()
    .filter(u -> u.getAge() > 30)
    .count();
```

### Optional

```java
// 避免NPE
Optional<User> user = Optional.ofNullable(userMapper.findById(id));

String name = user
    .map(User::getName)
    .filter(n -> n.length() > 0)
    .orElse("Unknown");

// 链式调用
String city = user
    .flatMap(u -> Optional.ofNullable(u.getAddress()))
    .map(Address::getCity)
    .orElseThrow(() -> new RuntimeException("地址不存在"));
```

### 新日期API

```java
// LocalDate/LocalTime/LocalDateTime
LocalDate date = LocalDate.of(2026, 7, 5);
LocalTime time = LocalTime.of(10, 30, 0);
LocalDateTime dateTime = LocalDateTime.of(date, time);

// 格式化
String formatted = dateTime.format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));

// 计算
LocalDate nextWeek = date.plus(1, ChronoUnit.WEEKS);
Duration duration = Duration.between(time, LocalTime.now());

// 时区
ZonedDateTime utc = ZonedDateTime.now(ZoneId.of("UTC"));
ZonedDateTime shanghai = utc.withZoneSameInstant(ZoneId.of("Asia/Shanghai"));
```

## 3. Java 11 LTS核心特性

### var局部变量

```java
// Java 10: var局部变量
var users = new ArrayList<User>();  // 推断为ArrayList<User>
var stream = users.stream();        // 推断为Stream<User>

// Java 11: var可用于Lambda参数
(@Nonnull var x, var y) -> x + y;
```

### HTTP Client

```java
// 同步请求
HttpClient client = HttpClient.newBuilder()
    .version(HttpClient.Version.HTTP_2)
    .connectTimeout(Duration.ofSeconds(10))
    .build();

HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/users"))
    .header("Accept", "application/json")
    .GET()
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.statusCode());
System.out.println(response.body());

// 异步请求
client.sendAsync(request, HttpResponse.BodyHandlers.ofString())
    .thenApply(HttpResponse::body)
    .thenAccept(System.out::println);
```

### String增强

```java
// Java 11 String新方法
"  hello  ".strip();        // "hello" (Unicode-aware trim)
"hello\nworld".lines();     // Stream: "hello", "world"
"abc".repeat(3);            // "abcabcabc"
"".isBlank();               // true
```

## 4. Java 17 LTS核心特性

### Sealed Class（密封类）

```java
// 密封类：限制哪些类可以继承
public sealed class Shape 
    permits Circle, Square, Triangle {}

final class Circle extends Shape { double radius; }
final class Square extends Shape { double side; }
non-sealed class Triangle extends Shape {}  // Triangle的子类不受限
```

### Pattern Matching for instanceof

```java
// 旧写法
if (obj instanceof String) {
    String s = (String) obj;
    System.out.println(s.length());
}

// Java 16+
if (obj instanceof String s) {
    System.out.println(s.length());  // 直接使用s，无需强转
}

// 结合条件
if (obj instanceof String s && s.length() > 5) {
    System.out.println("长字符串: " + s);
}
```

### Record（记录类）

```java
// 旧写法：POJO需要大量样板代码
public class Point {
    private final int x;
    private final int y;
    public Point(int x, int y) { this.x = x; this.y = y; }
    public int getX() { return x; }
    public int getY() { return y; }
    // equals, hashCode, toString...
}

// Java 14+ Record
public record Point(int x, int y) {}

// 自动生成：构造器、getter、equals、hashCode、toString
Point p = new Point(1, 2);
p.x();  // 1
p.y();  // 2
```

### Switch表达式

```java
// 旧写法
String result;
switch (day) {
    case MONDAY:
    case FRIDAY:
        result = "工作日";
        break;
    case SATURDAY:
    case SUNDAY:
        result = "周末";
        break;
    default:
        result = "未知";
}

// Java 14+ Switch表达式
String result = switch (day) {
    case MONDAY, FRIDAY -> "工作日";
    case SATURDAY, SUNDAY -> "周末";
    default -> "未知";
};

// 带yield的复杂逻辑
int numLetters = switch (day) {
    case MONDAY, FRIDAY, SUNDAY -> 6;
    case TUESDAY -> 7;
    default -> {
        String s = day.toString();
        yield s.length();
    }
};
```

## 5. Java 21 LTS核心特性

### 虚拟线程（Virtual Thread）

```java
// 传统线程（平台线程）：1MB栈空间，几千个上限
Thread thread = new Thread(() -> {
    // 阻塞操作会占用OS线程
    var data = httpClient.get("https://api.example.com/data");
    process(data);
});

// 虚拟线程：几KB，百万级
Thread vThread = Thread.ofVirtual().start(() -> {
    // 阻塞时自动让出OS线程，不浪费资源
    var data = httpClient.get("https://api.example.com/data");
    process(data);
});

// Executors批量创建
try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
    List<Future<String>> futures = IntStream.range(0, 10000)
        .mapToObj(i -> executor.submit(() -> fetchData(i)))
        .toList();
    
    List<String> results = futures.stream()
        .map(f -> {
            try { return f.get(); }
            catch (Exception e) { return null; }
        })
        .toList();
}
// 1万个并发HTTP请求，用虚拟线程轻松完成
```

### Pattern Matching for Switch

```java
// Java 21: Switch模式匹配
String format(Object obj) {
    return switch (obj) {
        case null -> "null";
        case Integer i when i > 0 -> "正整数: " + i;
        case Integer i -> "非正整数: " + i;
        case String s when s.length() > 10 -> "长字符串";
        case String s -> "短字符串: " + s;
        case int[] arr -> "数组长度: " + arr.length;
        case Point(int x, int y) -> "点(" + x + "," + y + ")";  // Record解构
        default -> obj.toString();
    };
}
```

### 结构化并发（Preview → 25正式版）

```java
// Java 21 Preview / Java 25 正式版
try (var scope = StructuredTaskScope.open()) {
    // 并行获取用户和订单
    StructuredTaskScope.Subtask<User> userTask = 
        scope.fork(() -> findUser(userId));
    StructuredTaskScope.Subtask<Order> orderTask = 
        scope.fork(() -> findOrder(userId));
    
    scope.join();  // 等待所有任务完成
    
    // 任意一个失败则整体失败
    scope.throwIfFailed();
    
    return new UserProfile(userTask.get(), orderTask.get());
}
```

## 6. Java 25核心特性

### 值类型（Value Types）Preview

```java
// 值类型：像int一样轻量，但可以有字段和方法
value class Point implements Shape {
    int x;
    int y;
    
    public double distance(Point other) {
        return Math.sqrt(Math.pow(x - other.x, 2) + Math.pow(y - other.y, 2));
    }
}

// 值类型特点：
// - 不可变
// - 无标识（两个相同值的Point相等，类似int）
// - 栈上分配（不进堆，无GC压力）
// - 内联扁平化（数组中连续存储，无指针）

// 对比
Point[] points = new Point[1000];
// 传统对象：1000个堆指针 → 缓存不友好
// 值类型：1000个Point内联在数组中 → 缓存友好
```

### 字符串模板正式版

```java
// Java 25: 字符串模板
String name = "World";
int x = 10, y = 20;

// STR处理器（内置）
String msg = STR."Hello \{name}!";              // "Hello World!"
String sum = STR."\{x} + \{y} = \{x + y}";     // "10 + 20 = 30"

// 多行模板
String json = STR."""
    {
        "name": "\{name}",
        "sum": \{x + y}
    }
    """;

// 自定义模板处理器
String FMT = StringTemplate.Processor.of(
    (StringTemplate st) -> String.format(st.interpolate())
);
```

### 作用域值（Scope Values）正式版

```java
// 替代ThreadLocal的现代方案
private static final ScopedValue<User> CURRENT_USER = ScopedValue.newInstance();

// 设置作用域值
ScopedValue.where(CURRENT_USER, user).run(() -> {
    // 在此作用域内可访问CURRENT_USER
    processOrder();
});

void processOrder() {
    User user = CURRENT_USER.get();  // 获取当前用户
    // ...
}

// 优势（vs ThreadLocal）：
// - 不可变（更安全）
// - 有界（作用域结束后自动清理）
// - 虚拟线程友好（不inherit到所有线程）
// - 无内存泄漏
```

## 7. 面试要点

### Q1: Java 8最重要的特性是什么？

```
1. Lambda表达式：函数式编程，简化匿名内部类
2. Stream API：声明式集合操作（filter/map/reduce）
3. Optional：空值安全处理
4. 新日期API（java.time）：不可变、线程安全、清晰API
5. 接口默认方法：接口可以有实现
```

### Q2: 虚拟线程和平台线程的区别？

```
平台线程（传统Thread）：
  - 1:1映射OS线程
  - 栈空间1MB
  - 创建/切换成本高
  - 上限几千个

虚拟线程（JDK 21）：
  - N:1映射OS线程（多路复用）
  - 栈空间几KB
  - 创建/切换成本极低
  - 上限百万级
  - 阻塞时不占用OS线程

适用：IO密集型场景（HTTP调用、数据库查询、文件读写）
不适用：CPU密集型（虚拟线程没有优势）
```

### Q3: Record和普通POJO的区别？

```
Record：
  - 不可变（字段final）
  - 自动生成构造器、getter、equals、hashCode、toString
  - 不能继承其他类（但可实现接口）
  - 适合DTO、值对象

POJO：
  - 可变
  - 需要手写或Lombok生成
  - 可继承
  - 适合实体类、可变状态
```

### Q4: Java 25的值类型有什么优势？

```
1. 性能：栈上分配，无GC压力
2. 缓存友好：内联扁平化，数组中连续存储
3. 语义清晰：值语义（两个相同值的对象相等）
4. 内存节省：无对象头、无指针

适用：高频创建的小对象（Point、Complex、Money等）
```

### Q5: ScopedValue和ThreadLocal的区别？

```
ThreadLocal：
  - 可变（set/get）
  - 子线程可继承（InheritableThreadLocal）
  - 容易内存泄漏
  - 虚拟线程下有性能问题

ScopedValue：
  - 不可变（设置后不可修改）
  - 有界（作用域结束自动清理）
  - 虚拟线程友好
  - 无内存泄漏
  - 线程安全（不可变）
```
