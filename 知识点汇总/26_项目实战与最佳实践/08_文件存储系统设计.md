## 1. 文件存储概述

### 1.1 核心需求

| 需求 | 描述 | 技术方案 |
|------|------|----------|
| 大文件上传 | 支持GB级文件上传 | 分片上传 |
| 断点续传 | 网络中断后可继续上传 | 分片状态记录 |
| 秒传 | 相同文件不重复上传 | MD5去重 |
| 高性能 | 上传下载快速 | CDN加速 |
| 安全性 | 防盗链、防篡改 | 签名URL |
| 可扩展 | 支持海量文件 | 对象存储 |

### 1.2 存储方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 本地磁盘 | 简单、快 | 不可扩展、单点 | 小规模应用 |
| NFS共享存储 | 多机共享 | 性能瓶颈 | 中小规模 |
| FastDFS | 分布式文件系统 | 维护复杂、社区不活跃 | 中等规模 |
| MinIO | S3兼容、轻量 | 需要自建 | 私有化部署 |
| 云OSS | 免运维、高可用 | 费用、厂商锁定 | 云原生应用 |

## 2. 分片上传设计

### 2.1 分片上传流程

```
分片上传完整流程:

1. 客户端: 计算文件MD5 → 请求初始化上传
2. 服务端: 检查MD5是否已存在(秒传) → 创建上传任务 → 返回uploadId
3. 客户端: 将文件切片(如每片5MB) → 并行上传分片
4. 服务端: 接收分片 → 存储到临时目录 → 记录分片状态
5. 客户端: 所有分片上传完成 → 请求合并
6. 服务端: 合并所有分片 → 校验完整性 → 移动到最终存储 → 返回文件URL
```

### 2.3 后端实现

```java
/**
 * 文件上传服务
 */
@Service
public class FileUploadService {

    @Autowired
    private FileUploadTaskMapper taskMapper;

    @Autowired
    private FileMetadataMapper metadataMapper;

    @Autowired
    private MinioClient minioClient;

    @Value("${file.chunk-size:5242880}") // 5MB
    private int chunkSize;

    @Value("${file.temp-dir:/data/file-temp}")
    private String tempDir;

    /**
     * 1. 初始化上传
     */
    public InitUploadResponse initUpload(InitUploadRequest request) {
        // 1. 检查秒传: 文件MD5是否已存在
        FileMetadata existing = metadataMapper.selectByMd5(request.getFileMd5());
        if (existing != null && existing.getStatus() == 1) {
            // 秒传成功
            InitUploadResponse response = new InitUploadResponse();
            response.setInstantUpload(true);
            response.setFileId(existing.getFileId());
            response.setUrl(existing.getCdnUrl());
            return response;
        }

        // 2. 检查是否有未完成的上传任务(断点续传)
        FileUploadTask existTask = taskMapper.selectByMd5AndStatus(
            request.getFileMd5(), 0);
        if (existTask != null) {
            // 返回已上传分片列表
            InitUploadResponse response = new InitUploadResponse();
            response.setInstantUpload(false);
            response.setUploadId(existTask.getUploadId());
            response.setUploadedChunks(parseChunkList(existTask.getUploadedChunks()));
            return response;
        }

        // 3. 创建新上传任务
        int chunkCount = (int) Math.ceil((double) request.getFileSize() / chunkSize);
        String uploadId = UUID.randomUUID().toString().replace("-", "");

        FileUploadTask task = new FileUploadTask();
        task.setUploadId(uploadId);
        task.setFileName(request.getFileName());
        task.setFileSize(request.getFileSize());
        task.setFileMd5(request.getFileMd5());
        task.setChunkSize(chunkSize);
        task.setChunkCount(chunkCount);
        task.setUploadedChunks("");
        task.setStatus(0);
        task.setUserId(SecurityUtils.getCurrentUserId());
        task.setBucketName("uploads");
        task.setObjectKey(uploadId + "/" + request.getFileName());
        task.setExpireTime(LocalDateTime.now().plusHours(24));
        taskMapper.insert(task);

        // 4. 创建临时目录
        new File(tempDir + "/" + uploadId).mkdirs();

        InitUploadResponse response = new InitUploadResponse();
        response.setInstantUpload(false);
        response.setUploadId(uploadId);
        response.setChunkSize(chunkSize);
        response.setChunkCount(chunkCount);
        response.setUploadedChunks(Collections.emptyList());
        return response;
    }

    /**
     * 2. 上传分片
     */
    @Transactional(rollbackFor = Exception.class)
    public ChunkUploadResponse uploadChunk(String uploadId, int chunkIndex, MultipartFile chunk) {
        FileUploadTask task = taskMapper.selectByUploadId(uploadId);
        if (task == null || task.getStatus() != 0) {
            throw new BizException("上传任务不存在或已完成");
        }

        // 检查分片是否已上传(幂等)
        Set<Integer> uploaded = parseChunkList(task.getUploadedChunks());
        if (uploaded.contains(chunkIndex)) {
            ChunkUploadResponse response = new ChunkUploadResponse();
            response.setUploaded(true);
            response.setChunkIndex(chunkIndex);
            return response;
        }

        // 保存分片到临时目录
        String chunkPath = tempDir + "/" + uploadId + "/" + chunkIndex;
        try {
            chunk.transferTo(new File(chunkPath));
        } catch (IOException e) {
            throw new BizException("分片保存失败: " + e.getMessage());
        }

        // 更新已上传分片列表
        uploaded.add(chunkIndex);
        String chunkListStr = uploaded.stream()
            .sorted()
            .map(String::valueOf)
            .collect(Collectors.joining(","));
        task.setUploadedChunks(chunkListStr);
        taskMapper.updateById(task);

        ChunkUploadResponse response = new ChunkUploadResponse();
        response.setUploaded(true);
        response.setChunkIndex(chunkIndex);
        response.setUploadedCount(uploaded.size());
        response.setTotalCount(task.getChunkCount());
        response.setAllCompleted(uploaded.size() == task.getChunkCount());
        return response;
    }

    /**
     * 3. 合并分片
     */
    @Transactional(rollbackFor = Exception.class)
    public FileMergeResponse mergeChunks(String uploadId) {
        FileUploadTask task = taskMapper.selectByUploadId(uploadId);
        if (task == null) {
            throw new BizException("上传任务不存在");
        }

        Set<Integer> uploaded = parseChunkList(task.getUploadedChunks());
        if (uploaded.size() != task.getChunkCount()) {
            throw new BizException("分片未上传完整");
        }

        // 1. 合并文件
        String mergedPath = tempDir + "/" + uploadId + "/merged";
        try (RandomAccessFile mergedFile = new RandomAccessFile(mergedPath, "rw")) {
            for (int i = 0; i < task.getChunkCount(); i++) {
                String chunkPath = tempDir + "/" + uploadId + "/" + i;
                byte[] chunkData = Files.readAllBytes(Paths.get(chunkPath));
                mergedFile.write(chunkData);
            }
        } catch (IOException e) {
            throw new BizException("文件合并失败: " + e.getMessage());
        }

        // 2. 校验MD5
        String actualMd5 = calculateMD5(mergedPath);
        if (!actualMd5.equals(task.getFileMd5())) {
            throw new BizException("文件校验失败: MD5不匹配");
        }

        // 3. 上传到对象存储
        String fileId = UUID.randomUUID().toString().replace("-", "");
        String objectKey = buildObjectKey(fileId, task.getFileName());
        try {
            minioClient.uploadObject(
                UploadObjectArgs.builder()
                    .bucket(task.getBucketName())
                    .object(objectKey)
                    .filename(mergedPath)
                    .contentType(getMimeType(task.getFileName()))
                    .build()
            );
        } catch (Exception e) {
            throw new BizException("对象存储上传失败: " + e.getMessage());
        }

        // 4. 保存文件元数据
        FileMetadata metadata = new FileMetadata();
        metadata.setFileId(fileId);
        metadata.setFileName(task.getFileName());
        metadata.setFileSize(task.getFileSize());
        metadata.setFileMd5(task.getFileMd5());
        metadata.setFileType(getFileType(task.getFileName()));
        metadata.setFileExt(getFileExt(task.getFileName()));
        metadata.setMimeType(getMimeType(task.getFileName()));
        metadata.setStorageType("minio");
        metadata.setBucketName(task.getBucketName());
        metadata.setObjectKey(objectKey);
        metadata.setCdnUrl(buildCdnUrl(objectKey));
        metadataMapper.insert(metadata);

        // 5. 更新任务状态
        task.setStatus(1);
        taskMapper.updateById(task);

        // 6. 清理临时文件
        cleanupTempFiles(uploadId);

        FileMergeResponse response = new FileMergeResponse();
        response.setFileId(fileId);
        response.setUrl(metadata.getCdnUrl());
        return response;
    }

    /**
     * 计算文件MD5
     */
    private String calculateMD5(String filePath) {
        try {
            return DigestUtils.md5Hex(new FileInputStream(filePath));
        } catch (IOException e) {
            throw new RuntimeException("MD5计算失败", e);
        }
    }

    /**
     * 清理临时文件
     */
    private void cleanupTempFiles(String uploadId) {
        try {
            FileUtils.deleteDirectory(new File(tempDir + "/" + uploadId));
        } catch (IOException e) {
            log.warn("临时文件清理失败: {}", uploadId, e);
        }
    }
}
```

## 3. 断点续传实现

### 3.1 前端实现

```javascript
class FileUploader {
    constructor(file, options = {}) {
        this.file = file;
        this.chunkSize = options.chunkSize || 5 * 1024 * 1024; // 5MB
        this.chunks = [];
        this.uploadId = null;
        this.uploadedChunks = new Set();
        this.maxConcurrent = options.maxConcurrent || 3; // 最大并发数
        this.onProgress = options.onProgress || (() => {});
        this.onComplete = options.onComplete || (() => {});
        this.onError = options.onError || (() => {});
    }

    /**
     * 开始上传
     */
    async start() {
        // 1. 计算文件MD5
        const fileMd5 = await this.calculateMD5();

        // 2. 初始化上传
        const initResp = await fetch('/api/file/init', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                fileName: this.file.name,
                fileSize: this.file.size,
                fileMd5: fileMd5
            })
        });
        const initData = await initResp.json();

        // 3. 秒传检查
        if (initData.data.instantUpload) {
            this.onComplete(initData.data);
            return;
        }

        this.uploadId = initData.data.uploadId;
        this.uploadedChunks = new Set(initData.data.uploadedChunks || []);

        // 4. 切分文件
        this.chunks = this.createChunks();

        // 5. 并行上传分片
        await this.uploadChunks();

        // 6. 合并
        const mergeResp = await fetch('/api/file/merge', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ uploadId: this.uploadId })
        });
        const mergeData = await mergeResp.json();
        this.onComplete(mergeData.data);
    }

    /**
     * 计算文件MD5 (使用spark-md5)
     */
    async calculateMD5() {
        return new Promise((resolve, reject) => {
            const spark = new SparkMD5.ArrayBuffer();
            const reader = new FileReader();
            const chunkSize = 2 * 1024 * 1024; // 2MB chunks for MD5
            let offset = 0;

            const readNext = () => {
                const end = Math.min(offset + chunkSize, this.file.size);
                reader.readAsArrayBuffer(this.file.slice(offset, end));
            };

            reader.onload = (e) => {
                spark.append(e.target.result);
                offset += chunkSize;
                if (offset < this.file.size) {
                    readNext();
                } else {
                    resolve(spark.end());
                }
            };
            reader.onerror = reject;
            readNext();
        });
    }

    /**
     * 切分文件
     */
    createChunks() {
        const chunks = [];
        let index = 0;
        let offset = 0;
        while (offset < this.file.size) {
            const end = Math.min(offset + this.chunkSize, this.file.size);
            chunks.push({
                index: index,
                blob: this.file.slice(offset, end),
                start: offset,
                end: end
            });
            offset = end;
            index++;
        }
        return chunks;
    }

    /**
     * 并行上传分片(带并发控制)
     */
    async uploadChunks() {
        const pendingChunks = this.chunks.filter(
            chunk => !this.uploadedChunks.has(chunk.index)
        );

        const queue = [...pendingChunks];
        const running = [];

        const uploadOne = async (chunk) => {
            const formData = new FormData();
            formData.append('uploadId', this.uploadId);
            formData.append('chunkIndex', chunk.index);
            formData.append('chunk', chunk.blob);

            const resp = await fetch('/api/file/chunk', {
                method: 'POST',
                body: formData
            });
            const data = await resp.json();

            this.uploadedChunks.add(chunk.index);
            this.onProgress({
                uploaded: this.uploadedChunks.size,
                total: this.chunks.length,
                percent: (this.uploadedChunks.size / this.chunks.length * 100).toFixed(1)
            });

            return data;
        };

        while (queue.length > 0 || running.length > 0) {
            while (running.length < this.maxConcurrent && queue.length > 0) {
                const chunk = queue.shift();
                const promise = uploadOne(chunk).finally(() => {
                    const idx = running.indexOf(promise);
                    if (idx > -1) running.splice(idx, 1);
                });
                running.push(promise);
            }
            if (running.length > 0) {
                await Promise.race(running);
            }
        }
    }

    /**
     * 暂停上传
     */
    pause() {
        this.paused = true;
    }

    /**
     * 恢复上传(断点续传)
     */
    async resume() {
        this.paused = false;
        await this.uploadChunks();
    }
}

// 使用示例
const uploader = new FileUploader(file, {
    chunkSize: 5 * 1024 * 1024,
    maxConcurrent: 3,
    onProgress: (progress) => {
        console.log(`上传进度: ${progress.percent}%`);
        updateProgressBar(progress.percent);
    },
    onComplete: (result) => {
        console.log('上传完成:', result.url);
    },
    onError: (error) => {
        console.error('上传失败:', error);
    }
});
uploader.start();
```

## 4. 秒传(MD5)

### 4.1 秒传原理

```
秒传流程:
1. 客户端计算文件MD5
2. 请求服务端检查MD5是否已存在
3. 如果存在 → 直接返回文件URL，不实际上传
4. 如果不存在 → 正常分片上传

秒传原理:
  文件MD5相同 = 文件内容完全相同
  两个用户上传同一文件，第二个用户秒传
  引用计数管理: ref_count + 1

删除文件时:
  ref_count - 1
  ref_count = 0 时才真正删除对象存储中的文件
```

### 4.2 秒传实现

```java
/**
 * 秒传服务
 */
@Service
public class InstantUploadService {

    @Autowired
    private FileMetadataMapper metadataMapper;

    /**
     * 检查秒传
     */
    public InstantCheckResult checkInstantUpload(String fileMd5) {
        FileMetadata existing = metadataMapper.selectByMd5(fileMd5);
        if (existing != null && existing.getStatus() == 1) {
            // 文件已存在，秒传成功
            // 增加引用计数
            metadataMapper.incrementRefCount(existing.getFileId());

            InstantCheckResult result = new InstantCheckResult();
            result.setInstantUpload(true);
            result.setFileId(existing.getFileId());
            result.setUrl(existing.getCdnUrl());
            result.setFileName(existing.getFileName());
            result.setFileSize(existing.getFileSize());
            return result;
        }

        // 文件不存在，需要上传
        InstantCheckResult result = new InstantCheckResult();
        result.setInstantUpload(false);
        return result;
    }
}

/**
 * 文件删除(引用计数)
 */
@Service
public class FileDeleteService {

    @Autowired
    private FileMetadataMapper metadataMapper;

    @Autowired
    private MinioClient minioClient;

    /**
     * 删除文件(减少引用计数)
     */
    @Transactional(rollbackFor = Exception.class)
    public void deleteFile(String fileId) {
        FileMetadata metadata = metadataMapper.selectByFileId(fileId);
        if (metadata == null) return;

        // 减少引用计数
        int newRefCount = metadata.getRefCount() - 1;
        if (newRefCount <= 0) {
            // 引用计数为0，真正删除
            metadata.setStatus(0);
            metadataMapper.updateById(metadata);

            // 删除对象存储中的文件
            try {
                minioClient.removeObject(
                    RemoveObjectArgs.builder()
                        .bucket(metadata.getBucketName())
                        .object(metadata.getObjectKey())
                        .build()
                );
            } catch (Exception e) {
                log.error("对象存储删除失败: {}", fileId, e);
            }
        } else {
            metadata.setRefCount(newRefCount);
            metadataMapper.updateById(metadata);
        }
    }
}
```

## 5. 对象存储(MinIO/S3)

### 5.1 MinIO配置

```java
/**
 * MinIO配置
 */
@Configuration
public class MinIOConfig {

    @Value("${minio.endpoint:http://minio:9000}")
    private String endpoint;

    @Value("${minio.access-key:minioadmin}")
    private String accessKey;

    @Value("${minio.secret-key:minioadmin}")
    private String secretKey;

    @Value("${minio.bucket:uploads}")
    private String bucket;

    @Bean
    public MinioClient minioClient() {
        return MinioClient.builder()
            .endpoint(endpoint)
            .credentials(accessKey, secretKey)
            .build();
    }

    /**
     * 初始化存储桶
     */
    @PostConstruct
    public void initBucket() {
        MinioClient client = minioClient();
        try {
            boolean exists = client.bucketExists(
                BucketExistsArgs.builder().bucket(bucket).build());
            if (!exists) {
                client.makeBucket(MakeBucketArgs.builder().bucket(bucket).build());

                // 设置桶策略: 公共读
                String policy = """
                    {
                        "Version": "2012-10-17",
                        "Statement": [{
                            "Effect": "Allow",
                            "Principal": {"AWS": ["*"]},
                            "Action": ["s3:GetObject"],
                            "Resource": ["arn:aws:s3:::%s/*"]
                        }]
                    }
                """.formatted(bucket);
                client.setBucketPolicy(
                    SetBucketPolicyArgs.builder()
                        .bucket(bucket)
                        .config(policy)
                        .build()
                );
            }
        } catch (Exception e) {
            log.error("MinIO桶初始化失败", e);
        }
    }
}
```

### 5.2 MinIO操作封装

```java
/**
 * 对象存储操作服务
 */
@Service
public class StorageService {

    @Autowired
    private MinioClient minioClient;

    @Value("${minio.bucket:uploads}")
    private String bucket;

    /**
     * 上传文件
     */
    public String upload(String objectKey, InputStream stream, long size, String contentType) {
        try {
            minioClient.putObject(
                PutObjectArgs.builder()
                    .bucket(bucket)
                    .object(objectKey)
                    .stream(stream, size, -1)
                    .contentType(contentType)
                    .build()
            );
            return objectKey;
        } catch (Exception e) {
            throw new RuntimeException("文件上传失败", e);
        }
    }

    /**
     * 生成预签名上传URL
     * 客户端直接上传到MinIO，不经过应用服务器
     */
    public String generatePresignedUploadUrl(String objectKey, int expiry) {
        try {
            return minioClient.getPresignedObjectUrl(
                GetPresignedObjectUrlArgs.builder()
                    .method(Method.PUT)
                    .bucket(bucket)
                    .object(objectKey)
                    .expiry(expiry)
                    .build()
            );
        } catch (Exception e) {
            throw new RuntimeException("生成上传URL失败", e);
        }
    }

    /**
     * 生成预签名下载URL
     */

                    .object(objectKey)
```

## 6. CDN加速

### 6.1 CDN架构

```
用户请求文件:
  1. 用户 → DNS解析 → CDN边缘节点
  2. CDN边缘节点有缓存 → 直接返回(缓存命中)
  3. CDN边缘节点无缓存 → 回源到源站(MinIO/OSS)
  4. 源站返回文件 → CDN缓存 → 返回用户

CDN层级:
  L1: 边缘节点(全国各省) — 缓存热点文件
  L2: 中间源 — 缓存未命中的回源请求
  L3: 源站(MinIO) — 文件原始存储
```

### 6.3 CDN URL生成

```java
/**
 * CDN URL服务
 */
@Service
public class CdnUrlService {

    @Value("${cdn.domain}")
    private String cdnDomain;

    @Value("${cdn.url-sign.enabled:false}")
    private boolean signEnabled;

    @Value("${cdn.url-sign.secret}")
    private String signSecret;

    /**
     * 生成CDN访问URL
     */
    public String buildCdnUrl(String objectKey) {
        String url = "https://" + cdnDomain + "/" + objectKey;

        if (signEnabled) {
            url = signUrl(url);
        }
        return url;
    }

    /**
     * URL签名(防盗链)
     * 格式: https://cdn.example.com/file.jpg?sign=xxx&expire=xxx
     */
    private String signUrl(String url) {
        long expireTime = System.currentTimeMillis() / 1000 + 3600;
        String signBase = url + signSecret + expireTime;
        String sign = DigestUtils.md5Hex(signBase);
        return url + "?sign=" + sign + "&expire=" + expireTime;
    }

    /**
     * 生成带图片处理的CDN URL
     */
    public String buildImageUrl(String objectKey, ImageProcess process) {
        String baseUrl = buildCdnUrl(objectKey);
        StringBuilder params = new StringBuilder();

        if (process.getWidth() != null && process.getHeight() != null) {
            params.append("?imageView2/2/w/").append(process.getWidth())
                  .append("/h/").append(process.getHeight());
        }
        if (process.getQuality() != null) {
            params.append(params.length() > 0 ? "&" : "?");
            params.append("q/").append(process.getQuality());
        }
        if (process.getFormat() != null) {
            params.append(params.length() > 0 ? "&" : "?");
            params.append("format/").append(process.getFormat());
        }

        return baseUrl + params;
    }
}
```

## 7. 文件防盗链

### 7.1 防盗链方案

```java
/**
 * 文件防盗链服务
 */
@Service
public class AntiLeechService {

    @Value("${anti-leech.allowed-domains:example.com,www.example.com}")
    private String allowedDomains;

    @Value("${anti-leech.token-secret}")
    private String tokenSecret;

    /**
     * 方案1: Referer检查
     * 检查请求来源是否为允许的域名
     */
    public boolean checkReferer(String referer) {
        if (StringUtils.isBlank(referer)) {
            // 空Referer: 允许直接访问(浏览器地址栏输入)或拒绝
            return true; // 根据策略决定
        }
        try {
            String domain = new URL(referer).getHost();
            return Arrays.asList(allowedDomains.split(",")).contains(domain);
        } catch (MalformedURLException e) {
            return false;
        }
    }

    /**
     * 方案2: 时间戳+签名验证
     * URL带过期时间和签名，过期或签名不匹配则拒绝
     */
    public String generateSignedUrl(String filePath, int expireSeconds) {
        long expireTime = System.currentTimeMillis() / 1000 + expireSeconds;
        String signBase = filePath + tokenSecret + expireTime;
        String sign = DigestUtils.md5Hex(signBase);

        return String.format("%s?expire=%d&sign=%s", filePath, expireTime, sign);
    }

    /**
     * 验证签名URL
     */
    public boolean verifySignedUrl(String filePath, long expireTime, String sign) {
        // 1. 检查是否过期
        long now = System.currentTimeMillis() / 1000;
        if (now > expireTime) {
            return false;
        }

        // 2. 验证签名
        String signBase = filePath + tokenSecret + expireTime;
        String expectedSign = DigestUtils.md5Hex(signBase);
        return expectedSign.equals(sign);
    }

    /**
     * 方案3: 一次性Token
     * 生成一次性访问Token，使用后失效
     */
    public String generateOneTimeToken(String fileId) {
        String token = UUID.randomUUID().toString().replace("-", "");
        // 存储到Redis，5分钟有效
        redisTemplate.opsForValue().set("file:token:" + token, fileId, 5, TimeUnit.MINUTES);
        return token;
    }

    public boolean consumeOneTimeToken(String token, String fileId) {
        String storedFileId = redisTemplate.opsForValue().get("file:token:" + token);
        if (storedFileId != null && storedFileId.equals(fileId)) {
            redisTemplate.delete("file:token:" + token);
            return true;
        }
        return false;
    }
}
```

```nginx
    # 获取URL参数
    set $expire $arg_expire;
    set $sign $arg_sign;

    # 验证签名 (Lua)
    access_by_lua_block {
        local secret = "your_secret_key"
        local file_path = ngx.var.uri
        local expire = ngx.var.arg_expire
        local sign = ngx.var.arg_sign

        -- 检查过期时间
        local now = ngx.time()
        if tonumber(expire) < now then
            ngx.exit(403)
        end

        -- 验证签名
        local md5 = require "md5"
        local expected = md5.sumhexa(file_path .. secret .. expire)
        if sign ~= expected then
            ngx.exit(403)
        end
    }

    proxy_pass http://minio_upstream;
}
```

```java
/**
 * 预签名URL直传方案
 * 客户端直接上传到MinIO，应用服务器只负责签名
 * 适合大文件上传，减少服务器带宽压力
 */
@RestController
@RequestMapping("/api/file")
public class FileDirectUploadController {

    @Autowired
    private StorageService storageService;

    @Autowired
    private FileMetadataMapper metadataMapper;

    /**
     * 获取预签名上传URL
     */
    @PostMapping("/presign")
    public Result presignUpload(@RequestBody PresignRequest request) {
        // 检查秒传
        FileMetadata existing = metadataMapper.selectByMd5(request.getFileMd5());
        if (existing != null) {
            return Result.success(Map.of(
                "instantUpload", true,
                "url", existing.getCdnUrl()
            ));
        }

        // 生成预签名URL
        String objectKey = buildObjectKey(request.getFileName());
        String uploadUrl = storageService.generatePresignedUploadUrl(objectKey, 3600);

        // 创建待完成记录
        FileMetadata pending = new FileMetadata();
        pending.setFileId(UUID.randomUUID().toString().replace("-", ""));
        pending.setFileName(request.getFileName());
        pending.setFileMd5(request.getFileMd5());
        pending.setObjectKey(objectKey);
        pending.setStatus(0); // 待上传
        metadataMapper.insert(pending);

        return Result.success(Map.of(
            "instantUpload", false,
            "uploadUrl", uploadUrl,
            "fileId", pending.getFileId()
        ));
    }

    /**
     * 上传完成回调
     */
    @PostMapping("/complete")
    public Result uploadComplete(@RequestBody CompleteRequest request) {
        FileMetadata metadata = metadataMapper.selectByFileId(request.getFileId());
        if (metadata == null) {
            return Result.fail("文件记录不存在");
        }

        // 校验文件
        InputStream stream = storageService.download(metadata.getObjectKey());
        String actualMd5 = DigestUtils.md5Hex(stream);
        if (!actualMd5.equals(metadata.getFileMd5())) {
            storageService.delete(metadata.getObjectKey());
            return Result.fail("文件校验失败");
        }

        // 更新状态
        metadata.setStatus(1);
        metadata.setCdnUrl(cdnUrlService.buildCdnUrl(metadata.getObjectKey()));
        metadataMapper.updateById(metadata);

        return Result.success(Map.of("url", metadata.getCdnUrl()));
    }
}
```

## 10. 面试题速查

**Q1: 大文件上传怎么做？**

分片上传：将文件切分为固定大小(如5MB)的分片，并行上传。流程：①初始化上传任务，返回uploadId；②并行上传分片，每片带序号；③所有分片上传完成后请求合并；④服务端按序号合并分片，校验MD5，上传到对象存储。前端控制并发数(3个)，避免带宽饱和。

**Q2: 断点续传怎么实现？**

上传过程中记录已上传分片序号。网络中断后重新初始化时，服务端返回已上传分片列表，前端跳过已上传的分片只上传缺失的。分片状态记录在数据库(upload_task表的uploaded_chunks字段)和前端本地存储中。分片上传接口做幂等处理，重复上传同一分片不会出错。

**Q3: 秒传原理是什么？**

客户端计算文件MD5，上传前先查询服务端是否已存在相同MD5的文件。如果存在说明文件内容完全相同，直接返回文件URL不实际上传。用引用计数(ref_count)管理文件被多少用户引用，删除时ref_count-1，为0时才真正删除对象存储中的文件。

**Q4: MinIO和云OSS怎么选？**

MinIO：开源、S3兼容、可自建部署，适合私有化场景和对数据主权有要求的场景。云OSS(阿里云OSS/AWS S3)：免运维、高可用、全球CDN，适合云原生应用。架构上用StorageService抽象存储层，底层可切换MinIO/OSS/本地存储，业务代码不感知存储实现。

**Q5: 文件防盗链有哪些方案？**

①Referer检查：检查请求来源域名是否在白名单中，简单但Referer可伪造；②签名URL：URL带过期时间+MD5签名，过期或篡改即拒绝；③一次性Token：生成一次性访问令牌，使用后失效；④IP限制：限制同一IP的访问频率。生产中多种方案组合使用。

**Q6: CDN如何加速文件访问？**

用户请求文件时DNS解析到最近的CDN边缘节点。CDN有缓存直接返回，无缓存回源到MinIO/OSS获取。图片/静态资源缓存30天，视频缓存7天。CDN支持图片处理参数(缩放/裁剪/格式转换)，减少源站处理压力。首次访问较慢(回源)，后续命中缓存极快。

**Q7: 预签名URL直传有什么好处？**

客户端直接上传到MinIO/OSS，不经过应用服务器。好处：①减少应用服务器带宽压力；②提高上传速度(少一跳)；③降低应用服务器内存占用。流程：应用生成预签名URL → 客户端直接PUT上传到MinIO → 上传完成后回调应用服务器校验和记录。

**Q8: 文件上传安全怎么保证？**

①文件类型校验：检查扩展名和MIME类型，防止上传可执行文件；②文件大小限制：限制单文件和总大小；③文件内容检查：检查文件头魔数(Magic Number)防止伪装；④重命名：上传后重命名，不使用原始文件名；⑤病毒扫描：对上传文件做病毒扫描；⑥隔离存储：上传目录不可执行。
