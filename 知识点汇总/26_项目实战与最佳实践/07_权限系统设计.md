## 1. 权限系统概述

### 1.1 权限系统全景

```
权限系统层次:
┌─────────────────────────────────────────────┐
│  功能权限                                      │
│  ├── 菜单权限: 控制用户能看到哪些菜单           │
│  ├── 按钮权限: 控制用户能执行哪些操作           │
│  └── 接口权限: 控制用户能调用哪些API            │
├─────────────────────────────────────────────┤
│  数据权限                                      │
│  ├── 行级权限: 控制用户能看到哪些行数据          │
│  └── 字段权限: 控制用户能看到哪些字段            │
├─────────────────────────────────────────────┤
│  字段权限                                      │
│  └── 字段脱敏: 手机号、身份证号部分隐藏         │
└─────────────────────────────────────────────┘
```

### 1.2 核心概念

| 概念 | 描述 | 示例 |
|------|------|------|
| 用户(User) | 系统使用者 | 张三、李四 |
| 角色(Role) | 权限的集合 | 管理员、运营、财务 |
| 权限(Permission) | 具体操作许可 | order:create、order:delete |
| 资源(Resource) | 被保护的对象 | 菜单、按钮、API、数据 |
| 部门(Dept) | 组织架构 | 销售部、技术部 |
| 数据范围(DataScope) | 数据可见范围 | 本人、本部门、本部门及以下、全部 |

## 2. RBAC模型设计

### 2.1 RBAC模型演进

```
RBAC0 (基础模型):
  User → Role → Permission

RBAC1 (引入角色继承):
  User → Role → Permission
         │
         └── 角色继承(经理继承员工权限)

RBAC2 (引入约束):
  - 互斥角色(不能同时是会计和出纳)
  - 角色数量限制
  - 先决条件角色(必须有员工角色才能有经理角色)

RBAC3 = RBAC1 + RBAC2 (完整模型)
```

### 2.3 权限加载

```java
/**
 * 权限加载服务
 */
@Service
public class PermissionLoaderService {

    @Autowired
    private SysUserMapper userMapper;

    @Autowired
    private SysRoleMapper roleMapper;

    @Autowired
    private SysPermissionMapper permissionMapper;

    /**
     * 获取用户所有权限编码
     */
    public Set<String> getUserPermissions(Long userId) {
        // 1. 查询用户所有角色
        List<Long> roleIds = roleMapper.selectRoleIdsByUserId(userId);
        if (roleIds.isEmpty()) {
            return Collections.emptySet();
        }

        // 2. 递归查询继承的角色(角色继承)
        Set<Long> allRoleIds = getAllInheritedRoleIds(roleIds);

        // 3. 查询角色对应的权限
        List<SysPermission> permissions = permissionMapper.selectByRoleIds(allRoleIds);

        // 4. 提取权限编码
        return permissions.stream()
            .filter(p -> p.getStatus() == 1)
            .map(SysPermission::getPermCode)
            .collect(Collectors.toSet());
    }

    /**
     * 递归获取继承的角色ID
     */
    private Set<Long> getAllInheritedRoleIds(List<Long> roleIds) {
        Set<Long> allIds = new HashSet<>(roleIds);
        for (Long roleId : roleIds) {
            List<Long> childIds = roleMapper.selectChildRoleIds(roleId);
            if (!childIds.isEmpty()) {
                allIds.addAll(getAllInheritedRoleIds(childIds));
            }
        }
        return allIds;
    }

    /**
     * 获取用户菜单树
     */
    public List<MenuTreeNode> getUserMenuTree(Long userId) {
        Set<String> permissions = getUserPermissions(userId);
        List<SysPermission> menus = permissionMapper.selectByType(1); // 菜单类型

        // 过滤出用户有权限的菜单
        List<SysPermission> userMenus = menus.stream()
            .filter(m -> permissions.contains(m.getPermCode()))
            .sorted(Comparator.comparing(SysPermission::getSort))
            .collect(Collectors.toList());

        // 构建树形结构
        return buildMenuTree(userMenus, 0L);
    }

    /**
     * 构建菜单树
     */
    private List<MenuTreeNode> buildMenuTree(List<SysPermission> menus, Long parentId) {
        return menus.stream()
            .filter(m -> m.getParentId().equals(parentId))
            .map(m -> {
                MenuTreeNode node = new MenuTreeNode();
                node.setId(m.getId());
                node.setName(m.getPermName());
                node.setPath(m.getResourceUrl());
                node.setComponent(m.getComponent());
                node.setIcon(m.getIcon());
                node.setChildren(buildMenuTree(menus, m.getId()));
                return node;
            })
            .collect(Collectors.toList());
    }
}
```

### 3.2 行级数据权限实现

```java
/**
 * 数据权限拦截器
 * 基于MyBatis拦截器自动拼接SQL条件
 */
@Intercepts({
    @Signature(type = StatementHandler.class, method = "prepare",
               args = {Connection.class, Integer.class})
})
@Component
public class DataPermissionInterceptor implements Interceptor {

    @Override
    public Object intercept(Invocation invocation) throws Throwable {
        StatementHandler handler = (StatementHandler) invocation.getTarget();
        MetaObject metaObject = SystemMetaObject.forObject(handler);

        // 获取原始SQL
        BoundSql boundSql = (BoundSql) metaObject.getValue("delegate.boundSql");
        String sql = boundSql.getSql();

        // 获取Mapper方法上的@DataPermission注解
        MappedStatement ms = (MappedStatement) metaObject.getValue("delegate.mappedStatement");
        String mapperMethodId = ms.getId();
        DataPermission dataPerm = getDataPermissionAnnotation(mapperMethodId);

        if (dataPerm == null) {
            return invocation.proceed(); // 不需要数据权限控制
        }

        // 获取当前用户
        Long userId = SecurityUtils.getCurrentUserId();
        if (userId == null) {
            return invocation.proceed();
        }

        // 构建数据权限SQL
        String dataScopeSql = buildDataScopeSql(userId, dataPerm);
        if (dataScopeSql != null) {
            String newSql = addConditionToSql(sql, dataScopeSql);
            reflectSetFieldValue(boundSql, "sql", newSql);
        }

        return invocation.proceed();
    }

    /**
     * 根据用户角色数据范围构建SQL条件
     */
    private String buildDataScopeSql(Long userId, DataPermission dataPerm) {
        // 获取用户角色中最大的数据范围(取最大范围)
        List<SysRole> roles = roleMapper.selectByUserId(userId);
        if (roles.isEmpty()) return null;

        // 取最大数据范围(ALL > DEPT_AND_CHILD > DEPT > SELF)
        DataScope maxScope = roles.stream()
            .map(r -> DataScope.fromCode(r.getDataScope()))
            .min(Comparator.comparingInt(DataScope::getCode))
            .orElse(DataScope.SELF);

        String tableAlias = dataPerm.tableAlias();
        String deptColumn = tableAlias + ".dept_id";
        String userColumn = tableAlias + ".create_by";

        switch (maxScope) {
            case ALL:
                return null; // 全部数据，不加条件

            case CUSTOM:
                // 自定义部门
                Set<Long> deptIds = roleDeptMapper.selectDeptIdsByRoleIds(
                    roles.stream().map(SysRole::getId).collect(Collectors.toList()));
                if (deptIds.isEmpty()) return "1=0";
                return deptColumn + " IN (" + StringUtils.join(deptIds, ",") + ")";

            case DEPT:
                // 本部门
                Long deptId = userMapper.selectDeptIdByUserId(userId);
                return deptColumn + " = " + deptId;

            case DEPT_AND_CHILD:
                // 本部门及以下
                Long userDeptId = userMapper.selectDeptIdByUserId(userId);
                List<Long> childDeptIds = deptMapper.selectChildDeptIds(userDeptId);
                childDeptIds.add(userDeptId);
                return deptColumn + " IN (" + StringUtils.join(childDeptIds, ",") + ")";

            case SELF:
                // 仅本人
                return userColumn + " = " + userId;

            default:
                return "1=0";
        }
    }

    /**
     * 使用JSqlParse向SQL添加条件
     */
    private String addConditionToSql(String sql, String condition) {
        Statement statement = CCJSqlParserUtil.parse(sql);
        if (statement instanceof Select) {
            PlainSelect select = (PlainSelect) ((Select) statement).getSelectBody();
            Expression where = select.getWhere();
            if (where == null) {
                select.setWhere(CCJSqlParserUtil.parseCondExpression(condition));
            } else {
                AndExpression and = new AndExpression(where,
                    CCJSqlParserUtil.parseCondExpression(condition));
                select.setWhere(and);
            }
        }
        return statement.toString();
    }
}

/**
 * 数据权限注解
 */
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface DataPermission {
    String tableAlias() default "t";  // 表别名
    String deptField() default "dept_id";  // 部门字段
    String userField() default "create_by";  // 用户字段
}
```

### 3.3 字段级权限

```java
/**
 * 字段权限控制
 * 控制用户能查看哪些字段
 */
@Service
public class FieldPermissionService {

    @Autowired
    private SysRoleMapper roleMapper;

    /**
     * 字段权限配置: 角色 → 可见字段
     */
    private static final Map<String, Set<String>> FIELD_PERMISSIONS = new HashMap<>();

    static {
        // 财务角色可以看到金额字段
        FIELD_PERMISSIONS.put("finance", Set.of("id", "order_no", "amount", "pay_amount", "status"));
        // 运营角色不能看到金额字段
        FIELD_PERMISSIONS.put("operator", Set.of("id", "order_no", "user_name", "status", "create_time"));
    }

    /**
     * 根据用户角色过滤字段
     */
    public <T> T filterFields(T entity, Long userId) {
        List<SysRole> roles = roleMapper.selectByUserId(userId);
        Set<String> visibleFields = roles.stream()
            .map(r -> FIELD_PERMISSIONS.get(r.getRoleCode()))
            .filter(Objects::nonNull)
            .flatMap(Set::stream)
            .collect(Collectors.toSet());

        if (visibleFields.isEmpty()) return entity;

        try {
            Class<?> clazz = entity.getClass();
            for (Field field : clazz.getDeclaredFields()) {
                if (!visibleFields.contains(field.getName()) && !field.getName().startsWith("serial")) {
                    field.setAccessible(true);
                    field.set(entity, null); // 隐藏字段设为null
                }
            }
        } catch (Exception e) {
            log.error("字段权限过滤失败", e);
        }
        return entity;
    }

    /**
     * 列表数据字段过滤
     */
    public <T> List<T> filterFieldsList(List<T> entities, Long userId) {
        return entities.stream()
            .map(e -> filterFields(e, userId))
            .collect(Collectors.toList());
    }
}
```

### 3.4 字段脱敏

```java
/**
 * 数据脱敏注解
 */
@Target(ElementType.FIELD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Sensitive {
    SensitiveType value();
}

public enum SensitiveType {
    PHONE,       // 手机号: 138****8888
    ID_CARD,     // 身份证: 110***********1234
    EMAIL,       // 邮箱: a***@example.com
    BANK_CARD,   // 银行卡: 6222****1234
    NAME,        // 姓名: 张*
    ADDRESS      // 地址: 北京市***
}

/**
 * 脱敏序列化器
 */
public class SensitiveSerializer extends JsonSerializer<String>
        implements ContextualSerializer {

    private SensitiveType type;

    @Override
    public void serialize(String value, JsonGenerator gen, SerializerProvider provider)
            throws IOException {
        gen.writeString(desensitize(value, type));
    }

    @Override
    public JsonSerializer<?> createContextual(SerializerProvider provider, BeanProperty property) {
        Sensitive annotation = property.getAnnotation(Sensitive.class);
        SensitiveSerializer serializer = new SensitiveSerializer();
        serializer.type = annotation.value();
        return serializer;
    }

    private String desensitize(String value, SensitiveType type) {
        if (value == null) return null;
        switch (type) {
            case PHONE:
                return value.replaceAll("(\\d{3})\\d{4}(\\d{4})", "$1****$2");
            case ID_CARD:
                return value.replaceAll("(\\d{3})\\d{11}(\\d{4})", "$1***********$2");
            case EMAIL:
                int at = value.indexOf('@');
                if (at <= 1) return value;
                return value.charAt(0) + "***" + value.substring(at);
            case BANK_CARD:
                return value.replaceAll("(\\d{4})\\d{8}(\\d{4})", "$1********$2");
            case NAME:
                return value.length() <= 1 ? value : value.charAt(0) + "*".repeat(value.length() - 1);
            case ADDRESS:
                return value.length() > 6 ? value.substring(0, 6) + "***" : "***";
            default:
                return value;
        }
    }
}

// 使用示例
@Data
public class UserVO {
    private Long id;
    private String username;

    @Sensitive(SensitiveType.PHONE)
    @JsonSerialize(using = SensitiveSerializer.class)
    private String phone;

    @Sensitive(SensitiveType.ID_CARD)
    @JsonSerialize(using = SensitiveSerializer.class)
    private String idCard;

    @Sensitive(SensitiveType.EMAIL)
    @JsonSerialize(using = SensitiveSerializer.class)
    private String email;
}
```

```javascript
const whiteList = ['/login', '/404', '/403'];

router.beforeEach(async (to, from, next) => {
    const token = getToken();
    if (token) {
        if (to.path === '/login') {
            next({ path: '/' });
        } else {
            // 检查是否已加载路由
            if (store.getters.routes.length === 0) {
                try {
                    // 从后端获取菜单和权限
                    const { menus, permissions } = await store.dispatch('user/getInfo');

                    // 生成动态路由
                    const accessRoutes = generateRoutes(menus);

                    // 添加路由
                    accessRoutes.forEach(route => {
                        router.addRoute(route);
                    });

                    // 存储按钮权限
                    store.commit('user/SET_PERMISSIONS', permissions);

                    next({ ...to, replace: true });
                } catch (error) {
                    await store.dispatch('user/resetToken');
                    next(`/login?redirect=${to.path}`);
                }
            } else {
                next();
            }
        }
    } else {
        if (whiteList.includes(to.path)) {
            next();
        } else {
            next(`/login?redirect=${to.path}`);
        }
    }
});

// 根据后端菜单数据生成Vue路由
function generateRoutes(menus) {
    const routes = [];
    menus.forEach(menu => {
        const route = {
            path: menu.path,
            name: menu.name,
            component: () => import(`@/views/${menu.component}`),
            meta: {
                title: menu.meta.title,
                icon: menu.meta.icon,
                permissions: menu.permissions || []
            }
        };
        if (menu.children && menu.children.length > 0) {
            route.children = generateRoutes(menu.children);
        }
        routes.push(route);
    });
    return routes;
}
```

```javascript
        const { value } = binding;
```

### 5.2 后端接口权限校验

```java
/**
 * 权限校验注解
 */
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface RequiresPermission {
    String value();           // 权限编码
    Logical logical() default Logical.AND;  // 多个权限的逻辑关系
}

/**
 * AOP权限校验
 */
@Aspect
@Component
public class PermissionAspect {

    @Autowired
    private PermissionLoaderService permissionLoader;

    @Autowired
    private StringRedisTemplate redisTemplate;

    @Around("@annotation(requiresPermission)")
    public Object checkPermission(ProceedingJoinPoint pjp,
            RequiresPermission requiresPermission) throws Throwable {
        Long userId = SecurityUtils.getCurrentUserId();
        if (userId == null) {
            throw new UnauthorizedException("未登录");
        }

        // 从缓存获取用户权限
        Set<String> userPermissions = getUserPermissionsFromCache(userId);

        String requiredPerm = requiresPermission.value();
        if (!userPermissions.contains(requiredPerm)) {
            throw new ForbiddenException("无权限: " + requiredPerm);
        }

        return pjp.proceed();
    }

    private Set<String> getUserPermissionsFromCache(Long userId) {
        String key = "user:permissions:" + userId;
        String cached = redisTemplate.opsForValue().get(key);
        if (cached != null) {
            return JSON.parseObject(cached, new TypeReference<Set<String>>() {});
        }

        Set<String> permissions = permissionLoader.getUserPermissions(userId);
        redisTemplate.opsForValue().set(key, JSON.toJSONString(permissions), 30, TimeUnit.MINUTES);
        return permissions;
    }
}
```

```java
/**
 * 权限缓存管理
 * 用户权限变更时主动刷新缓存
 */
@Service
public class PermissionCacheManager {

    @Autowired
    private StringRedisTemplate redisTemplate;

    private static final String USER_PERM_KEY = "user:permissions:";
    private static final String USER_ROLE_KEY = "user:roles:";
    private static final String USER_MENU_KEY = "user:menus:";

    /**
     * 用户权限缓存
     * 结构: Set<权限编码>
     */
    public void cacheUserPermissions(Long userId, Set<String> permissions) {
        String key = USER_PERM_KEY + userId;
        redisTemplate.delete(key);
        if (!permissions.isEmpty()) {
            redisTemplate.opsForSet().add(key, permissions.toArray(new String[0]));
        }
        redisTemplate.expire(key, 2, TimeUnit.HOURS);
    }

    public Set<String> getCachedPermissions(Long userId) {
        String key = USER_PERM_KEY + userId;
        Set<String> members = redisTemplate.opsForSet().members(key);
        return members != null ? members : Collections.emptySet();
    }

    /**
     * 角色权限变更时，清除该角色下所有用户的权限缓存
     */
    public void evictByRole(Long roleId) {
        List<Long> userIds = userRoleMapper.selectUserIdsByRoleId(roleId);
        for (Long userId : userIds) {
            redisTemplate.delete(USER_PERM_KEY + userId);
            redisTemplate.delete(USER_MENU_KEY + userId);
        }
        log.info("角色 {} 权限变更，清除 {} 个用户缓存", roleId, userIds.size());
    }

    /**
     * 用户角色变更时，清除该用户缓存
     */
    public void evictByUser(Long userId) {
        redisTemplate.delete(USER_PERM_KEY + userId);
        redisTemplate.delete(USER_ROLE_KEY + userId);
        redisTemplate.delete(USER_MENU_KEY + userId);
    }

    /**
     * 全量刷新(谨慎使用)
     */
    public void evictAll() {
        Set<String> keys = redisTemplate.keys(USER_PERM_KEY + "*");
        if (keys != null) {
            redisTemplate.delete(keys);
        }
    }
}
```

```java
public class PermissionChangeEvent extends ApplicationEvent {
    private ChangeType changeType;
    private Long targetId;  // userId 或 roleId

    public enum ChangeType {
        USER_ROLE_CHANGED,      // 用户角色变更
        ROLE_PERMISSION_CHANGED, // 角色权限变更
        ROLE_STATUS_CHANGED,    // 角色状态变更
        PERMISSION_UPDATED      // 权限信息更新
    }
}

/**
 * 权限变更监听器
 */
@Component
public class PermissionChangeListener {

    @Autowired
    private PermissionCacheManager cacheManager;

    @EventListener
    @Async
    public void onPermissionChange(PermissionChangeEvent event) {
        switch (event.getChangeType()) {
            case USER_ROLE_CHANGED:
                cacheManager.evictByUser(event.getTargetId());
                break;
            case ROLE_PERMISSION_CHANGED:
            case ROLE_STATUS_CHANGED:
                cacheManager.evictByRole(event.getTargetId());
                break;
            case PERMISSION_UPDATED:
                cacheManager.evictAll();
                break;
        }
    }
}
```

```java
/**
 * JWT认证过滤器
 * 解析Token → 加载用户权限 → 设置SecurityContext
 */
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    @Autowired
    private JwtTokenProvider tokenProvider;

    @Autowired
    private UserDetailsService userDetailsService;

    @Override
    protected void doFilterInternal(HttpServletRequest request,
            HttpServletResponse response, FilterChain filterChain)
            throws ServletException, IOException {

        try {
            String token = extractToken(request);
            if (StringUtils.isNotBlank(token) && tokenProvider.validateToken(token)) {
                Long userId = tokenProvider.getUserIdFromToken(token);

                // 加载用户权限
                UserDetails userDetails = userDetailsService.loadUserByUsername(
                    tokenProvider.getUsernameFromToken(token));

                UsernamePasswordAuthenticationToken authentication =
                    new UsernamePasswordAuthenticationToken(
                        userDetails, null, userDetails.getAuthorities());

                authentication.setDetails(
                    new WebAuthenticationDetailsSource().buildDetails(request));

                SecurityContextHolder.getContext().setAuthentication(authentication);
            }
        } catch (Exception e) {
            log.error("JWT认证失败", e);
        }

        filterChain.doFilter(request, response);
    }

    private String extractToken(HttpServletRequest request) {
        String bearer = request.getHeader("Authorization");
        if (StringUtils.isNotBlank(bearer) && bearer.startsWith("Bearer ")) {
            return bearer.substring(7);
        }
        return null;
    }
}
```

## 8. 权限缓存优化

### 8.1 多级缓存

```java
/**
 * 权限多级缓存: Caffeine(本地) + Redis(分布式)
 */
@Service
public class PermissionCacheService {

    // L1: 本地缓存 (Caffeine) — 5分钟
    private final Cache<String, Set<String>> localCache = Caffeine.newBuilder()
        .maximumSize(10000)
        .expireAfterWrite(5, TimeUnit.MINUTES)
        .build();

    @Autowired
    private StringRedisTemplate redisTemplate;

    @Autowired
    private PermissionLoaderService permissionLoader;

    /**
     * 获取用户权限(多级缓存)
     */
    public Set<String> getUserPermissions(Long userId) {
        String key = "perm:user:" + userId;

        // L1: 本地缓存
        Set<String> cached = localCache.getIfPresent(key);
        if (cached != null) return cached;

        // L2: Redis
        String redisKey = "user:permissions:" + userId;
        Set<String> redisCached = redisTemplate.opsForSet().members(redisKey);
        if (redisCached != null && !redisCached.isEmpty()) {
            localCache.put(key, redisCached);
            return redisCached;
        }

        // L3: 数据库
        Set<String> dbPerms = permissionLoader.getUserPermissions(userId);
        if (!dbPerms.isEmpty()) {
            redisTemplate.opsForSet().add(redisKey, dbPerms.toArray(new String[0]));
            redisTemplate.expire(redisKey, 2, TimeUnit.HOURS);
        }
        localCache.put(key, dbPerms);
        return dbPerms;
    }

    /**
     * 批量权限检查(使用BitMap优化)
     * 适合权限项固定的场景
     */
    public boolean hasPermission(Long userId, String permissionCode) {
        // 将权限编码映射到位索引
        int bitIndex = PermissionCodeMapper.getBitIndex(permissionCode);
        if (bitIndex < 0) return false;

        // 从Redis BitMap中检查
        String bitmapKey = "perm:bitmap:" + userId;
        return Boolean.TRUE.equals(redisTemplate.opsForValue().getBit(bitmapKey, bitIndex));
    }

    /**
     * 初始化用户权限BitMap
     */
    public void initPermissionBitmap(Long userId, Set<String> permissions) {
        String bitmapKey = "perm:bitmap:" + userId;
        redisTemplate.delete(bitmapKey);

        for (String perm : permissions) {
            int bitIndex = PermissionCodeMapper.getBitIndex(perm);
            if (bitIndex >= 0) {
                redisTemplate.opsForValue().setBit(bitmapKey, bitIndex, true);
            }
        }
        redisTemplate.expire(bitmapKey, 2, TimeUnit.HOURS);
    }
}
```

```java
/**

@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface AuditLog {
    String module() default "";    // 模块
    String action() default "";    // 操作
    String description() default ""; // 描述
}

@Aspect
@Component
public class AuditLogAspect {

    @Autowired
    private AuditLogMapper auditLogMapper;

    @Around("@annotation(auditLog)")
    public Object record(ProceedingJoinPoint pjp, AuditLog auditLog) throws Throwable {
        Long userId = SecurityUtils.getCurrentUserId();
        String username = SecurityUtils.getCurrentUsername();
        String ip = RequestUtil.getClientIp();
        String requestURI = RequestUtil.getRequestURI();
        String method = RequestUtil.getMethod();
        String params = JSON.toJSONString(RequestUtil.getParams());

        AuditLogRecord record = new AuditLogRecord();
        record.setUserId(userId);
        record.setUsername(username);
        record.setModule(auditLog.module());
        record.setAction(auditLog.action());
        record.setDescription(auditLog.description());
        record.setRequestUri(requestURI);
        record.setHttpMethod(method);
       .setRequestParams(params);
        record.setClientIp(ip);
        record.setOperateTime(LocalDateTime.now());

        long startTime = System.currentTimeMillis();
        try {
            Object result = pjp.proceed();
            record.setStatus(1); // 成功
            record.setResult(JSON.toJSONString(result));
            return result;
        } catch (Throwable e) {
            record.setStatus(0); // 失败
            record.setErrorMsg(e.getMessage());
            throw e;
        } finally {
            record.setCostTime(System.currentTimeMillis() - startTime);
            // 异步保存日志
            CompletableFuture.runAsync(() -> auditLogMapper.insert(record));
        }
    }
}
```

## 10. 面试题速查

**Q1: RBAC模型是什么？比ACL好在哪里？**

RBAC（基于角色的访问控制）：用户→角色→权限。用户不直接关联权限，而是通过角色间接获得权限。优势：① 批量管理，角色变更自动影响所有关联用户；② 职责分离，用户管理和权限管理解耦；③ 支持角色继承和互斥约束。ACL（访问控制列表）直接维护用户-资源映射，用户量大时维护成本高。

**Q2: 数据权限怎么实现？**

行级权限通过MyBatis拦截器自动拼接SQL条件。根据角色的数据范围(全部/本部门/本部门及以下/本人/自定义)添加WHERE条件。用JSqlParse解析SQL并注入条件，对业务代码零侵入。字段级权限通过序列化时过滤，无权查看的字段设为null或脱敏。

**Q3: 菜单权限和按钮权限怎么配合？**

菜单权限控制页面可见性：后端返回用户菜单树，前端动态注册路由。按钮权限控制操作可见性：前端通过v-permission指令，检查用户权限列表中是否包含该按钮权限编码。后端接口同时校验权限(@RequiresPermission注解)，前后端双重保障。

**Q4: 权限缓存怎么做？**

多级缓存：Caffeine本地缓存(5分钟)→Redis分布式缓存(2小时)→数据库。用户登录时加载权限到缓存，权限变更时通过事件驱动清除相关缓存(角色权限变更清除该角色所有用户的缓存)。权限项固定时可用BitMap优化存储空间和查询性能。

**Q5: 角色继承怎么实现？**

角色表parent_id字段指向父角色。加载权限时递归查询父角色权限合并。注意：① 避免循环继承(校验parent_id不能形成环)；② 权限取并集(子角色拥有父角色的所有权限+自己的权限)；③ 数据范围取最大范围(子角色的数据范围不能小于父角色)。

**Q6: Spring Security怎么集成自定义权限？**

① 实现UserDetailsService加载用户权限；② JWT过滤器解析Token设置SecurityContext；③ @PreAuthorize注解做方法级权限控制；④ 自定义AccessDeniedHandler处理403；⑤ 自定义AuthenticationEntryPoint处理401。核心是将自定义权限编码映射为Spring Security的GrantedAuthority。

**Q7: 权限系统怎么做审计？**

AOP切面+@AuditLog注解，记录操作人、操作时间、模块、动作、请求参数、响应结果、耗时、IP等信息。日志异步写入数据库(不阻塞主流程)。支持按时间、用户、模块查询审计日志，满足安全合规要求。

**Q8: 数据权限拦截器怎么防止绕过？**

① MyBatis拦截器在SQL层面注入条件，业务代码无法绕过；② 同时在Service层用@DataPermission注解声明数据范围需求；③ 管理员账号的超级权限要严格限制，避免全量查询被滥用；④ 审计日志记录所有数据查询操作，定期审计异常查询。
