## 1. 业务背景与需求分析

### 1.1 业务规模

假设一个中型电商平台，日均订单量50万，大促期间峰值QPS可达5万，商品SKU数超过1000万，用户数2000万。

### 1.2 核心业务流程

```
用户浏览商品 → 加入购物车 → 使用优惠券 → 创建订单 → 扣减库存 → 支付 → 发货 → 确认收货
```

### 1.3 技术挑战

| 挑战 | 描述 |
|------|------|
| 高并发 | 大促时QPS从日常5000飙升至50000 |
| 数据一致性 | 库存扣减、优惠券核销涉及跨服务事务 |
| 服务拆分粒度 | 拆太细则通信成本高，拆太粗则耦合严重 |
| 系统可用性 | 核心链路任何一个服务不可用都会影响下单 |
| 数据膨胀 | 订单、商品数据持续增长需要分库分表 |

## 2. 微服务拆分方案

### 2.1 拆分原则

- **单一职责**：每个服务只负责一个业务领域
- **高内聚低耦合**：服务内部功能高度相关，服务间依赖最小化
- **独立部署**：每个服务可以独立开发、测试、部署
- **数据自治**：每个服务拥有独立数据库，不直接访问其他服务的数据库

### 2.2 服务拆分图

```
                    ┌─────────────┐
```

### 3.2 商品搜索

使用Elasticsearch构建商品搜索引擎，支持全文检索、多维度筛选和排序：

```java
@Service
public class ProductSearchService {

    @Autowired
    private RestHighLevelClient esClient;

    public SearchResponse searchProducts(ProductSearchDTO searchDTO) throws IOException {
        SearchRequest searchRequest = new SearchRequest("product_index");
        SearchSourceBuilder sourceBuilder = new SearchSourceBuilder();

        // 构建布尔查询
        BoolQueryBuilder boolQuery = QueryBuilders.boolQuery();

        // 关键词搜索
        if (StringUtils.isNotBlank(searchDTO.getKeyword())) {
            boolQuery.must(QueryBuilders.multiMatchQuery(searchDTO.getKeyword(),
                "title", "subtitle", "brandName")
                .type(MultiMatchQueryBuilder.Type.BEST_FIELDS)
                .tieBreaker(0.3f));
        }

        // 分类过滤
        if (searchDTO.getCategoryId() != null) {
            boolQuery.filter(QueryBuilders.termQuery("categoryId", searchDTO.getCategoryId()));
        }

        // 价格区间过滤
        if (searchDTO.getMinPrice() != null && searchDTO.getMaxPrice() != null) {
            boolQuery.filter(QueryBuilders.rangeQuery("price")
                .gte(searchDTO.getMinPrice())
                .lte(searchDTO.getMaxPrice()));
        }

        // 品牌过滤
        if (CollectionUtils.isNotEmpty(searchDTO.getBrandIds())) {
            boolQuery.filter(QueryBuilders.termsQuery("brandId", searchDTO.getBrandIds()));
        }

        sourceBuilder.query(boolQuery);

        // 排序
        if ("price_asc".equals(searchDTO.getSort())) {
            sourceBuilder.sort("price", SortOrder.ASC);
        } else if ("price_desc".equals(searchDTO.getSort())) {
            sourceBuilder.sort("price", SortOrder.DESC);
        } else {
            // 默认按综合相关度排序
            sourceBuilder.sort("_score", SortOrder.DESC);
        }

        // 分页
        sourceBuilder.from((searchDTO.getPage() - 1) * searchDTO.getSize());
        sourceBuilder.size(searchDTO.getSize());

        // 聚合：返回品牌和分类的聚合数据用于侧边栏筛选
        sourceBuilder.aggregation(AggregationBuilders.terms("brand_agg").field("brandId").size(20));
        sourceBuilder.aggregation(AggregationBuilders.terms("category_agg").field("categoryId").size(20));

        searchRequest.source(sourceBuilder);
        return esClient.search(searchRequest, RequestOptions.DEFAULT);
    }
}
```

### 3.3 商品详情页缓存策略

商品详情页是高流量入口，采用多级缓存：

```java
@Service
public class ProductDetailService {

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    @Autowired
    private ProductService productService;

    @Autowired
    private LocalCache localCache; // Caffeine本地缓存

    /**
     * 多级缓存获取商品详情
     * L1: Caffeine本地缓存 (5分钟) — 防止热点Key打穿Redis
     * L2: Redis分布式缓存 (30分钟) — 跨实例共享
     * L3: MySQL数据库 — 数据源头
     */
    public ProductDetailVO getProductDetail(Long spuId) {
        String cacheKey = "product:detail:" + spuId;

        // L1: 本地缓存
        ProductDetailVO detail = localCache.getIfPresent(cacheKey);
        if (detail != null) {
            return detail;
        }

        // L2: Redis缓存
        detail = (ProductDetailVO) redisTemplate.opsForValue().get(cacheKey);
        if (detail != null) {
            localCache.put(cacheKey, detail);
            return detail;
        }

        // L3: 数据库查询 + 防缓存击穿(分布式锁)
        String lockKey = "lock:product:" + spuId;
        RLock lock = redissonClient.getLock(lockKey);
        try {
            // 尝试加锁，最多等待3秒
            if (lock.tryLock(3, TimeUnit.SECONDS)) {
                // 双重检查
                detail = (ProductDetailVO) redisTemplate.opsForValue().get(cacheKey);
                if (detail != null) {
                    localCache.put(cacheKey, detail);
                    return detail;
                }

                // 查询数据库
                detail = productService.queryDetailFromDB(spuId);
                if (detail != null) {
                    // 写入Redis，设置随机过期时间防止缓存雪崩
                    int expire = 1800 + ThreadLocalRandom.current().nextInt(300);
                    redisTemplate.opsForValue().set(cacheKey, detail, expire, TimeUnit.SECONDS);
                    localCache.put(cacheKey, detail);
                } else {
                    // 空值缓存，防止缓存穿透
                    redisTemplate.opsForValue().set(cacheKey, "NULL", 60, TimeUnit.SECONDS);
                }
                return detail;
            } else {
                // 获取锁失败，短暂等待后重试读取缓存
                Thread.sleep(50);
                return (ProductDetailVO) redisTemplate.opsForValue().get(cacheKey);
            }
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new RuntimeException("获取商品详情中断", e);
        } finally {
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }
}
```

### 4.1 订单状态机

```java
/**
 * 订单状态枚举
 */
public enum OrderStatus {
    PENDING_PAYMENT(1, "待付款"),
    PENDING_SHIPMENT(2, "待发货"),
    PENDING_DELIVERY(3, "待收货"),
    COMPLETED(4, "已完成"),
    CANCELLED(5, "已取消"),
    REFUNDING(6, "退款中"),
    REFUNDED(7, "已退款");

    private final int code;
    private final String desc;

    OrderStatus(int code, String desc) {
        this.code = code;
        this.desc = desc;
    }
}

/**
 * 订单状态流转规则
 */
public class OrderStateMachine {

    private static final Map<OrderStatus, Set<OrderStatus>> TRANSITIONS = new HashMap<>();

    static {
        TRANSITIONS.put(OrderStatus.PENDING_PAYMENT,
            new HashSet<>(Arrays.asList(OrderStatus.PENDING_SHIPMENT, OrderStatus.CANCELLED)));
        TRANSITIONS.put(OrderStatus.PENDING_SHIPMENT,
            new HashSet<>(Arrays.asList(OrderStatus.PENDING_DELIVERY, OrderStatus.REFUNDING)));
        TRANSITIONS.put(OrderStatus.PENDING_DELIVERY,
            new HashSet<>(Arrays.asList(OrderStatus.COMPLETED, OrderStatus.REFUNDING)));
        TRANSITIONS.put(OrderStatus.REFUNDING,
            new HashSet<>(Arrays.asList(OrderStatus.REFUNDED, OrderStatus.PENDING_SHIPMENT)));
    }

    public static boolean canTransition(OrderStatus from, OrderStatus to) {
        Set<OrderStatus> allowed = TRANSITIONS.get(from);
        return allowed != null && allowed.contains(to);
    }

    public static void transition(Order order, OrderStatus target) {
        if (!canTransition(order.getStatus(), target)) {
            throw new OrderStatusException(
                String.format("订单状态不能从[%s]变更为[%s]", order.getStatus().getDesc(), target.getDesc()));
        }
        order.setStatus(target);
    }
}
```

### 4.2 创建订单核心流程

```java
@Service
public class OrderCreateService {

    @Autowired
    private InventoryService inventoryService;

    @Autowired
    private CouponService couponService;

    @Autowired
    private OrderMapper orderMapper;

    @Autowired
    private RocketMQTemplate rocketMQTemplate;

    /**
     * 创建订单 — 涉及多个服务协作
     * 1. 校验商品和价格
     * 2. 扣减库存（预占）
     * 3. 核销优惠券
     * 4. 创建订单
     * 5. 发送延迟消息（超时取消）
     */
    @Transactional(rollbackFor = Exception.class)
    public OrderCreateResult createOrder(OrderCreateRequest request) {
        // 生成订单号
        String orderNo = generateOrderNo(request.getUserId());

        // 1. 校验商品信息并计算价格
        BigDecimal totalAmount = BigDecimal.ZERO;
        List<OrderItem> orderItems = new ArrayList<>();
        for (OrderItemRequest itemReq : request.getItems()) {
            ProductSku sku = productService.getSku(itemReq.getSkuId());
            if (sku == null || sku.getStatus() != 1) {
                throw new BizException("商品不存在或已下架");
            }
            // 校验前端传过来的价格是否与后台一致
            if (sku.getPrice().compareTo(itemReq.getPrice()) != 0) {
                throw new BizException("商品价格已变更，请刷新重试");
            }
            totalAmount = totalAmount.add(sku.getPrice().multiply(BigDecimal.valueOf(itemReq.getQuantity())));

            OrderItem item = new OrderItem();
            item.setOrderNo(orderNo);
            item.setSkuId(sku.getId());
            item.setSpuId(sku.getSpuId());
            item.setQuantity(itemReq.getQuantity());
            item.setUnitPrice(sku.getPrice());
            orderItems.add(item);
        }

        // 2. 扣减库存（预占模式）
        for (OrderItem item : orderItems) {
            boolean success = inventoryService.deductStock(item.getSkuId(), item.getQuantity());
            if (!success) {
                throw new BizException("商品库存不足: " + item.getSkuId());
            }
        }

        // 3. 优惠券核销
        BigDecimal payAmount = totalAmount;
        if (request.getCouponId() != null) {
            CouponDeductResult deductResult = couponService.deduct(
                request.getCouponId(), request.getUserId(), totalAmount);
            payAmount = totalAmount.subtract(deductResult.getDeductAmount());
            if (payAmount.compareTo(BigDecimal.ZERO) < 0) {
                payAmount = BigDecimal.ZERO;
            }
        }

        // 4. 创建订单
        Order order = new Order();
        order.setOrderNo(orderNo);
        order.setUserId(request.getUserId());
        order.setTotalAmount(totalAmount);
        order.setPayAmount(payAmount);
        order.setCouponId(request.getCouponId());
        order.setStatus(OrderStatus.PENDING_PAYMENT);
        order.setExpireTime(LocalDateTime.now().plusMinutes(30)); // 30分钟超时
        orderMapper.insert(order);
        orderItemMapper.batchInsert(orderItems);

        // 5. 发送延迟消息，30分钟后检查订单是否已支付
        rocketMQTemplate.asyncSend(
            "order-timeout-topic:" + orderNo,
            MessageBuilder.withPayload(orderNo).build(),
            new DefaultMQProducer() {{ setDelayTimeLevel(16); }} // RocketMQ延迟级别16≈30分钟
        );

        return new OrderCreateResult(orderNo, payAmount);
    }

    /**
     * 生成唯一订单号
     * 格式: 年月日(8位) + 时分秒毫秒(9位) + 机器ID(2位) + 序列号(3位) = 22位
     */
    private String generateOrderNo(Long userId) {
        String datePart = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMddHHmmssSSS"));
        String machineId = String.format("%02d", MachineUtil.getMachineId());
        String sequence = String.format("%03d", SequenceUtil.nextId());
        return datePart + machineId + sequence;
    }
}
```

### 4.3 订单超时取消

```java
@RocketMQMessageListener(
    topic = "order-timeout-topic",
    consumerGroup = "order-timeout-consumer",
    messageModel = MessageModel.CLUSTERING
)
@Component
public class OrderTimeoutConsumer implements RocketMQListener<String> {

    @Autowired
    private OrderService orderService;

    @Autowired
    private InventoryService inventoryService;

    @Autowired
    private CouponService couponService;

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void onMessage(String orderNo) {
        Order order = orderService.getByOrderNo(orderNo);
        if (order == null) {
            return;
        }

        // 仅待付款订单才需要超时取消
        if (order.getStatus() != OrderStatus.PENDING_PAYMENT) {
            return;
        }

        // 更新订单状态为已取消
        orderService.updateStatus(orderNo, OrderStatus.CANCELLED, "超时未支付自动取消");

        // 回滚库存预占
        List<OrderItem> items = orderItemMapper.findByOrderNo(orderNo);
        for (OrderItem item : items) {
            inventoryService.rollbackStock(item.getSkuId(), item.getQuantity());
        }

        // 退回优惠券
        if (order.getCouponId() != null) {
            couponService.returnCoupon(order.getCouponId(), order.getUserId());
        }
    }
}
```

### 5.2 支付回调处理

```java
@Service
public class PayCallbackService {

    @Autowired
    private OrderService orderService;

    @Autowired
    private PayMapper payMapper;

    /**
     * 处理支付宝异步回调
     * 关键点：幂等性处理 + 验签 + 状态校验
     */
    @Transactional(rollbackFor = Exception.class)
    public String handleAlipayCallback(Map<String, String> params) {
        // 1. 验签
        boolean signVerified = AlipaySignature.rsaCheckV1(params,
            alipayPublicKey, "UTF-8", "RSA2");
        if (!signVerified) {
            return "fail";
        }

        // 2. 幂等性检查
        String tradeNo = params.get("trade_no");
        PayRecord existRecord = payMapper.selectByTradeNo(tradeNo);
        if (existRecord != null && existRecord.getStatus() == PayStatus.SUCCESS) {
            return "success"; // 已处理过，直接返回成功
        }

        // 3. 校验金额
        String orderNo = params.get("out_trade_no");
        Order order = orderService.getByOrderNo(orderNo);
        if (order == null) {
            return "fail";
        }

        BigDecimal notifyAmount = new BigDecimal(params.get("total_amount"));
        if (notifyAmount.compareTo(order.getPayAmount()) != 0) {
            log.warn("支付金额不一致, 订单: {}, 应付: {}, 实付: {}",
                orderNo, order.getPayAmount(), notifyAmount);
            return "fail";
        }

        // 4. 更新支付记录
        PayRecord payRecord = new PayRecord();
        payRecord.setOrderNo(orderNo);
        payRecord.setTradeNo(tradeNo);
        payRecord.setChannel(PayChannel.ALIPAY);
        payRecord.setAmount(notifyAmount);
        payRecord.setStatus(PayStatus.SUCCESS);
        payRecord.setCallbackTime(LocalDateTime.now());
        payRecord.setCallbackContent(JSON.toJSONString(params));
        payMapper.insert(payRecord);

        // 5. 更新订单状态
        orderService.paySuccess(orderNo, tradeNo);

        // 6. 发送支付成功事件（触发发货、积分等后续流程）
        eventPublisher.publishEvent(new PaySuccessEvent(orderNo, payRecord));

        return "success";
    }
}
```

## 6. 库存服务设计

### 6.1 库存扣减方案

库存是电商系统中最容易出问题的环节，核心矛盾是**并发扣减与数据一致性**。

#### 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 数据库行锁 | 简单可靠 | 性能差 | 低并发 |
| 乐观锁 | 无锁性能好 | CAS重试浪费 | 中等并发 |
| Redis预减 | 性能极高 | 需与DB同步 | 高并发 |
| 分布式锁 | 强一致 | 性能有瓶颈 | 中高并发 |

### 6.2 Redis + 数据库双写方案

```java
@Service
public class InventoryService {

    @Autowired
    private StringRedisTemplate redisTemplate;

    @Autowired
    private InventoryMapper inventoryMapper;

    @Autowired
    private RocketMQTemplate rocketMQTemplate;

    private static final String STOCK_KEY = "inventory:stock:";

    /**
     * 库存预减 — Redis原子操作
     * 返回值: >=0 表示扣减成功，<0 表示库存不足
     */
    public boolean deductStock(Long skuId, int quantity) {
        String key = STOCK_KEY + skuId;
        // Lua脚本保证原子性
        String luaScript =
            "local stock = tonumber(redis.call('get', KEYS[1])) " +
            "if stock == nil then return -1 end " +
            "if stock < tonumber(ARGV[1]) then return -2 end " +
            "redis.call('decrby', KEYS[1], ARGV[1]) " +
            "return stock - tonumber(ARGV[1])";

        Long result = redisTemplate.execute(
            new DefaultRedisScript<>(luaScript, Long.class),
            Collections.singletonList(key),
            String.valueOf(quantity)
        );

        if (result != null && result >= 0) {
            // 异步扣减数据库库存
            InventoryDeductMessage msg = new InventoryDeductMessage(skuId, quantity, "ORDER");
            rocketMQTemplate.asyncSend("inventory-deduct-topic", msg, new SendCallback() {
                @Override
                public void onSuccess(SendResult sendResult) {
                    log.info("库存扣减消息发送成功, skuId: {}, quantity: {}", skuId, quantity);
                }
                @Override
                public void onException(Throwable e) {
                    log.error("库存扣减消息发送失败, skuId: {}", skuId, e);
                    // 消息发送失败需要回滚Redis
                    redisTemplate.opsForValue().increment(key, quantity);
                }
            });
            return true;
        }
        return false;
    }

    /**
     * 数据库库存扣减 — 乐观锁
     */
    public boolean deductStockFromDB(Long skuId, int quantity) {
        int retry = 0;
        while (retry < 3) {
            Inventory inventory = inventoryMapper.selectBySkuId(skuId);
            if (inventory == null || inventory.getStock() < quantity) {
                return false;
            }
            // CAS方式更新
            int rows = inventoryMapper.deductWithOptimisticLock(
                skuId, quantity, inventory.getVersion());
            if (rows > 0) {
                return true;
            }
            retry++;
        }
        return false;
    }
}
```

```java
@Service
public class CouponReceiveService {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    @Autowired
    private CouponTemplateMapper templateMapper;

    @Autowired
    private UserCouponMapper userCouponMapper;

    /**
     * 高并发领取优惠券
     * 使用Redis原子操作防止超发
     */
    public void receiveCoupon(Long userId, Long templateId) {
        CouponTemplate template = templateMapper.selectById(templateId);
        if (template == null || template.getStatus() != 1) {
            throw new BizException("优惠券不存在或已下线");
        }

        if (LocalDateTime.now().isBefore(template.getStartTime()) ||
            LocalDateTime.now().isAfter(template.getEndTime())) {
            throw new BizException("优惠券不在领取时间内");
        }

        // Redis原子扣减库存
        String stockKey = "coupon:stock:" + templateId;
        String userKey = "coupon:received:" + templateId + ":" + userId;

        String luaScript =
            "if redis.call('sismember', KEYS[2], ARGV[1]) == 1 then return -1 end " +  // 已领取过
            "local stock = tonumber(redis.call('get', KEYS[1])) " +
            "if stock == nil or stock <= 0 then return 0 end " +  // 库存不足
            "redis.call('decr', KEYS[1]) " +
            "redis.call('sadd', KEYS[2], ARGV[1]) " +  // 记录已领取
            "return 1";

        Long result = redisTemplate.execute(
            new DefaultRedisScript<>(luaScript, Long.class),
            Arrays.asList(stockKey, userKey),
            String.valueOf(userId)
        );

        if (result == null || result == 0) {
            throw new BizException("优惠券已抢完");
        }
        if (result == -1) {
            throw new BizException("您已领取过该优惠券");
        }

        // 异步写入数据库
        UserCoupon userCoupon = new UserCoupon();
        userCoupon.setUserId(userId);
        userCoupon.setTemplateId(templateId);
        userCoupon.setStatus(0);
        userCoupon.setExpireTime(template.getEndTime());
        userCouponMapper.insert(userCoupon);
    }
}
```

### 7.3 优惠券价格计算

```java
/**
 * 优惠券核销与抵扣计算
 */
@Service
public class CouponDeductService {

    public CouponDeductResult deduct(Long couponId, Long userId, BigDecimal orderAmount) {
        UserCoupon coupon = userCouponMapper.selectById(couponId);
        if (coupon == null || coupon.getUserId() != userId) {
            throw new BizException("优惠券不存在");
        }
        if (coupon.getStatus() != 0) {
            throw new BizException("优惠券已使用或已过期");
        }

        CouponTemplate template = templateMapper.selectById(coupon.getTemplateId());

        // 校验使用门槛
        if (orderAmount.compareTo(template.getThreshold()) < 0) {
            throw new BizException("订单金额不满足优惠券使用门槛");
        }

        // 计算抵扣金额
        BigDecimal deductAmount;
        switch (template.getType()) {
            case 1: // 满减券
                deductAmount = template.getFaceValue();
                break;
            case 2: // 折扣券
                deductAmount = orderAmount.multiply(
                    BigDecimal.ONE.subtract(template.getDiscountRate()));
                break;
            case 3: // 无门槛券
                deductAmount = template.getFaceValue();
                break;
            default:
                throw new BizException("不支持的优惠券类型");
        }

        // 抵扣金额不能超过订单金额
        if (deductAmount.compareTo(orderAmount) > 0) {
            deductAmount = orderAmount;
        }

        // 核销优惠券
        coupon.setStatus(1);
        coupon.setUseTime(LocalDateTime.now());
        userCouponMapper.updateById(coupon);

        return new CouponDeductResult(deductAmount, couponId);
    }
}
```

### 9.2 TCC模式实现（库存扣减）

```java
/**
 * TCC模式：Try-Confirm-Cancel
 * 适用于库存扣减这种需要强一致性的场景
 */
@Service
public class InventoryTccService {

    /**
     * Try阶段：冻结库存
     * 将可用库存减少，冻结库存增加
     */
    public boolean tryDeduct(BusinessActionContext context, Long skuId, int quantity) {
        // 幂等检查
        if (isProcessed(context.getXid())) {
            return true;
        }

        int rows = inventoryMapper.freezeStock(skuId, quantity);
        if (rows == 0) {
            return false; // 库存不足
        }
        return true;
    }

    /**
     * Confirm阶段：确认扣减
     * 将冻结库存清零，完成扣减
     */
    public boolean confirmDeduct(BusinessActionContext context) {
        if (isProcessed(context.getXid())) {
            return true;
        }

        Long skuId = (Long) context.getActionContext("skuId");
        int quantity = (int) context.getActionContext("quantity");

        inventoryMapper.confirmDeduct(skuId, quantity);
        markProcessed(context.getXid());
        return true;
    }

    /**
     * Cancel阶段：取消扣减
     * 将冻结库存回滚到可用库存
     */
    public boolean cancelDeduct(BusinessActionContext context) {
        if (isProcessed(context.getXid())) {
            return true;
        }

        Long skuId = (Long) context.getActionContext("skuId");
        int quantity = (int) context.getActionContext("quantity");

        inventoryMapper.unfreezeStock(skuId, quantity);
        markProcessed(context.getXid());
        return true;
    }

    private boolean isProcessed(String xid) {
        return tccLogMapper.countByXid(xid) > 0;
    }

    private void markProcessed(String xid) {
        tccLogMapper.insert(xid);
    }
}
```

```java
    @Autowired
    private RocketMQTemplate rocketMQTemplate;

    /**
     * 在同一个本地事务中写业务数据和消息表
     */
    @Transactional(rollbackFor = Exception.class)
    public void createOrderWithMessage(Order order) {
        // 写订单表
        orderMapper.insert(order);
        // 写本地消息表（与订单在同一个数据库事务中）
        LocalMessage message = new LocalMessage();
        message.setBusinessKey(order.getOrderNo());
        message.setTopic("order-created-topic");
        message.setContent(JSON.toJSONString(order));
        message.setStatus(0); // 待发送
        localMessageMapper.insert(message);
    }

    /**
     * 定时任务扫描未发送的消息
     */
    @Scheduled(fixedRate = 5000)
    public void sendPendingMessages() {
        List<LocalMessage> messages = localMessageMapper.selectPending(100);
        for (LocalMessage msg : messages) {
            try {
                rocketMQTemplate.convertAndSend(msg.getTopic(), msg.getContent());
                localMessageMapper.updateStatus(msg.getId(), 1); // 已发送
            } catch (Exception e) {
                log.error("消息发送失败, id: {}", msg.getId(), e);
                // 重试次数+1，超过阈值告警
                localMessageMapper.incrementRetryCount(msg.getId());
            }
        }
    }
}
```

## 11. 面试题速查

**Q1: 电商系统微服务怎么拆分？拆分粒度如何控制？**

按业务领域拆分，核心服务包括商品、订单、支付、库存、优惠券、用户服务等。粒度控制原则：一个服务对应一个业务领域，能独立部署和扩展。拆太细则RPC调用链过长，拆太粗则失去独立扩展能力。经验法则是：一个团队（5-8人）能负责1-2个服务。

**Q2: 创建订单时如何保证库存、优惠券、订单三个服务的数据一致性？**

采用TCC模式：Try阶段冻结库存和预核销优惠券，Confirm阶段确认扣减，Cancel阶段回滚。如果对一致性要求不是特别强，可以用RocketMQ事务消息 + 本地消息表方案，最终一致性即可。

**Q3: 商品详情页如何扛住高并发？**

多级缓存：Caffeine本地缓存 → Redis分布式缓存 → 数据库。热点商品额外配置：CDN缓存静态资源、页面静态化、热点探测自动缓存预热。防缓存击穿用分布式锁，防缓存穿透用布隆过滤器或空值缓存，防缓存雪崩用随机过期时间。

**Q4: 订单超时取消怎么实现？对比各方案？**

三种方案：① RocketMQ延迟消息（推荐，可靠性高）；② Redis过期Key通知（不可靠，可能丢失）；③ 定时任务扫描（有延迟，数据库压力大）。推荐用延迟消息，30分钟后消费检查订单状态，未支付则取消并回滚库存和优惠券。

**Q5: 支付回调如何保证幂等性？**

通过第三方交易号（trade_no）做幂等键。回调处理前先查支付记录表，已存在且成功的直接返回success。数据库层面加唯一索引兜底。同时校验金额是否一致、订单状态是否合法。

**Q6: 库存扣减如何防超卖？**

Redis Lua脚本原子扣减：先检查库存是否充足，再扣减，整个过程一个Lua脚本完成。Redis扣减成功后异步MQ扣减数据库库存。数据库层面用乐观锁CAS兜底。大促前将热点商品库存预热到Redis。

**Q7: 服务降级策略有哪些？**

读服务降级：返回缓存数据、返回默认值、返回空列表。写服务降级：同步转异步（先存MQ后处理）、拒绝写入并提示稍后重试。非核心功能直接降级关闭（如商品推荐、评论展示）。核心链路（下单、支付）不降级，增加冗余实例保障。

**Q8: 优惠券防超发怎么做？**

Redis原子操作：Lua脚本判断是否已领取过 + 判断库存是否充足 + 扣减库存 + 记录已领取用户，整个过程原子完成。数据库层面用唯一索引(user_id, template_id)防止重复领取。领取后异步写数据库。
