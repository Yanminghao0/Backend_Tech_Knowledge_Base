# 分治算法 (Divide and Conquer Algorithms)

欢迎来到分治算法专题！本目录收集了经典分治算法的详细解析、实现代码和应用指南，帮助你系统掌握这一重要算法设计范式。

## 📚 目录概述
分治算法是一种将复杂问题分解为更小子问题来解决的策略，通常包含三个步骤：
1. **分解**：将原问题分解为多个规模较小的相似子问题
2. **解决**：递归解决各个子问题
3. **合并**：将子问题的解合并为原问题的解

这种策略不仅能高效解决许多复杂问题，还能培养我们的问题拆解能力和递归思维。

## 📈 学习路径
推荐按以下顺序学习：
1. **二分查找**：理解分治的基本思想和边界处理
2. **归并排序**：掌握分治的完整流程（分解-解决-合并）
3. **快速排序**：学习随机化算法和原地划分技术
4. **大整数乘法**：了解分治优化策略和数学建模
5. **最近点对**：探索分治在几何问题中的应用

每个算法都包含详细的原理讲解、多种实现方式对比、复杂度分析和实际应用场景，帮助你从理论到实践全面掌握。
