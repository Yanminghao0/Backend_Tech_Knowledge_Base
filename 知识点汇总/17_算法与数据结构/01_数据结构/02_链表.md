# 链表（Linked List）

> 非连续存储的线性数据结构，通过指针连接节点，支持高效插入删除

---

## 📋 基本信息

- **类型**: 线性数据结构
- **时间复杂度**: 
  - 访问：O(n) - 需要从头遍历
  - 查找：O(n) - 需要遍历
  - 插入：O(1) - 已知位置
  - 删除：O(1) - 已知位置
- **空间复杂度**: O(n) - 需要额外指针空间
- **特点**: 
  - ✅ 插入删除高效
  - ✅ 动态大小
  - ❌ 不支持随机访问
  - ❌ 额外指针开销

---

## 🎯 核心特性

### 1. 节点结构
链表由一系列节点组成，每个节点包含：
- **数据域**：存储数据
- **指针域**：指向下一个（或上一个）节点

### 2. 非连续存储
节点在内存中不连续，通过指针链接

### 3. 动态分配
可以动态地增加或删除节点，无需预分配固定空间

## 💻 代码实现

### 1. 单链表节点定义

```java
/**
 * 单链表节点
 */
class ListNode {
    int val;           // 数据域
    ListNode next;     // 指针域：指向下一个节点
    
    ListNode(int val) {
        this.val = val;
        this.next = null;
    }
    
    ListNode(int val, ListNode next) {
        this.val = val;
        this.next = next;
    }
}
```

### 2. 双链表节点定义

```java
/**
 * 双链表节点
 */
class DoublyListNode {
    int val;
    DoublyListNode prev;  // 指向前一个节点
    DoublyListNode next;  // 指向后一个节点
    
    DoublyListNode(int val) {
        this.val = val;
        this.prev = null;
        this.next = null;
    }
}
```

### 3. 基本操作

#### (1) 头部插入

```java
/**
 * 在链表头部插入节点
 * 时间复杂度：O(1)
 */
public ListNode insertAtHead(ListNode head, int val) {
    ListNode newNode = new ListNode(val);
    newNode.next = head;
    return newNode; // 返回新的头节点
}
```

#### (2) 尾部插入

```java
/**
 * 在链表尾部插入节点
 * 时间复杂度：O(n)
 */
public ListNode insertAtTail(ListNode head, int val) {
    ListNode newNode = new ListNode(val);
    
    if (head == null) {
        return newNode;
    }
    
    ListNode current = head;
    while (current.next != null) {
        current = current.next;
    }
    current.next = newNode;
    return head;
}
```

#### (3) 删除节点

```java
/**
 * 删除值为val的第一个节点
 * 时间复杂度：O(n)
 */
public ListNode deleteNode(ListNode head, int val) {
    // 处理头节点删除
    if (head != null && head.val == val) {
        return head.next;
    }
    
    ListNode current = head;
    while (current != null && current.next != null) {
        if (current.next.val == val) {
            current.next = current.next.next;
            break;
        }
        current = current.next;
    }
    return head;
}
```

#### (4) 遍历链表

```java
/**
 * 遍历打印链表
 * 时间复杂度：O(n)
 */
public void printList(ListNode head) {
    ListNode current = head;
    while (current != null) {
        System.out.print(current.val + " -> ");
        current = current.next;
    }
    System.out.println("null");
}
```

## 📊 经典问题

### 1. 反转链表

#### (1) 迭代法

```java
/**
 * LeetCode 206: 反转链表（迭代）
 * 时间复杂度：O(n)，空间复杂度：O(1)
 */
public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;
    
    while (curr != null) {
        ListNode next = curr.next;  // 保存下一个节点
        curr.next = prev;           // 反转指针
        prev = curr;                // prev前进
        curr = next;                // curr前进
    }
    
    return prev; // prev是新的头节点
}
```

#### (2) 递归法

```java
/**
 * LeetCode 206: 反转链表（递归）
 * 时间复杂度：O(n)，空间复杂度：O(n)
 */
public ListNode reverseList(ListNode head) {
    // 递归终止条件
    if (head == null || head.next == null) {
        return head;
    }
    
    // 递归反转后续链表
    ListNode newHead = reverseList(head.next);
    
    // 反转当前节点
    head.next.next = head;
    head.next = null;
    
    return newHead;
}
```

### 2. 快慢指针

#### (1) 找链表中点

```java
/**
 * LeetCode 876: 链表的中间节点
 * 快指针走2步，慢指针走1步，快指针到末尾时慢指针在中点
 */
public ListNode middleNode(ListNode head) {
    ListNode slow = head;
    ListNode fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;       // 慢指针走1步
        fast = fast.next.next;  // 快指针走2步
    }
    
    return slow; // 慢指针指向中点
}
```

#### (2) 环检测

```java
/**
 * LeetCode 141: 环形链表
 * 判断链表是否有环
 */
public boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) {
        return false;
    }
    
    ListNode slow = head;
    ListNode fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        
        if (slow == fast) {  // 快慢指针相遇，有环
            return true;
        }
    }
    
    return false;
}
```

#### (3) 找环的入口

```java
/**
 * LeetCode 142: 环形链表 II
 * 找到环的入口节点
 */
public ListNode detectCycle(ListNode head) {
    if (head == null || head.next == null) {
        return null;
    }
    
    // 第一步：判断是否有环
    ListNode slow = head;
    ListNode fast = head;
    
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        
        if (slow == fast) {
            // 第二步：找入口
            ListNode ptr = head;
            while (ptr != slow) {
                ptr = ptr.next;
                slow = slow.next;
            }
            return ptr; // 入口节点
        }
    }
    
    return null;
}
```

### 3. 合并链表

```java
/**
 * LeetCode 21: 合并两个有序链表
 */
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0);  // 哨兵节点
    ListNode curr = dummy;
    
    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) {
            curr.next = l1;
            l1 = l1.next;
        } else {
            curr.next = l2;
            l2 = l2.next;
        }
        curr = curr.next;
    }
    
    // 连接剩余节点
    curr.next = (l1 != null) ? l1 : l2;
    
    return dummy.next;
}
```

### 4. 删除倒数第N个节点

```java
/**
 * LeetCode 19: 删除链表的倒数第N个节点
 * 双指针法：让fast先走n步
 */
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    
    ListNode fast = dummy;
    ListNode slow = dummy;
    
    // fast先走n+1步
    for (int i = 0; i <= n; i++) {
        fast = fast.next;
    }
    
    // fast和slow一起走
    while (fast != null) {
        fast = fast.next;
        slow = slow.next;
    }
    
    // slow.next就是要删除的节点
    slow.next = slow.next.next;
    
    return dummy.next;
}
```

### 5. 回文链表

```java
/**
 * LeetCode 234: 回文链表
 * 找中点 + 反转后半部分 + 比较
 */
public boolean isPalindrome(ListNode head) {
    if (head == null || head.next == null) {
        return true;
    }
    
    // 1. 找到中点
    ListNode slow = head;
    ListNode fast = head;
    while (fast.next != null && fast.next.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    
    // 2. 反转后半部分
    ListNode secondHalf = reverseList(slow.next);
    
    // 3. 比较前后两部分
    ListNode p1 = head;
    ListNode p2 = secondHalf;
    while (p2 != null) {
        if (p1.val != p2.val) {
            return false;
        }
        p1 = p1.next;
        p2 = p2.next;
    }
    
    return true;
}
```

## 🔍 应用场景

### 1. Java核心库
- **LinkedList**: 基于双向链表实现
- **HashMap**: 链表法解决哈希冲突（JDK 1.8前）
- **ConcurrentHashMap**: 分段锁 + 链表/红黑树

### 2. 操作系统
- **进程调度**: 就绪队列、等待队列
- **内存管理**: 空闲块链表

### 3. 实际应用
- **LRU缓存**: 双向链表 + 哈希表
- **浏览器历史**: 前进后退功能
- **音乐播放器**: 播放列表

## ⚠️ 注意事项

### 1. 空指针异常

```java
// ❌ 错误：未判空
public void operate(ListNode head) {
    int val = head.val; // 可能NPE
}

// ✅ 正确：先判空
public void operate(ListNode head) {
    if (head == null) {
        return;
    }
    int val = head.val;
}
```

### 2. 使用哨兵节点简化边界处理

```java
// 使用哨兵节点避免特殊处理头节点
ListNode dummy = new ListNode(0);
dummy.next = head;

// 操作dummy.next，最后返回dummy.next
return dummy.next;
```

### 3. 快慢指针判断条件

```java
// 正确的判断条件
while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
}
```

### 4. 链表反转时避免断链

```java
// 必须先保存next节点
ListNode next = curr.next;  // ✅ 先保存
curr.next = prev;           // 再修改
```

## 🎓 最佳实践

### 1. 选择链表 vs 数组

```java
// 频繁插入删除 → 链表
LinkedList<Integer> list = new LinkedList<>();

// 频繁随机访问 → 数组
ArrayList<Integer> list = new ArrayList<>();
```

### 2. 双指针技巧
- **快慢指针**: 找中点、检测环
- **前后指针**: 删除倒数第N个节点
- **对撞指针**: 回文判断

### 3. 哨兵节点技巧
使用哨兵节点简化边界处理，避免特殊判断头节点

### 4. 递归思维
很多链表问题可以用递归优雅解决，但注意栈空间开销
