# 栈（Stack）

> 后进先出（LIFO）的线性数据结构，只能在栈顶进行操作

---

## 📋 基本信息

- **类型**: 线性数据结构
- **时间复杂度**: 
  - 入栈（push）：O(1)
  - 出栈（pop）：O(1)
  - 查看栈顶（peek）：O(1)
  - 搜索（search）：O(n)
- **空间复杂度**: O(n)
- **特点**: 
  - ✅ 后进先出（LIFO - Last In First Out）
  - ✅ 只能在栈顶操作
  - ✅ 插入删除O(1)
  - ❌ 不支持随机访问

---

## 🎯 核心特性

### 1. LIFO原则
后进先出，类似叠盘子，最后放上去的最先拿下来

### 2. 受限的线性表
只能在栈顶进行插入（push）和删除（pop）操作

### 3. 递归的天然数据结构
函数调用栈就是基于栈实现的

## 💻 代码实现

### 1. 数组实现栈

```java
/**
 * 基于数组的栈实现
 */
class ArrayStack {
    private int[] data;
    private int top;      // 栈顶指针
    private int capacity; // 栈容量
    
    public ArrayStack(int capacity) {
        this.capacity = capacity;
        this.data = new int[capacity];
        this.top = -1; // 空栈时top=-1
    }
    
    // 入栈
    public void push(int value) {
        if (isFull()) {
            throw new RuntimeException("栈已满");
        }
        data[++top] = value;
    }
    
    // 出栈
    public int pop() {
        if (isEmpty()) {
            throw new RuntimeException("栈为空");
        }
        return data[top--];
    }
    
    // 查看栈顶
    public int peek() {
        if (isEmpty()) {
            throw new RuntimeException("栈为空");
        }
        return data[top];
    }
    
    // 判断栈空
    public boolean isEmpty() {
        return top == -1;
    }
    
    // 判断栈满
    public boolean isFull() {
        return top == capacity - 1;
    }
    
    // 获取栈大小
    public int size() {
        return top + 1;
    }
}
```

### 2. 链表实现栈

```java
/**
 * 基于链表的栈实现
 */
class LinkedStack {
    private class Node {
        int val;
        Node next;
        Node(int val) {
            this.val = val;
        }
    }
    
    private Node top;  // 栈顶
    private int size;
    
    public LinkedStack() {
        this.top = null;
        this.size = 0;
    }
    
    // 入栈
    public void push(int value) {
        Node newNode = new Node(value);
        newNode.next = top;
        top = newNode;
        size++;
    }
    
    // 出栈
    public int pop() {
        if (isEmpty()) {
            throw new RuntimeException("栈为空");
        }
        int value = top.val;
        top = top.next;
        size--;
        return value;
    }
    
    // 查看栈顶
    public int peek() {
        if (isEmpty()) {
            throw new RuntimeException("栈为空");
        }
        return top.val;
    }
    
    // 判断栈空
    public boolean isEmpty() {
        return top == null;
    }
    
    // 获取栈大小
    public int size() {
        return size;
    }
}
```

## 📊 经典问题

### 1. 括号匹配

```java
/**
 * LeetCode 20: 有效的括号
 * 判断括号字符串是否有效
 */
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    
    for (char c : s.toCharArray()) {
        // 左括号入栈
        if (c == '(' || c == '[' || c == '{') {
            stack.push(c);
        } else {
            // 右括号：检查是否匹配
            if (stack.isEmpty()) {
                return false;
            }
            char top = stack.pop();
            if ((c == ')' && top != '(') ||
                (c == ']' && top != '[') ||
                (c == '}' && top != '{')) {
                return false;
            }
        }
    }
    
    return stack.isEmpty(); // 栈应为空
}
```

### 2. 最小栈

```java
/**
 * LeetCode 155: 最小栈
 * 支持O(1)时间获取最小值的栈
 */
class MinStack {
    private Deque<Integer> dataStack;  // 数据栈
    private Deque<Integer> minStack;   // 最小值栈
    
    public MinStack() {
        dataStack = new ArrayDeque<>();
        minStack = new ArrayDeque<>();
    }
    
    public void push(int val) {
        dataStack.push(val);
        
        // 更新最小值栈
        if (minStack.isEmpty() || val <= minStack.peek()) {
            minStack.push(val);
        }
    }
    
    public void pop() {
        int val = dataStack.pop();
        
        // 如果弹出的是最小值，最小值栈也弹出
        if (!minStack.isEmpty() && val == minStack.peek()) {
            minStack.pop();
        }
    }
    
    public int top() {
        return dataStack.peek();
    }
    
    public int getMin() {
        return minStack.peek();
    }
}
```

### 3. 单调栈

#### (1) 下一个更大元素

```java
/**
 * LeetCode 496: 下一个更大元素 I
 * 找到每个元素右侧第一个比它大的元素
 */
public int[] nextGreaterElement(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    Arrays.fill(result, -1);
    
    Deque<Integer> stack = new ArrayDeque<>(); // 单调递减栈（存索引）
    
    for (int i = 0; i < n; i++) {
        // 当前元素比栈顶大，栈顶元素找到了答案
        while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
            int idx = stack.pop();
            result[idx] = nums[i];
        }
        stack.push(i);
    }
    
    return result;
}
```

#### (2) 柱状图中最大的矩形

```java
/**
 * LeetCode 84: 柱状图中最大的矩形
 * 单调栈经典应用
 */
public int largestRectangleArea(int[] heights) {
    int n = heights.length;
    Deque<Integer> stack = new ArrayDeque<>();
    int maxArea = 0;
    
    for (int i = 0; i <= n; i++) {
        int h = (i == n) ? 0 : heights[i];
        
        // 当前高度小于栈顶，计算栈顶能形成的最大面积
        while (!stack.isEmpty() && h < heights[stack.peek()]) {
            int height = heights[stack.pop()];
            int width = stack.isEmpty() ? i : i - stack.peek() - 1;
            maxArea = Math.max(maxArea, height * width);
        }
        
        stack.push(i);
    }
    
    return maxArea;
}
```

### 4. 表达式求值

```java
/**
 * LeetCode 224: 基本计算器
 * 计算包含 +, -, (, ) 的表达式
 */
public int calculate(String s) {
    Deque<Integer> stack = new ArrayDeque<>();
    int result = 0;
    int number = 0;
    int sign = 1;  // 1表示正号，-1表示负号
    
    for (char c : s.toCharArray()) {
        if (Character.isDigit(c)) {
            number = number * 10 + (c - '0');
        } else if (c == '+') {
            result += sign * number;
            number = 0;
            sign = 1;
        } else if (c == '-') {
            result += sign * number;
            number = 0;
            sign = -1;
        } else if (c == '(') {
            // 遇到左括号，保存当前结果和符号
            stack.push(result);
            stack.push(sign);
            result = 0;
            sign = 1;
        } else if (c == ')') {
            // 遇到右括号，计算括号内结果
            result += sign * number;
            number = 0;
            result *= stack.pop();  // 弹出括号前的符号
            result += stack.pop();  // 加上括号前的结果
        }
    }
    
    result += sign * number;
    return result;
}
```

### 5. 每日温度

```java
/**
 * LeetCode 739: 每日温度
 * 找到下一个更高温度需要等待的天数
 */
public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] result = new int[n];
    Deque<Integer> stack = new ArrayDeque<>(); // 单调栈存储索引
    
    for (int i = 0; i < n; i++) {
        // 当前温度高于栈顶温度
        while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
            int idx = stack.pop();
            result[idx] = i - idx; // 等待天数
        }
        stack.push(i);
    }
    
    return result;
}
```

## 🔍 应用场景

### 1. 程序执行
- **函数调用栈**: 保存函数调用关系
- **递归实现**: 递归本质是栈
- **中断处理**: 保存CPU状态

### 2. 表达式处理
- **表达式求值**: 中缀转后缀
- **括号匹配**: 编译器语法检查
- **计算器实现**: 四则运算

### 3. 浏览器功能
- **前进后退**: 历史记录栈
- **撤销重做**: Undo/Redo栈

### 4. 算法应用
- **深度优先搜索（DFS）**: 显式栈或递归
- **单调栈**: 解决区间最值问题
- **回溯算法**: 保存状态

## ⚠️ 注意事项

### 1. 栈空/栈满检查

```java
// ❌ 错误：未检查栈空
public int pop() {
    return stack.pop(); // 可能抛异常
}

// ✅ 正确：先检查
public int pop() {
    if (isEmpty()) {
        throw new EmptyStackException();
    }
    return stack.pop();
}
```

### 2. Java Stack vs Deque

```java
// ❌ 不推荐：Stack是遗留类，基于Vector（线程安全但慢）
Stack<Integer> stack = new Stack<>();

// ✅ 推荐：使用Deque接口
Deque<Integer> stack = new ArrayDeque<>();
```

### 3. 单调栈的维护

```java
// 单调递增栈：栈底到栈顶递增
while (!stack.isEmpty() && nums[i] < nums[stack.peek()]) {
    stack.pop();
}
stack.push(i);

// 单调递减栈：栈底到栈顶递减
while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
    stack.pop();
}
stack.push(i);
```
