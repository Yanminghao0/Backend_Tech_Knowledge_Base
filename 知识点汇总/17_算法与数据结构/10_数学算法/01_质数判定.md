# 质数判定

质数是指在大于1的自然数中，除了1和它本身以外不再有其他因数的自然数。质数判定是数论中的基础问题，在密码学、哈希函数、随机数生成等领域有重要应用。

---

## 📋 基本信息
- **定义**：只能被1和自身整除的大于1的整数
- **别名**：素数
- **相关概念**：合数（非质数的大于1的整数）、孪生素数、梅森素数
- **历史背景**：欧几里得在《几何原本》中首次证明了无穷多个质数的存在
- **质数定理**：小于n的质数个数 π(n) ≈ n / ln(n)

## 🎯 算法原理

### 1. 试除法
- **基本思想**：检查n是否能被2到√n之间的任何整数整除
- **步骤**：
  1. 如果n ≤ 1，不是质数
  2. 如果n ≤ 3，是质数
  3. 如果n是偶数或能被3整除，不是质数
  4. 检查从5到√n的所有数，步长为6（因为大于3的质数都形如6k±1）

**为什么只需检查到√n？**
如果n = a × b且a ≤ b，则a ≤ √n。因此如果n有大于√n的因子，必然有对应的小于√n的因子，检查到√n即可。

**为什么6k±1优化有效？**
任何整数可表示为 6k, 6k±1, 6k±2, 6k+3：
- 6k = 2 × 3k（偶数，已被2排除）
- 6k+2 = 2 × (3k+1)（偶数）
- 6k+3 = 3 × (2k+1)（被3排除）
- 6k+4 = 2 × (3k+2)（偶数）
- 只有 6k±1 可能是质数

### 2. 埃拉托斯特尼筛法（Sieve of Eratosthenes）
- **基本思想**：创建一个布尔数组，标记从2开始的所有倍数为非质数
- **步骤**：
  1. 创建长度为n+1的数组is_prime，初始化为True
  2. 将is_prime[0]和is_prime[1]设为False
  3. 对于每个i从2到√n，如果is_prime[i]为True，则将i的所有倍数标记为False
  4. 剩余为True的索引即为质数

**为什么从i²开始标记？**
对于质数i，比i小的倍数 2i, 3i, ..., (i-1)×i 已经被更小的质数标记过了。例如当i=5时，10=2×5已被2标记，15=3×5已被3标记，因此从25=5²开始即可。

### 3. 米勒-拉宾素性测试（Miller-Rabin Primality Test）
- **基本思想**：基于费马小定理的概率测试，可用于大整数的素性判定
- **步骤**：
  1. 写出n-1 = d×2^s（其中d为奇数）
  2. 选择随机整数a（1 < a < n）
  3. 计算x = a^d mod n
  4. 如果x == 1或x == n-1，可能是质数
  5. 否则重复平方x s-1次，如果始终不等于n-1，则不是质数
  6. 多次测试提高准确性

**数学基础 — 费马小定理**：若p为质数且gcd(a,p)=1，则 a^(p-1) ≡ 1 (mod p)。逆否命题：若a^(n-1) ≢ 1 (mod n)，则n必为合数。

**确定性测试的底数选择**：
对于 32位整数，只需测试 a ∈ {2, 7, 61} 即可确定性判定。
对于 64位整数，只需测试 a ∈ {2, 325, 9375, 28178, 450775, 9780504, 1795265022}。

## 💻 代码实现

### 试除法实现
```python
import math

def is_prime_trial_division(n):
    if n <= 1:
        return False
    if n <= 3:
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    w = 2
    while i * i <= n:
        if n % i == 0:
            return False
        i += w
        w = 6 - w  # 切换步长 2 和 4
    return True
```

### Java试除法实现
```java
public class PrimeCheck {
    // 基本试除法 O(√n)
    public static boolean isPrime(long n) {
        if (n <= 1) return false;
        if (n <= 3) return true;
        if (n % 2 == 0 || n % 3 == 0) return false;
        
        for (long i = 5; i * i <= n; i += 6) {
            if (n % i == 0 || n % (i + 2) == 0) {
                return false;
            }
        }
        return true;
    }
}
```

### 埃拉托斯特尼筛法实现
```python
def sieve_of_eratosthenes(n):
    if n < 2:
        return []
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(math.sqrt(n)) + 1):
        if is_prime[i]:
            # 标记i的倍数为非质数
            for j in range(i*i, n+1, i):
                is_prime[j] = False
    return [i for i, prime in enumerate(is_prime) if prime]
```

### 线性筛法（欧拉筛）— O(n)时间复杂度
```python
def euler_sieve(n):
    if n < 2:
        return []
    is_prime = [True] * (n + 1)
    primes = []
    is_prime[0] = is_prime[1] = False
    
    for i in range(2, n + 1):
        if is_prime[i]:
            primes.append(i)
        # 每个合数只被其最小质因子筛掉一次
        for p in primes:
            if p * i > n:
                break
            is_prime[p * i] = False
            if i % p == 0:
                break  # 保证每个合数只被筛一次
    return primes
```

**线性筛原理**：每个合数只被其最小质因子筛掉一次。当 `i % p == 0` 时，p是i的最小质因子，因此 p×i 的最小质因子也是p。对于更大的质因子p'，p'×i 的最小质因子是p而非p'，不应在此处筛除，所以break。

### 米勒-拉宾素性测试实现
```python
import random

def is_prime_miller_rabin(n, k=5):
    if n <= 1:
        return False
    elif n <= 3:
        return True
    elif n % 2 == 0:
        return False
    
    # 写成n-1 = d * 2^s
    d = n - 1
    s = 0
    while d % 2 == 0:
        d //= 2
        s += 1
    
    # 进行k次测试
    for _ in range(k):
        a = random.randint(2, min(n - 2, 1 << 20))
        x = pow(a, d, n)
        if x == 1 or x == n - 1:
            continue
        for __ in range(s - 1):
            x = pow(x, 2, n)
            if x == n - 1:
                break
        else:
            # 如果没有通过测试
            return False
    return True
```

### 确定性米勒-拉宾测试（64位整数）
```java
public class MillerRabin {
    // 确定性测试底数集合，适用于 n < 2^64
    private static final long[] BASES = {2, 325, 9375, 28178, 450775, 9780504, 1795265022};
    
    public static boolean isPrime(long n) {
        if (n < 2) return false;
        if (n == 2 || n == 3) return true;
        if (n % 2 == 0) return false;
        
        // 分解 n-1 = d * 2^s
        long d = n - 1;
        int s = 0;
        while ((d & 1) == 0) {
            d >>= 1;
            s++;
        }
        
        for (long a : BASES) {
            if (a % n == 0) continue;
            long x = modPow(a, d, n);
            if (x == 1 || x == n - 1) continue;
            boolean composite = true;
            for (int i = 0; i < s - 1; i++) {
                x = modPow(x, 2, n);
                if (x == n - 1) {
                    composite = false;
                    break;
                }
            }
            if (composite) return false;
        }
        return true;
    }
    
    // 快速幂取模
    private static long modPow(long base, long exp, long mod) {
        long result = 1;
        base %= mod;
        while (exp > 0) {
            if ((exp & 1) == 1) {
                result = multiply(result, base, mod);
            }
            base = multiply(base, base, mod);
            exp >>= 1;
        }
        return result;
    }
    
    // 防溢出乘法
    private static long multiply(long a, long b, long mod) {
        return java.math.BigInteger.valueOf(a)
            .multiply(java.math.BigInteger.valueOf(b))
            .mod(java.math.BigInteger.valueOf(mod))
            .longValue();
    }
}
```

## 📊 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|------------|------------|------|
| 试除法（基本）| O(√n) | O(1) | 简单直观，适合小整数 |
| 试除法（6k±1优化）| O(√n / 3) | O(1) | 减少2/3的检查次数 |
| 埃拉托斯特尼筛法 | O(n log log n) | O(n) | 适合一次性生成多个小质数 |
| 线性筛（欧拉筛）| O(n) | O(n) | 每个合数只筛一次，最优筛法 |
| 米勒-拉宾测试 | O(k log³n) | O(1) | 概率算法，适合大整数，k为测试次数 |
| 确定性米勒-拉宾 | O(log³n) | O(1) | 64位以内确定性判定 |

### 各算法适用范围
| 数据规模 | 推荐算法 |
|---------|---------|
| n < 10^12 | 试除法（6k±1优化）|
| 批量生成 n < 10^7 | 埃拉托斯特尼筛法 |
| 批量生成 n < 10^8 | 线性筛（欧拉筛）|
| n < 10^18 | 确定性米勒-拉宾 |

## 🔍 应用场景
- **密码学**：RSA加密算法中的密钥生成需要大质数（通常512位或1024位）
- **哈希函数**：质数用于减少哈希冲突（如HashMap的容量选择质数）
- **随机数生成**：质数模数确保更好的分布特性
- **算法竞赛**：数论问题的基础组件（如欧拉函数、莫比乌斯函数）
- **计算机安全**：数字签名和认证系统（DSA、ElGamal）
- **编码理论**：Reed-Solomon纠错编码使用质数域运算

```python
import random

def generate_rsa_keypair(bit_length=1024):
    # 生成两个大质数p和q
    p = generate_large_prime(bit_length)
    q = generate_large_prime(bit_length)
    while p == q:
        q = generate_large_prime(bit_length)
    
    n = p * q  # 模数
    phi = (p - 1) * (q - 1)  # 欧拉函数
    
    # 选择公钥指数e（通常为65537）
    e = 65537
    
    # 计算私钥指数d
    d = modular_inverse(e, phi)
    
    return (e, n), (d, n)  # 公钥, 私钥

def generate_large_prime(bits):
    while True:
        candidate = random.getrandbits(bits) | (1 << (bits - 1)) | 1
        if isprime(candidate):
            return candidate
```

## ⚠️ 边界条件分析
1. **n = 0或n = 1**：不是质数（特殊情况）
2. **n = 2**：最小的质数，也是唯一的偶质数
3. **n = 3**：质数
4. **负数**：不是质数
5. **大整数溢出**：`i * i <= n` 中 `i * i` 可能溢出，需用大整数或改为 `i <= n / i`
6. **米勒-拉宾的特殊情况**：n=2, 3需要单独处理
7. **筛法的空间限制**：n过大时数组无法分配，需使用分段筛法

### 分段筛法（处理超大范围）
```python
def segmented_sieve(low, high):
    """筛选[low, high]范围内的质数"""
    if high < 2:
        return []
    
    # 先筛出√high以内的质数
    limit = int(math.sqrt(high)) + 1
    base_primes = sieve_of_eratosthenes(limit)
    
    # 分段筛
    size = high - low + 1
    is_prime = [True] * size
    if low == 0:
        is_prime[0] = is_prime[1] = False if high >= 1 else is_prime[0]
    elif low == 1:
        is_prime[0] = False
    
    for p in base_primes:
        # 找到第一个>=low的p的倍数
        start = max(p * p, ((low + p - 1) // p) * p)
        for j in range(start, high + 1, p):
            is_prime[j - low] = False
    
    return [low + i for i, prime in enumerate(is_prime) if prime]
```

## 📝 面试常见问题

**Q1: 为什么试除法只需要检查到√n？**
> 如果n = a × b（a ≤ b），则a² ≤ a×b = n，即a ≤ √n。因此n的因子中至少有一个 ≤ √n，检查到√n即可判断n是否为质数。

**Q2: 米勒-拉宾测试的错误率是多少？**
> 单次测试的错误率至多为 1/4。进行k次独立测试后，错误率至多为 (1/4)^k。k=10时错误率小于 10^-6，k=20时小于 10^-12。

**Q3: 埃拉托斯特尼筛法为什么时间复杂度是O(n log log n)？**
> 筛法的总操作数为 n/2 + n/3 + n/5 + n/7 + ... = n × Σ(1/p)，其中p遍历所有≤√n的质数。根据梅尔滕斯定理，Σ(1/p) ≈ ln ln n，因此总复杂度为 O(n log log n)。

**Q4: 线性筛如何保证每个合数只被筛一次？**
> 每个合数只被其最小质因子筛掉。当遍历到i时，用已知质数p去筛p×i。当 `i % p == 0` 时，说明p是i的最小质因子，此时p也是p×i的最小质因子，筛完后break，防止更大的质数重复筛同一个合数。

## 🎓 最佳实践
- 小整数判定：优先使用试除法（6k±1优化）
- 生成一定范围内所有质数：使用埃拉托斯特尼筛法
- 大整数判定（>10^18）：使用米勒-拉宾测试，建议k=5以上
- 64位以内确定性判定：使用确定性米勒-拉宾（固定底数集合）
- 安全关键应用：使用经过验证的质数库，如GMP、OpenSSL
- 算法优化：结合6k±1优化、位运算等技巧提高性能
- 超大范围筛质数：使用分段筛法控制内存
