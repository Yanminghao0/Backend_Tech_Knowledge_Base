# 快速幂

快速幂（Exponentiation by Squaring）是一种高效计算幂运算的算法，通过将指数分解为二进制形式，将时间复杂度从O(n)降低到O(log n)，广泛应用于密码学、数值计算和算法优化等领域。

---

## 📋 基本信息
- **定义**：通过指数的二进制分解，使用平方和乘法快速计算a^b的算法
- **别名**：平方求幂法、二进制 exponentiation
- **核心思想**：a^(2k) = (a^k)^2，a^(2k+1) = a^(2k) × a
- **数学性质**：任何正整数都可以表示为2的幂次之和，即b = 2^k1 + 2^k2 + ... + 2^kn

## 🎯 算法原理

### 1. 基本快速幂算法
- **基本思想**：将指数b表示为二进制，对每一位为1的位置累乘当前底数的2^i次幂
- **步骤**：
  1. 初始化结果result = 1，当前底数base = a
  2. 将指数b转换为二进制，从最低位开始处理
  3. 如果当前位为1，将result乘以base
  4. 将base平方（base = base^2），指数右移一位（b = b >> 1）
  5. 重复步骤3-4直到指数为0

### 2. 模块化快速幂
- **基本思想**：在快速幂计算过程中同步进行模运算，避免数值溢出
- **核心公式**：(a × b) mod m = [(a mod m) × (b mod m)] mod m
- **应用场景**：密码学中的RSA算法、模运算相关问题

### 3. 递归快速幂
- **基本思想**：通过递归将指数分解为偶数和奇数两种情况处理
- **递归公式**：
  - 若b为偶数：a^b = (a^(b/2))^2
  - 若b为奇数：a^b = a × (a^((b-1)/2))^2
  - 边界条件：a^0 = 1

## 💻 代码实现

### 迭代版快速幂
```python
def fast_power_iterative(base, exponent):
    result = 1
    while exponent > 0:
        # 如果当前位为1，则乘以当前底数
        if exponent % 2 == 1:
            result *= base
        # 底数平方，指数右移一位
        base *= base
        exponent = exponent // 2
    return result
```

### 模块化快速幂
```python
def modular_exponentiation(base, exponent, mod):
    if mod == 1:
        return 0
    result = 1
    base = base % mod
    while exponent > 0:
        if exponent % 2 == 1:
            result = (result * base) % mod
        exponent = exponent // 2
        base = (base * base) % mod
    return result
```

### 递归版快速幂
```python
def fast_power_recursive(base, exponent):
    if exponent == 0:
        return 1
    elif exponent % 2 == 0:
        half = fast_power_recursive(base, exponent // 2)
        return half * half
    else:
        half = fast_power_recursive(base, (exponent - 1) // 2)
        return base * half * half
```

### 处理负指数的快速幂
```python
def fast_power_with_negative(base, exponent):
    if exponent < 0:
        return 1 / fast_power_iterative(base, -exponent)
    return fast_power_iterative(base, exponent)
```

## 📊 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|------------|------------|------|
| 迭代版快速幂 | O(log n) | O(1) | 效率高，无递归栈开销 |
| 递归版快速幂 | O(log n) | O(log n) | 简洁但有递归栈开销 |
| 模块化快速幂 | O(log n) | O(1) | 适合大整数运算，防止溢出 |

## 🔍 应用场景
- **密码学**：RSA加密算法中的密钥生成和加密解密过程
- **数值计算**：科学计算中的大指数运算
- **算法优化**：矩阵快速幂求解线性递推关系（如斐波那契数列）
- **数据校验**：模运算在哈希和校验和计算中的应用
- **游戏开发**：概率计算和数值模拟中的指数增长模型

## ⚠️ 注意事项
- 处理负数指数时需特别处理，可能涉及浮点数运算
- 整数溢出问题：大指数运算可能超出语言的整数表示范围
- 浮点数精度：非常大或非常小的结果可能导致精度损失
- 底数为0且指数为0时无意义，需特殊处理
- 模块化快速幂中，模为1时结果恒为0

## 🎓 最佳实践
- **性能优先**：优先选择迭代实现，避免递归栈开销
- **大整数处理**：使用模块化快速幂防止溢出
- **Python环境**：对于超大数计算，利用Python的无限精度整数特性
- **数值稳定性**：处理浮点数时考虑使用对数转换或分治策略
- **矩阵快速幂**：结合快速幂思想优化矩阵幂运算，用于求解线性递推

```python
# 矩阵快速幂示例（求解斐波那契数列）
def matrix_mult(a, b):
    return [
        [a[0][0]*b[0][0] + a[0][1]*b[1][0], a[0][0]*b[0][1] + a[0][1]*b[1][1]],
        [a[1][0]*b[0][0] + a[1][1]*b[1][0], a[1][0]*b[0][1] + a[1][1]*b[1][1]]
    ]

def matrix_pow(mat, power):
    result = [[1, 0], [0, 1]]  # 单位矩阵
    while power > 0:
        if power % 2 == 1:
            result = matrix_mult(result, mat)
        mat = matrix_mult(mat, mat)
        power //= 2
    return result

def fibonacci(n):
    if n <= 0:
        return 0
    mat = [[1, 1], [1, 0]]
    return matrix_pow(mat, n-1)[0][0]
```
