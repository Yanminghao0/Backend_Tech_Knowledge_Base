# 最大公约数

最大公约数（GCD）是指两个或多个整数共有约数中最大的一个，是数论中的基础概念，在分数化简、模运算、密码学等领域有广泛应用。

---

## 📋 基本信息
- **定义**：两个或多个整数共有约数中最大的一个
- **英文缩写**：GCD (Greatest Common Divisor)
- **相关概念**：最小公倍数(LCM)、互质数(最大公约数为1的两个数)
- **数学性质**：GCD(a,b) = GCD(b,a mod b)，GCD(a,b) × LCM(a,b) = |a×b|
- **表示法**：GCD(a,b) 或 (a,b)，如 GCD(12,18) = 6

## 🎯 算法原理

### 1. 欧几里得算法（辗转相除法）
- **基本思想**：基于定理GCD(a,b) = GCD(b,a mod b)，通过反复取余将问题规模缩小
- **步骤**：
  1. 若b为0，则GCD为a
  2. 否则计算a除以b的余数r
  3. 将问题转化为求GCD(b,r)
  4. 重复步骤1-3直到余数为0

**数学证明**：
设 d = GCD(a,b)，则 d|a 且 d|b。
因为 a = bq + r（r = a mod b），所以 r = a - bq。
由于 d|a 且 d|b，故 d|(a - bq)，即 d|r。
因此 d 是 b 和 r 的公约数，即 GCD(a,b) = GCD(b, a mod b)。

**示例演算**：
```
GCD(48, 18):
  48 = 18 × 2 + 12  →  GCD(18, 12)
  18 = 12 × 1 + 6   →  GCD(12, 6)
  12 = 6 × 2 + 0    →  GCD(6, 0) = 6

所以 GCD(48, 18) = 6
```

### 2. 扩展欧几里得算法
- **基本思想**：在求GCD(a,b)的同时，找到整数x和y使得 ax + by = GCD(a,b)
- **步骤**：
  1. 当b=0时，x=1,y=0，返回GCD=a
  2. 否则递归计算GCD(b,a mod b)
  3. 回代求解x和y：x=y', y=x'-(a//b)y'
  4. 其中(x',y')是上一步递归得到的解

**数学基础 — 贝祖定理（Bézout's Identity）**：
对于任意整数a和b，存在整数x和y，使得 ax + by = GCD(a,b)。

**示例演算**：
```
求  ax + by = GCD(48, 18) = 6 的解

递归过程:
  GCD(48, 18): 48 = 18×2 + 12
  GCD(18, 12): 18 = 12×1 + 6
  GCD(12, 6):  12 = 6×2 + 0
  GCD(6, 0):   返回 (6, 1, 0)

回代过程:
  GCD(12, 6):  x=0, y=1-12//6×0=1   → (6, 0, 1)
  GCD(18, 12): x=1, y=0-18//12×1=-1  → (6, 1, -1)
  GCD(48, 18): x=-1, y=1-48//18×(-1)=1+2=3  → (6, -1, 3)

验证: 48×(-1) + 18×3 = -48 + 54 = 6 ✓
通解: x = -1 + k×(18/6) = -1 + 3k
      y = 3 - k×(48/6) = 3 - 8k  (k为任意整数)
```

### 3. 更相减损术（中国传统算法）
- **基本思想**：基于定理GCD(a,b) = GCD(a-b,b) (a > b)
- **步骤**：
  1. 若a=b，则GCD为a
  2. 若a > b，则a = a - b
  3. 若b > a，则b = b - a
  4. 重复步骤1-3直到a=b

**出处**：《九章算术》（公元1世纪），中国古代数学经典。

### 4. 二进制GCD算法（Stein算法）
- **基本思想**：利用位运算加速GCD计算，避免取模运算
- **步骤**：
  1. 若a=0，返回b；若b=0，返回a
  2. 若a和b均为偶数：GCD(a,b) = 2 × GCD(a/2, b/2)
  3. 若a为偶数b为奇数：GCD(a,b) = GCD(a/2, b)
  4. 若a为奇数b为偶数：GCD(a,b) = GCD(a, b/2)
  5. 若a和b均为奇数：GCD(a,b) = GCD(|a-b|/2, min(a,b))

## 💻 代码实现

### 欧几里得算法（迭代版）
```python
def gcd_euclidean(a, b):
    while b != 0:
        a, b = b, a % b
    return a
```

### 欧几里得算法（递归版）
```python
def gcd_recursive(a, b):
    if b == 0:
        return a
    return gcd_recursive(b, a % b)
```

### Java实现
```java
public class GCD {
    // 迭代版
    public static long gcd(long a, long b) {
        while (b != 0) {
            long temp = b;
            b = a % b;
            a = temp;
        }
        return Math.abs(a);  // 处理负数
    }
    
    // 递归版
    public static long gcdRecursive(long a, long b) {
        return b == 0 ? Math.abs(a) : gcdRecursive(b, a % b);
    }
}
```

### 扩展欧几里得算法
```python
def extended_gcd(a, b):
    """返回 (gcd, x, y) 使得 a*x + b*y = gcd"""
    if b == 0:
        return (a, 1, 0)
    else:
        g, x1, y1 = extended_gcd(b, a % b)
        x = y1
        y = x1 - (a // b) * y1
        return (g, x, y)

# 迭代版扩展欧几里得
def extended_gcd_iterative(a, b):
    old_r, r = a, b
    old_s, s = 1, 0
    old_t, t = 0, 1
    while r != 0:
        quotient = old_r // r
        old_r, r = r, old_r - quotient * r
        old_s, s = s, old_s - quotient * s
        old_t, t = t, old_t - quotient * t
    return (old_r, old_s, old_t)  # (gcd, x, y)
```

### Java扩展欧几里得算法
```java
public static long[] extGCD(long a, long b) {
    if (b == 0) {
        return new long[]{a, 1, 0};
    }
    long[] result = extGCD(b, a % b);
    long g = result[0];
    long x = result[2];
    long y = result[1] - (a / b) * result[2];
    return new long[]{g, x, y};
}
```

### 更相减损术
```python
def gcd_subtraction(a, b):
    while a != b:
        if a > b:
            a -= b
        else:
            b -= a
    return a
```

### 二进制GCD算法（Stein算法）
```python
def binary_gcd(a, b):
    if a == 0:
        return b
    if b == 0:
        return a
    
    # 找到公共的2的幂因子
    shift = 0
    while ((a | b) & 1) == 0:
        a >>= 1
        b >>= 1
        shift += 1
    
    # 去掉a中剩余的2的因子
    while (a & 1) == 0:
        a >>= 1
    
    while b != 0:
        # 去掉b中剩余的2的因子
        while (b & 1) == 0:
            b >>= 1
        # 保证a <= b
        if a > b:
            a, b = b, a
        b -= a
    
    return a << shift
```

### 内置函数实现（Python）
```python
import math

def gcd_builtin(a, b):
    return math.gcd(a, b)  # Python 3.5+

# 最小公倍数计算
def lcm(a, b):
    return a * b // gcd_euclidean(a, b)

# Python 3.9+ 直接使用 math.lcm
# math.lcm(a, b)
```

### 多个数的GCD
```python
from functools import reduce
import math

def gcd_multiple(numbers):
    return reduce(math.gcd, numbers)

def lcm_multiple(numbers):
    return reduce(lambda a, b: a * b // math.gcd(a, b), numbers)
```

## 📊 复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|------------|------------|------|
| 欧几里得算法（迭代）| O(log min(a,b)) | O(1) | 效率最高，通用首选 |
| 欧几里得算法（递归）| O(log min(a,b)) | O(log min(a,b)) | 代码简洁，有栈开销 |
| 扩展欧几里得算法 | O(log min(a,b)) | O(log min(a,b)) | 同时求解贝祖等式 |
| 更相减损术 | O(max(a,b)) | O(1) | 适合a,b相差较小的情况 |
| 二进制GCD（Stein）| O(log²max(a,b)) | O(1) | 无除法运算，适合硬件实现 |

### 欧几里得算法复杂度推导

欧几里得算法的迭代次数由拉梅定理（Lamé's Theorem）给出：
> 如果欧几里得算法需要n步计算GCD(a,b)，且a > b > 0，则 b ≥ F(n+1)，其中F(n)是斐波那契数列。

由于斐波那契数列增长速度为 φ^n（φ ≈ 1.618，黄金比例），因此迭代次数至多为 O(log_φ b) = O(log b)。

**最坏情况**：连续的斐波那契数对，如 GCD(F(n+1), F(n)) 需要 n 次迭代。

```
GCD(13, 8):  (F7, F6)
  13 = 8×1 + 5   → GCD(8, 5)
  8 = 5×1 + 3    → GCD(5, 3)
  5 = 3×1 + 2    → GCD(3, 2)
  3 = 2×1 + 1    → GCD(2, 1)
  2 = 1×2 + 0    → GCD(1, 0) = 1
共5次迭代（最多）
```

## 🔍 应用场景
- **分数运算**：化简分数（分子分母同除以GCD）
- **模运算**：求解线性同余方程 ax ≡ b (mod m)
- **密码学**：RSA加密算法中的密钥生成
- **工程测量**：等比例缩放、最大公约数长度单位
- **数据压缩**：JPEG压缩中的离散余弦变换
- **历法计算**：计算回归年和朔望月的最小公倍数
- **信号处理**：采样率转换中的因子计算

### 应用：求解模逆元
```python
def mod_inverse(a, m):
    """求 a 在模 m 下的逆元，即 a*x ≡ 1 (mod m)"""
    g, x, _ = extended_gcd(a, m)
    if g != 1:
        return None  # 逆元不存在，a和m不互质
    return x % m  # 确保结果为正数

# 示例: 求 3 在模 11 下的逆元
# 3 * 4 = 12 ≡ 1 (mod 11)，所以逆元为4
print(mod_inverse(3, 11))  # 输出: 4
```

### 应用：求解线性同余方程
```python
def solve_linear_congruence(a, b, m):
    """求解 ax ≡ b (mod m)"""
    g, x, _ = extended_gcd(a, m)
    if b % g != 0:
        return None  # 无解
    
    # 特解
    m_reduced = m // g
    x0 = (x * (b // g)) % m
    
    # 所有解（模m下共g个）
    solutions = [(x0 + k * m_reduced) % m for k in range(g)]
    return sorted(solutions)

# 示例: 求解 6x ≡ 4 (mod 10)
# GCD(6,10)=2, 4%2==0, 有解
# 解为 x ≡ 4 (mod 5), 即 x ∈ {4, 9} (mod 10)
print(solve_linear_congruence(6, 4, 10))  # 输出: [4, 9]
```

### 应用：中国剩余定理（CRT）
```python
def chinese_remainder_theorem(remainders, moduli):
    """中国剩余定理: 求解同余方程组 x ≡ r_i (mod m_i)"""
    from functools import reduce
    
    M = reduce(lambda a, b: a * b, moduli)
    x = 0
    for r, m in zip(remainders, moduli):
        Mi = M // m
        yi = mod_inverse(Mi, m)
        x = (x + r * Mi * yi) % M
    return x

# 示例: x ≡ 2 (mod 3), x ≡ 3 (mod 5), x ≡ 2 (mod 7)
# 解为 x ≡ 23 (mod 105)
print(chinese_remainder_theorem([2, 3, 2], [3, 5, 7]))  # 输出: 23
```

## ⚠️ 边界条件分析
1. **零值处理**：GCD(a, 0) = |a|，GCD(0, 0) = 0（定义）
2. **负数处理**：通常取绝对值后计算，GCD(-12, 18) = GCD(12, 18) = 6
3. **a < b**：欧几里得算法第一次迭代自动交换（a%b = a 当 a < b 时）
4. **大数溢出**：计算LCM时 `a * b` 可能溢出，应先除后乘：`lcm(a,b) = a // gcd(a,b) * b`
5. **扩展GCD的解不唯一**：通解为 x = x0 + k×(b/d), y = y0 - k×(a/d)（d为GCD，k为任意整数）
6. **Python的math.gcd**：返回非负整数，Python 3.9+支持负数输入

## 📝 面试常见问题

**Q1: 为什么欧几里得算法的时间复杂度是O(log min(a,b))？**
> 由拉梅定理，算法的迭代次数不超过斐波那契数列中小于b的最大项的索引，而斐波那契数列指数增长（φ^n），因此迭代次数为O(log b)。

**Q2: 扩展欧几里得算法有什么用？**
> 1. 求解模逆元（RSA密钥计算的核心步骤）
> 2. 求解线性同余方程 ax ≡ b (mod m)
> 3. 求解丢番图方程 ax + by = c
> 4. 中国剩余定理（CRT）的实现
> 5. RSA中计算私钥指数 d = e^(-1) mod φ(n)

**Q3: 更相减损术和欧几里得算法哪个更优？**
> 欧几里得算法更优。更相减损术在最坏情况下（如GCD(n, 1)）需要O(n)次减法，而欧几里得算法只需O(log n)次取模。但二进制GCD结合了更相减损术的思想和位运算，在某些场景下比欧几里得算法更快。

**Q4: 如何处理大数的GCD计算？**
> - Python原生支持大整数，直接使用math.gcd即可
> - C++/Java需要使用大整数库（如GMP、BigInteger）
> - 二进制GCD算法避免取模运算，适合硬件实现
> - 对于特大的数，可以使用Lehmer算法（基于高精度取模的优化）

## 🎓 最佳实践
- **性能优先**：选择欧几里得算法（迭代实现），避免递归栈开销
- **求解方程**：使用扩展欧几里得算法
- **Python环境**：优先使用math.gcd（经过优化的C实现）
- **大数计算**：考虑使用内置函数或专门的数学库（如gmpy2）
- **多个数GCD**：使用reduce函数结合gcd函数
- **硬件优化场景**：使用二进制GCD（Stein算法），避免昂贵的除法/取模运算
- **计算LCM时防溢出**：先除后乘 `lcm(a,b) = a // gcd(a,b) * b`
