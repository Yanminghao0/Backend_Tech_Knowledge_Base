# 数组（Array）

> 最基础的线性数据结构，元素在内存中连续存储，支持随机访问

---

## 📋 基本信息

- **类型**: 线性数据结构
- **时间复杂度**: 
  - 访问：O(1) - 通过索引直接访问
  - 查找：O(n) - 需要遍历
  - 插入：O(n) - 需要移动元素
  - 删除：O(n) - 需要移动元素
- **空间复杂度**: O(n)
- **特点**: 
  - ✅ 随机访问速度快
  - ✅ 缓存友好（连续内存）
  - ❌ 大小固定（静态数组）
  - ❌ 插入删除效率低

---

## 🎯 核心特性

### 1. 连续内存存储
数组元素在内存中连续排列，地址计算公式：
```
address(arr[i]) = base_address + i × element_size
```

### 2. 随机访问
通过索引可以 O(1) 访问任意元素

### 3. 缓存友好
连续存储利于CPU缓存，遍历性能优异

## 💻 常见操作

### 1. 初始化

```java
// 静态初始化
int[] arr1 = {1, 2, 3, 4, 5};

// 动态初始化
int[] arr2 = new int[5];

// 动态数组（可变长度）
List<Integer> list = new ArrayList<>();
```

### 2. 遍历

```java
// 方式1：for循环
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// 方式2：增强for循环
for (int num : arr) {
    System.out.println(num);
}

// 方式3：Stream API（Java 8+）
Arrays.stream(arr).forEach(System.out::println);
```

### 3. 插入元素

```java
/**
 * 在指定位置插入元素（需要移动后续元素）
 * 时间复杂度：O(n)
 */
public void insert(int[] arr, int pos, int value, int size) {
    // 从后往前移动元素
    for (int i = size - 1; i >= pos; i--) {
        arr[i + 1] = arr[i];
    }
    arr[pos] = value;
}
```

### 4. 删除元素

```java
/**
 * 删除指定位置的元素
 * 时间复杂度：O(n)
 */
public void delete(int[] arr, int pos, int size) {
    // 从前往后移动元素
    for (int i = pos; i < size - 1; i++) {
        arr[i] = arr[i + 1];
    }
}
```

## 📊 经典问题

### 1. 双指针技巧

#### (1) 对撞指针 - 两数之和 II

```java
/**
 * LeetCode 167: 两数之和 II（有序数组）
 * 给定有序数组，找两个数使其和为目标值
 */
public int[] twoSum(int[] numbers, int target) {
    int left = 0, right = numbers.length - 1;
    
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) {
            return new int[]{left + 1, right + 1}; // 题目要求索引从1开始
        } else if (sum < target) {
            left++;  // 和太小，左指针右移
        } else {
            right--; // 和太大，右指针左移
        }
    }
    return new int[]{-1, -1};
}
```

#### (2) 快慢指针 - 移除元素

```java
/**
 * LeetCode 27: 移除元素
 * 原地移除数组中值为val的元素
 */
public int removeElement(int[] nums, int val) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        if (nums[fast] != val) {
            nums[slow++] = nums[fast];
        }
    }
    return slow; // 新数组长度
}
```

### 2. 滑动窗口

```java
/**
 * LeetCode 209: 长度最小的子数组
 * 找出和≥target的最短连续子数组
 */
public int minSubArrayLen(int target, int[] nums) {
    int left = 0, sum = 0;
    int minLen = Integer.MAX_VALUE;
    
    for (int right = 0; right < nums.length; right++) {
        sum += nums[right]; // 扩大窗口
        
        // 当满足条件时，尝试缩小窗口
        while (sum >= target) {
            minLen = Math.min(minLen, right - left + 1);
            sum -= nums[left++]; // 缩小窗口
        }
    }
    
    return minLen == Integer.MAX_VALUE ? 0 : minLen;
}
```

### 3. 前缀和

```java
/**
 * LeetCode 560: 和为K的子数组
 * 找出和为k的连续子数组个数
 */
public int subarraySum(int[] nums, int k) {
    Map<Integer, Integer> prefixSum = new HashMap<>();
    prefixSum.put(0, 1); // 前缀和为0出现1次
    
    int sum = 0, count = 0;
    for (int num : nums) {
        sum += num; // 计算前缀和
        
        // 如果存在前缀和sum-k，说明存在子数组和为k
        count += prefixSum.getOrDefault(sum - k, 0);
        
        // 记录当前前缀和
        prefixSum.put(sum, prefixSum.getOrDefault(sum, 0) + 1);
    }
    
    return count;
}
```

### 4. 二维数组

```java
/**
 * LeetCode 54: 螺旋矩阵
 * 顺时针螺旋遍历矩阵
 */
public List<Integer> spiralOrder(int[][] matrix) {
    List<Integer> result = new ArrayList<>();
    if (matrix == null || matrix.length == 0) return result;
    
    int top = 0, bottom = matrix.length - 1;
    int left = 0, right = matrix[0].length - 1;
    
    while (top <= bottom && left <= right) {
        // 上边：从左到右
        for (int i = left; i <= right; i++) {
            result.add(matrix[top][i]);
        }
        top++;
        
        // 右边：从上到下
        for (int i = top; i <= bottom; i++) {
            result.add(matrix[i][right]);
        }
        right--;
        
        // 下边：从右到左
        if (top <= bottom) {
            for (int i = right; i >= left; i--) {
                result.add(matrix[bottom][i]);
            }
            bottom--;
        }
        
        // 左边：从下到上
        if (left <= right) {
            for (int i = bottom; i >= top; i--) {
                result.add(matrix[i][left]);
            }
            left++;
        }
    }
    
    return result;
}
```

## ⚠️ 注意事项

### 1. 数组越界
```java
// ❌ 错误：可能越界
for (int i = 0; i <= arr.length; i++) { // 注意是<=
    System.out.println(arr[i]); // 当i=arr.length时越界
}

// ✅ 正确
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### 2. 空指针异常
```java
int[] arr = null;
// ❌ 使用前需判空
if (arr != null && arr.length > 0) {
    // 安全操作
}
```

### 3. 引用传递
```java
// 数组是引用类型，传递的是引用
public void modify(int[] arr) {
    arr[0] = 999; // 会修改原数组
}
```

### 4. Arrays工具类陷阱
```java
// Arrays.asList返回的是固定大小的List
List<Integer> list = Arrays.asList(1, 2, 3);
// list.add(4); // ❌ UnsupportedOperationException

// 正确做法
List<Integer> list = new ArrayList<>(Arrays.asList(1, 2, 3));
list.add(4); // ✅ 可以添加
```

## 🎓 最佳实践

### 1. 选择合适的数据结构
```java
// 固定大小，频繁访问 → 使用数组
int[] arr = new int[100];

// 大小不确定，频繁增删 → 使用ArrayList
List<Integer> list = new ArrayList<>();

// 需要线程安全 → 使用CopyOnWriteArrayList
List<Integer> safeList = new CopyOnWriteArrayList<>();
```

### 2. 双指针优化
很多数组问题可以用双指针将O(n²)优化到O(n)

### 3. 善用辅助数据结构
- 配合哈希表：降低查找复杂度到O(1)
- 配合堆：快速找最值
- 前缀和数组：快速计算区间和

### 4. 空间换时间
```java
// 使用额外数组存储中间结果，避免重复计算
int[] dp = new int[n]; // 动态规划
int[] preSum = new int[n]; // 前缀和
```
