## 🎓 学习路线

### 阶段一：基础必备（1-2个月）
1. **数据结构**: 数组、链表、栈、队列、哈希表
2. **基础算法**: 冒泡、选择、插入、二分查找
3. **练习**: LeetCode Easy 50题

### 阶段二：进阶提升（2-3个月）
1. **高级数据结构**: 树、堆、图
2. **经典算法**: 快排、归并、DFS、BFS
3. **初级动态规划**: 背包问题、LIS
4. **练习**: LeetCode Medium 100题

### 阶段三：算法深化（3-4个月）
1. **动态规划专题**: 各类DP问题
2. **图论算法**: 最短路径、最小生成树
3. **字符串算法**: KMP、Trie
4. **练习**: LeetCode Medium/Hard 150题

### 阶段四：竞赛级别（长期）
1. **高级算法**: 线段树、树状数组、AC自动机
2. **数学算法**: 数论、组合数学
3. **练习**: LeetCode Hard、Codeforces

## 💡 刷题策略

### 按数据结构分类
- **数组**: 双指针、滑动窗口、前缀和
- **链表**: 快慢指针、反转、合并
- **树**: 遍历、递归、层次遍历
- **图**: DFS、BFS、拓扑排序

### 按算法分类
- **排序**: 各种排序算法实现
- **查找**: 二分及其变种
- **动态规划**: 背包、LIS、LCS
- **贪心**: 区间问题、最优解
- **回溯**: 排列组合、N皇后

### LeetCode题目推荐
- **Easy**: 熟悉语法和基本操作
- **Medium**: 锻炼算法思维
- **Hard**: 提升综合能力
