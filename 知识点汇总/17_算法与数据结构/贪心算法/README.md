## 🧠 核心思想

```
贪心算法三要素：
┌─────────────────────────────────────┐
│  1. 贪心选择性质：局部最优→全局最优    │
│  2. 最优子结构：最优解包含子问题最优解  │
│  3. 无后效性：当前选择不影响后续状态   │
└─────────────────────────────────────┘

解题步骤：
1. 确定贪心策略（按什么标准选择）
2. 证明贪心选择性质（数学证明/反证法）
3. 编码实现
4. 验证正确性（边界用例测试）
```

## 📊 贪心 vs 动态规划

| 维度 | 贪心 | 动态规划 |
|------|------|----------|
| 选择方式 | 当前最优，不回头 | 枚举所有选择 |
| 最优解 | 不保证全局最优 | 保证全局最优 |
| 时间复杂度 | 通常O(n log n) | 通常O(n²)或O(nW) |
| 空间复杂度 | O(1)或O(n) | O(n²)或O(nW) |
| 典型问题 | 最小生成树、Dijkstra | 0-1背包、LCS |

**关键区别**：0-1背包不能用贪心（选了就不能选），分数背包可以用贪心（可以选部分）。

## 💡 应用场景

- **资源分配**：分数背包、任务调度
- **图论算法**：最小生成树(Kruskal/Prim)、最短路径(Dijkstra)
- **编码压缩**：霍夫曼编码
- **调度问题**：活动选择、区间覆盖
- **组合优化**：集合覆盖、区间合并
