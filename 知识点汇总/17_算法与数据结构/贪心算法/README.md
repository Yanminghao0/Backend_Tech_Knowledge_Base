# 贪心算法

> 贪心算法（Greedy Algorithm）是一种在每一步选择中都采取在当前状态下最好或最优（即最有利）的选择，从而希望导致结果是最好或最优的算法。它通过局部最优选择来达到全局最优，但并非所有问题都适用。本目录收集了经典的贪心算法问题及实现。

## ✨ 贪心算法特点

- **局部最优选择**：在每一步选择中都采取当前状态下最好的选择
- **无后效性**：过去的选择不会影响未来的状态
- **最优子结构**：问题的最优解包含子问题的最优解
- **适用条件**：仅当贪心选择性质成立时才能得到全局最优解

## 📝 应用场景

贪心算法广泛应用于：
- 资源分配问题（如分数背包）
- 图论算法（最小生成树、最短路径）
- 编码压缩（霍夫曼编码）
- 调度问题（活动选择、任务调度）
- 组合优化问题（区间覆盖、集合覆盖）
