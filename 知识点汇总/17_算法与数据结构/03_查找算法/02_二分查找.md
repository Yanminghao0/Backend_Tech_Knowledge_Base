# 二分查找（Binary Search）

> 在有序数组中高效查找目标值，时间复杂度O(log n)

---

## 📋 基本信息

- **类型**: 查找算法
- **时间复杂度**: O(log n)
- **空间复杂度**: O(1) - 迭代法，O(log n) - 递归法
- **适用条件**: 
  - ✅ 数组必须有序
  - ✅ 支持随机访问（数组）
- **核心思想**: 每次将搜索区间缩小一半

---

## 🎯 算法原理

### 1. 基本思路
1. 取中间位置 mid = (left + right) / 2
2. 比较 arr[mid] 与 target
3. 相等则找到，返回索引
4. arr[mid] < target，在右半部分查找
5. arr[mid] > target，在左半部分查找
6. 重复直到找到或区间为空

### 2. 核心流程图
```mermaid
graph TD
    A[开始] --> B[初始化 left=0, right=n-1]
    B --> C{left <= right?}
    C -- 是 --> D[计算 mid = left + (right-left)/2]
    D --> E{arr[mid] == target?}
    E -- 是 --> F[返回 mid，查找成功]
    E -- 否 --> G{arr[mid] < target?}
    G -- 是 --> H[left = mid + 1]
    G -- 否 --> I[right = mid - 1]
    H --> C
    I --> C
    C -- 否 --> J[返回 -1，查找失败]
    F --> K[结束]
    J --> K
```

### 3. 时间复杂度推导
- 每次查找将问题规模减半
- T(n) = T(n/2) + O(1)
- 解得 T(n) = O(log n)

## 💻 代码实现

### 1. 基础模板（闭区间 [left, right]）

```java
/**
 * 标准二分查找
 * 在有序数组中查找target，返回索引，不存在返回-1
 */
public int binarySearch(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1; // 闭区间
    
    while (left <= right) {  // 注意是 <=
        int mid = left + (right - left) / 2; // 防止溢出
        
        if (nums[mid] == target) {
            return mid;  // 找到目标
        } else if (nums[mid] < target) {
            left = mid + 1;  // 目标在右半部分
        } else {
            right = mid - 1; // 目标在左半部分
        }
    }
    
    return -1; // 未找到
}
```

### 2. 左闭右开区间模板 [left, right)

```java
/**
 * 二分查找（左闭右开区间）
 */
public int binarySearchLeftOpen(int[] nums, int target) {
    int left = 0;
    int right = nums.length; // 左闭右开
    
    while (left < right) {  // 注意是 <
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid;  // 不包含right，所以是mid而不是mid-1
        }
    }
    
    return -1;
}
```

### 3. 递归实现

```java
/**
 * 二分查找（递归版本）
 */
public int binarySearchRecursive(int[] nums, int target) {
    return binarySearchHelper(nums, target, 0, nums.length - 1);
}

private int binarySearchHelper(int[] nums, int target, int left, int right) {
    if (left > right) {
        return -1;  // 递归终止条件
    }
    
    int mid = left + (right - left) / 2;
    
    if (nums[mid] == target) {
        return mid;
    } else if (nums[mid] < target) {
        return binarySearchHelper(nums, target, mid + 1, right);
    } else {
        return binarySearchHelper(nums, target, left, mid - 1);
    }
}
```

## 📊 经典变种问题

### 1. 查找第一个等于target的位置

```java
/**
 * LeetCode 34: 在排序数组中查找元素的第一个位置
 */
public int findFirst(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    int result = -1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            result = mid;      // 记录结果
            right = mid - 1;   // 继续向左查找
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return result;
}
```

### 5. 旋转数组中查找

```java
/**
 * LeetCode 33: 搜索旋转排序数组
 * 数组在某个点被旋转（如[0,1,2,4,5,6,7]变为[4,5,6,7,0,1,2]）
 */
public int searchRotated(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            return mid;
        }
        
        // 判断哪一半是有序的
        if (nums[left] <= nums[mid]) {
            // 左半部分有序
            if (nums[left] <= target && target < nums[mid]) {
                right = mid - 1;  // target在左半部分
            } else {
                left = mid + 1;   // target在右半部分
            }
        } else {
            // 右半部分有序
            if (nums[mid] < target && target <= nums[right]) {
                left = mid + 1;   // target在右半部分
            } else {
                right = mid - 1;  // target在左半部分
            }
        }
    }
    
    return -1;
}
```

### 6. 寻找峰值

```java
/**
 * LeetCode 162: 寻找峰值
 * 峰值元素是指其值严格大于左右相邻值的元素
 */
public int findPeakElement(int[] nums) {
    int left = 0, right = nums.length - 1;
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] < nums[mid + 1]) {
            left = mid + 1;  // 峰值在右侧
        } else {
            right = mid;     // 峰值在左侧或就是mid
        }
    }
    
    return left;
}
```

### 7. 最小化最大值问题

```java
/**
 * LeetCode 410: 分割数组的最大值
 * 二分答案：对结果进行二分查找
 */
public int splitArray(int[] nums, int k) {
    int left = 0, right = 0;
    
    // 确定二分范围
    for (int num : nums) {
        left = Math.max(left, num);    // 最小可能：最大元素
        right += num;                   // 最大可能：所有元素和
    }
    
    // 二分查找答案
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (canSplit(nums, k, mid)) {
            right = mid;  // 可以分割，尝试更小的值
        } else {
            left = mid + 1;
        }
    }
    
    return left;
}

// 判断是否可以将数组分成k个子数组，每个子数组和不超过maxSum
private boolean canSplit(int[] nums, int k, int maxSum) {
    int count = 1;
    int sum = 0;
    
    for (int num : nums) {
        if (sum + num > maxSum) {
            count++;
            sum = num;
        } else {
            sum += num;
        }
    }
    
    return count <= k;
}
```

## 🔍 应用场景

### 1. 数据查找
- **有序数组查找**: 字典、目录
- **数据库索引**: B树索引本质是多路二分查找
- **版本控制**: Git bisect查找引入bug的提交

### 2. 求值问题
- **求平方根**: 在[0, x]范围内二分
- **求幂运算**: 快速幂算法
- **求解方程**: 数值计算中的二分法

### 3. 最优化问题
- **最小化最大值**: 如分割数组、木材切割
- **最大化最小值**: 如最大化最小距离
- **二分答案**: 将判定问题转化为搜索问题

## ⚠️ 注意事项

### 1. 防止整数溢出

```java
// ❌ 错误：可能溢出
int mid = (left + right) / 2;

// ✅ 正确：防止溢出
int mid = left + (right - left) / 2;
// 或者使用无符号右移
int mid = (left + right) >>> 1;
```

### 2. 边界条件选择

```java
// 闭区间 [left, right]
while (left <= right) {  // <=
    // ...
    right = mid - 1;  // -1
}

// 左闭右开 [left, right)
while (left < right) {  // <
    // ...
    right = mid;  // 不需要-1
}
```

### 3. 循环不变量
保持搜索区间的定义一致，避免死循环

```java
// 区间定义：[left, right] 中可能存在target
// 循环不变量：每次迭代后，target在[left, right]中或不存在
```

### 4. 判断条件
根据题目要求选择合适的判断条件

```java
// 查找精确值
if (nums[mid] == target) return mid;

// 查找边界
if (nums[mid] < target) {
    // 目标在右侧
} else {
    // 目标在左侧或就是mid
}
```

## 🎓 最佳实践

### 1. 二分查找三要素
- **初始化**: 正确设置left和right
- **循环条件**: while (left <= right) 或 while (left < right)
- **更新操作**: left = mid + 1 或 right = mid - 1

### 2. 模板选择
```java
// 模板1：查找精确值（闭区间）
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] == target) return mid;
    else if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
}

// 模板2：查找边界（左闭右开）
while (left < right) {
    int mid = left + (right - left) / 2;
    if (check(mid)) right = mid;
    else left = mid + 1;
}
```

### 3. 二分答案技巧
将"求最值"转化为"判定问题"，然后二分查找答案

### 4. 调试技巧
打印每次迭代的left、right、mid值，观察搜索区间变化
