# 线性DP

> 线性动态规划是状态之间呈线性关系的动态规划问题，其核心是找到状态转移的线性规律

---

## 📋 基本信息

- **定义**：状态之间呈线性关系的动态规划问题，状态转移通常只依赖于前一个或前几个状态
- **核心特征**：问题可以分解为按顺序排列的子问题，且子问题的解可以按线性顺序计算
- **时间复杂度**：通常为O(n)或O(n²)，取决于状态转移的复杂度
- **空间复杂度**：可优化至O(1)或O(n)，根据是否需要保存所有状态

## 🎯 问题分析

### 适用场景
线性DP适用于解决具有以下特征的问题：
1. **序列优化问题**：如最长递增子序列、最长公共子序列等
2. **区间优化问题**：如最大子数组和、环形子数组和等
3. **计数问题**：如不同路径数量、解码方法数量等
4. **资源分配问题**：如分割等和子集、划分数组为连续子序列等

### 典型特征
- 问题的解可以表示为一系列按顺序排列的状态
- 每个状态只依赖于之前的有限个状态
- 可以按线性顺序（通常是从左到右）计算所有状态

### 动态规划三要素
1. **状态定义**：明确dp数组的含义
2. **状态转移方程**：描述状态之间的递推关系
3. **初始条件和边界**：确定起始状态和特殊值

## 💻 代码实现

### 1. 最长递增子序列 (LIS) — O(n²) DP解法
```java
public int lengthOfLIS(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    int[] dp = new int[nums.length];
    int maxLen = 1;
    Arrays.fill(dp, 1);
    
    for (int i = 1; i < nums.length; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[i] > nums[j]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
                maxLen = Math.max(maxLen, dp[i]);
            }
        }
    }
    return maxLen;
}
```

### 1b. 最长递增子序列 — O(n log n) 贪心+二分解法
```java
public int lengthOfLISOptimized(int[] nums) {
    int[] tails = new int[nums.length];
    int size = 0;
    
    for (int num : nums) {
        int lo = 0, hi = size;
        // 二分查找第一个 >= num 的位置
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (tails[mid] < num) {
                lo = mid + 1;
            } else {
                hi = mid;
            }
        }
        tails[lo] = num;
        if (lo == size) size++; // num比所有tail都大，扩展LIS
    }
    return size;
}
```

**tails数组含义**：`tails[i]` 表示长度为 i+1 的所有递增子序列中，末尾元素的最小值。维护一个单调递增的tails数组，通过二分查找高效更新。

### 2. 最大子数组和 (Kadane算法)
```java
public int maxSubArray(int[] nums) {
    int maxSum = nums[0];
    int currentSum = nums[0];
    
    for (int i = 1; i < nums.length; i++) {
        currentSum = Math.max(nums[i], currentSum + nums[i]);
        maxSum = Math.max(maxSum, currentSum);
    }
    return maxSum;
}
```

### 2b. 最大子数组和 — 返回起止索引
```java
public int[] maxSubArrayWithIndex(int[] nums) {
    int maxSum = nums[0], currentSum = nums[0];
    int start = 0, end = 0, tempStart = 0;
    
    for (int i = 1; i < nums.length; i++) {
        if (currentSum + nums[i] < nums[i]) {
            currentSum = nums[i];
            tempStart = i;
        } else {
            currentSum += nums[i];
        }
        if (currentSum > maxSum) {
            maxSum = currentSum;
            start = tempStart;
            end = i;
        }
    }
    return new int[]{maxSum, start, end};
}
```

### 3. 爬楼梯问题
```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int prevPrev = 1; // f(n-2)
    int prev = 2;    // f(n-1)
    int current = 0;
    
    for (int i = 3; i <= n; i++) {
        current = prevPrev + prev;
        prevPrev = prev;
        prev = current;
    }
    return current;
}
```

### 4. 打家劫舍 — 经典不相邻选择DP
```java
public int rob(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    if (nums.length == 1) return nums[0];
    
    int prev2 = nums[0];
    int prev1 = Math.max(nums[0], nums[1]);
    
    for (int i = 2; i < nums.length; i++) {
        int current = Math.max(prev1, prev2 + nums[i]);
        prev2 = prev1;
        prev1 = current;
    }
    return prev1;
}
```

### 5. 乘积最大子数组 — 同时维护最大值和最小值
```java
public int maxProduct(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    
    int maxProd = nums[0];
    int minProd = nums[0];
    int result = nums[0];
    
    for (int i = 1; i < nums.length; i++) {
        // 保存旧值（因为更新maxProd会影响minProd的计算）
        int tempMax = maxProd;
        int tempMin = minProd;
        
        maxProd = Math.max(nums[i], Math.max(tempMax * nums[i], tempMin * nums[i]));
        minProd = Math.min(nums[i], Math.min(tempMax * nums[i], tempMin * nums[i]));
        
        result = Math.max(result, maxProd);
    }
    return result;
}
```

### 6. 最长公共子序列 (LCS)
```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
```

### 7. 解码方法 — 计数型DP
```java
public int numDecodings(String s) {
    if (s == null || s.length() == 0 || s.charAt(0) == '0') return 0;
    
    int n = s.length();
    int[] dp = new int[n + 1];
    dp[0] = 1;
    dp[1] = 1;
    
    for (int i = 2; i <= n; i++) {
        // 单字符解码
        int oneDigit = s.charAt(i - 1) - '0';
        if (oneDigit >= 1 && oneDigit <= 9) {
            dp[i] += dp[i - 1];
        }
        // 双字符解码
        int twoDigits = (s.charAt(i - 2) - '0') * 10 + oneDigit;
        if (twoDigits >= 10 && twoDigits <= 26) {
            dp[i] += dp[i - 2];
        }
    }
    return dp[n];
}
```

```python
# 最长递增子序列 O(n log n)
from bisect import bisect_left

def length_of_LIS(nums):
    tails = []
    for num in nums:
        pos = bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)

# 最大子数组和
def max_sub_array(nums):
    max_sum = curr_sum = nums[0]
    for num in nums[1:]:
        curr_sum = max(num, curr_sum + num)
        max_sum = max(max_sum, curr_sum)
    return max_sum

# 打家劫舍
def rob(nums):
    if not nums: return 0
    if len(nums) == 1: return nums[0]
    prev2, prev1 = nums[0], max(nums[0], nums[1])
    for i in range(2, len(nums)):
        prev2, prev1 = prev1, max(prev1, prev2 + nums[i])
    return prev1
```

## 📊 状态转移方程

### 1. 最长递增子序列
- **状态定义**：`dp[i]` 表示以第i个元素结尾的最长递增子序列长度
- **转移方程**：`dp[i] = max(dp[j] + 1) for j < i and nums[i] > nums[j]`
- **初始状态**：`dp[i] = 1` 对所有i成立
- **最终结果**：`max(dp[i]) for all i`
- **空间优化**：无法优化为O(1)，但可用贪心+二分优化到O(n log n)时间

### 2. 最大子数组和
- **状态定义**：`dp[i]` 表示以第i个元素结尾的最大子数组和
- **转移方程**：`dp[i] = max(nums[i], dp[i-1] + nums[i])`
- **初始状态**：`dp[0] = nums[0]`
- **最终结果**：`max(dp[i]) for all i`
- **空间优化**：O(1) — 只需保存前一个状态

### 3. 爬楼梯问题
- **状态定义**：`dp[i]` 表示爬到第i级台阶的方法数
- **转移方程**：`dp[i] = dp[i-1] + dp[i-2]`
- **初始状态**：`dp[1] = 1, dp[2] = 2`
- **空间优化**：O(1) — 滚动变量

### 4. 打家劫舍
- **状态定义**：`dp[i]` 表示偷窃前i个房屋能获得的最大金额
- **转移方程**：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
- **初始状态**：`dp[0] = nums[0], dp[1] = max(nums[0], nums[1])`
- **空间优化**：O(1) — 滚动变量

### 5. 乘积最大子数组
- **状态定义**：`maxDp[i]` 表示以i结尾的子数组最大乘积；`minDp[i]` 表示最小乘积
- **转移方程**：
  - `maxDp[i] = max(nums[i], maxDp[i-1]*nums[i], minDp[i-1]*nums[i])`
  - `minDp[i] = min(nums[i], maxDp[i-1]*nums[i], minDp[i-1]*nums[i])`
- **初始状态**：`maxDp[0] = minDp[0] = nums[0]`
- **空间优化**：O(1) — 滚动变量

### 6. 最长公共子序列
- **状态定义**：`dp[i][j]` 表示text1前i个字符与text2前j个字符的LCS长度
- **转移方程**：
  - 若 `text1[i-1] == text2[j-1]`：`dp[i][j] = dp[i-1][j-1] + 1`
  - 否则：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`
- **初始状态**：`dp[0][j] = dp[i][0] = 0`
- **空间优化**：O(min(m,n)) — 滚动数组

## ⚠️ 边界条件分析

1. **空数组/空字符串**：需单独处理返回0或1（计数问题返回1，求值问题返回0）
2. **单元素数组**：通常直接返回该元素
3. **全负数数组**（最大子数组和）：Kadane算法仍有效，返回最大的单个负数
4. **包含0的乘积问题**：0会重置乘积，需要特殊处理
5. **前导0的解码方法**：如"06"无法解码，返回0
6. **环形数组**（打家劫舍II）：首尾不能同时选择，拆分为两个线性问题分别求解

### 环形打家劫舍详解
```java
// 环形数组：首尾相邻，不能同时偷第一个和最后一个
public int robCircular(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    if (nums.length == 1) return nums[0];
    
    // 情况1：偷第一个，不偷最后一个 → 范围[0, n-2]
    // 情况2：不偷第一个，偷最后一个 → 范围[1, n-1]
    return Math.max(
        robRange(nums, 0, nums.length - 2),
        robRange(nums, 1, nums.length - 1)
    );
}

private int robRange(int[] nums, int start, int end) {
    if (start == end) return nums[start];
    int prev2 = nums[start];
    int prev1 = Math.max(nums[start], nums[start + 1]);
    for (int i = start + 2; i <= end; i++) {
        int current = Math.max(prev1, prev2 + nums[i]);
        prev2 = prev1;
        prev1 = current;
    }
    return prev1;
}
```

## 📝 面试常见问题

**Q1: 线性DP的空间优化方法有哪些？**
> 1. **滚动数组**：当dp[i]只依赖dp[i-1]和dp[i-2]时，用两个变量替代整个数组
> 2. **原地修改**：在原数组上直接修改，不需要额外dp数组
> 3. **降维**：二维dp如果某维可以滚动，则用一维数组替代

**Q2: LIS的O(n log n)解法原理是什么？**
> 维护一个tails数组，`tails[i]` 表示长度为i+1的递增子序列的最小末尾值。对新元素进行二分查找：如果比所有tails都大，则扩展LIS长度；否则替换第一个≥它的元素，保持tails的递增性，使后续有更小的末尾值。

**Q3: Kadane算法为什么有效？**
> Kadane算法的核心是：以i结尾的最大子数组和，要么是nums[i]本身（重新开始），要么是dp[i-1]+nums[i]（延续前面的子数组）。这保证了每个位置的局部最优解，全局最优解必然在某个局部最优中。

**Q4: 乘积最大子数组为什么需要同时维护最小值？**
> 负数乘负数得正数。当前的最小乘积（可能为负）乘以一个负数，可能变成最大值。因此需要同时维护最大和最小乘积。

## 🎓 最佳实践

1. **从简单问题入手**：先解决基础问题，再尝试优化和处理复杂变体
2. **可视化状态转移**：使用表格或图表辅助理解状态之间的关系
3. **掌握空间优化技巧**：熟练运用滚动数组、状态压缩等方法减少空间消耗
4. **多维度思考**：尝试从不同角度定义状态，寻找更优的状态转移方程
5. **归纳总结**：将相似问题归类，总结通用解题模板
6. **先写暴力递归**：从递归→记忆化→DP是推导状态转移方程的有效路径
7. **画DP表格**：手动填写小规模数据的DP表格，验证状态转移正确性

### 线性DP通用解题模板
```
1. 定义状态：dp[i] 表示什么？
2. 写出转移方程：dp[i] 与哪些状态有关？
3. 确定初始条件和边界
4. 确定计算顺序（通常从左到右）
5. 确定最终结果在哪（max(dp)、dp[n-1]等）
6. 考虑空间优化
7. 验证边界条件
```
