# 微服务部署运维

## 目录概述
本目录包含微服务部署与运维的实战文档，共计1篇核心指南：
- **微服务部署策略实战.md**：微服务部署模式与运维最佳实践

## 核心价值
提供微服务架构下的部署运维解决方案，覆盖部署策略、环境管理和运维自动化，解决微服务规模化部署的复杂性，保障系统稳定运行与高效迭代。

## 关键技术点
- 部署策略：蓝绿部署、金丝雀发布、灰度发布、影子部署
- 容器化：Docker容器编排、Kubernetes部署配置、资源限制
- 环境管理：多环境隔离、配置管理、基础设施即代码(IaC)
- 运维自动化：CI/CD流水线、自动扩缩容、故障自愈
- 监控告警：部署指标监控、日志聚合分析、异常检测
