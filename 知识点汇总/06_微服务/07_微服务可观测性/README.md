## 📋 文档列表

### 1. 分布式追踪实战 ⭐⭐⭐ 核心
📄 [分布式追踪实战.md](./01_分布式追踪实战.md)

**核心内容**：
- ✅ **分布式追踪原理**：Trace、Span、Context Propagation
- ✅ **OpenTelemetry**：统一可观测性标准，Traces + Metrics + Logs
- ✅ **SkyWalking实战**：Java Agent接入、自定义Span、告警规则
- ✅ **Jaeger实战**：分布式追踪系统部署与使用
- ✅ **采样策略**：头部采样、尾部采样、自适应采样
- ✅ **跨服务调用链**：HTTP、gRPC、MQ消息链路打通

**核心技术概念**：
```
Trace（追踪）→ 完整请求链路
  └── Span（跨度）→ 单个操作
       ├── SpanContext（上下文）→ 传播标识
       ├── Attributes（属性）→ 业务标签
       └── Events（事件）→ 时间节点
```

**适用场景**：请求链路追踪、性能瓶颈定位、故障根因分析

### 2. 微服务可观测性实战 ⭐⭐⭐ 核心
📄 [微服务可观测性实战.md](./02_微服务可观测性实战.md)

**核心内容**：
- ✅ **三大支柱**：Metrics（指标）、Logs（日志）、Traces（追踪）
- ✅ **监控指标体系**：RED方法（速率、错误率、持续时间）、USE方法（使用率、饱和度、错误）
- ✅ **日志管理**：结构化日志、ELK/Loki集中式日志、日志关联追踪
- ✅ **告警体系**：多级告警策略、告警聚合与去重、根因分析自动化
- ✅ **Grafana仪表盘**：监控大盘设计、告警通知集成

**可观测性工具选型**：

| 类别 | 工具 | 特点 |
|------|------|------|
| 指标监控 | Prometheus + Grafana | 时序数据、PromQL查询 |
| 分布式追踪 | SkyWalking / Jaeger | Java Agent无侵入 |
| 日志收集 | ELK / Loki + Promtail | 全文检索 vs 标签过滤 |
| 告警管理 | Alertmanager / Grafana | 多渠道通知、告警聚合 |

**适用场景**：系统监控、故障预警、性能分析、容量规划

## 核心价值
提供微服务架构下的可观测性建设方案，解决分布式系统的问题定位、性能优化和故障预警难题，保障系统稳定性与可维护性。

## 关键技术点
- 分布式追踪：TraceID生成、Span上下文传递、采样策略配置
- 监控指标：RED方法(速率、错误率、持续时间)、USE方法(使用率、饱和度、错误)
- 日志管理：结构化日志采集、集中式日志分析、日志关联分析
- 告警体系：多级告警策略、告警聚合、根因分析自动化
