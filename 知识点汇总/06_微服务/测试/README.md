# 微服务测试

## 目录概述
本目录包含微服务测试体系构建的实战文档，共计1篇核心指南：
- **微服务测试实战.md**：微服务全链路测试策略与实践

## 核心价值
提供微服务架构下的测试解决方案，覆盖单元测试、集成测试、契约测试到端到端测试全流程，保障微服务迭代质量与系统稳定性。

## 关键技术点
- 测试策略：测试金字塔在微服务中的应用、测试环境管理
- 契约测试：Spring Cloud Contract、Pact框架实践
- API测试：RESTful API自动化测试、接口性能测试
- 端到端测试：Selenium/Cypress在微服务场景的应用
- 测试数据：动态测试数据生成、环境隔离方案
