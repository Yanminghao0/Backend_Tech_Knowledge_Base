# 微服务通信协议

## 目录概述
本目录包含微服务通信技术的实战文档，共计2篇核心指南：
- **Dubbo与gRPC详解.md**：RPC框架深度对比与实践
- **微服务通信模式实战.md**：同步/异步通信模式设计与实现

## 核心价值
提供微服务间通信的完整解决方案，涵盖高性能RPC框架选型、通信模式设计及跨服务调用优化，解决服务间数据传输效率、协议兼容性和通信可靠性问题。

## 关键技术点
- RPC框架：Dubbo架构(注册中心/协议层/服务治理)、gRPC原理(HTTP/2/Protobuf)
- 通信模式：同步RPC、异步消息、请求-响应、发布-订阅、双向流
- 协议选型：Dubbo协议、gRPC协议、RESTful API、GraphQL
- 性能优化：连接池管理、序列化方案、压缩策略、超时重试机制
