# 微服务服务治理

## 目录概述
本目录包含微服务架构中服务治理相关的核心实战文档，共计3篇关键指南：
- **Sentinel流量控制详解.md**：分布式系统流量防卫兵，包含限流、熔断、降级全场景实现
- **服务网格实战.md**：Service Mesh架构落地指南，涵盖Istio部署与流量管理
- **熔断器实战.md**：微服务容错机制实现，包含Resilience4j与Spring Cloud Circuit Breaker应用

## 核心价值
提供微服务稳定性保障方案，解决分布式系统中的流量控制、服务容错和通信管理问题，确保高并发场景下的系统可靠性。

## 关键技术点
- 流量控制：基于QPS/线程数的限流策略、热点参数限流、集群限流
- 服务容错：熔断器模式、舱壁模式、超时重试机制
- 服务网格：数据平面与控制平面架构、流量劫持、mTLS加密
- 可观测性：治理指标监控、告警阈值配置、故障演练
