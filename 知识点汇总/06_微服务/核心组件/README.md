# 微服务核心组件

## 目录概述
本目录包含微服务架构中的关键基础设施组件实战文档，共计5篇核心指南：
- **API Gateway实战指南.md**：API网关设计与实现
- **Nacos核心机制详解.md**：服务发现与配置管理平台
- **Spring Cloud Alibaba全家桶.md**：微服务一站式解决方案
- **分布式配置中心实战.md**：配置统一管理与动态刷新
- **服务注册与发现实战.md**：服务治理基础组件

## 核心价值
提供微服务架构的基础设施搭建指南，涵盖服务注册发现、配置管理、API路由等关键能力，帮助开发者快速构建稳定可靠的微服务体系。

## 关联主题
- 服务治理：服务熔断、限流、降级策略
- 可观测性：分布式追踪、监控告警体系
- 部署运维：容器化部署与CI/CD流程
