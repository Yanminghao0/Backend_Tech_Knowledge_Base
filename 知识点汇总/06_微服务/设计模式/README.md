# 微服务设计模式

## 目录概述
本目录包含微服务架构设计模式的实战文档，共计1篇核心指南：
- **微服务设计模式详解.md**：微服务架构关键设计模式解析与实践

## 核心价值
提供微服务架构设计的方法论指导，通过经典设计模式解决服务拆分、通信、数据管理等核心问题，帮助架构师构建弹性、可扩展的微服务系统。

## 关键技术点
- 服务拆分：领域驱动设计(DDD)、康威定律应用、服务边界划分
- 通信模式：同步RPC、异步消息、请求-响应模式、事件流模式
- 数据管理：每服务一数据库、共享数据库、CQRS模式
- 可靠性模式：断路器、舱壁、限流器、退避与重试
- 部署模式：蓝绿部署、金丝雀发布、影子部署
