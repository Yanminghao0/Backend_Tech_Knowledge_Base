## 🔧 技术选型

### 注册中心对比

| 特性 | Nacos | Eureka | Consul |
|------|-------|--------|--------|
| 服务发现 | ✅ | ✅ | ✅ |
| 配置管理 | ✅ | ❌ | ✅ |
| 一致性 | CP+AP | AP | CP |
| 推荐度 | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐ |

### RPC框架对比

| 场景 | 推荐方案 | 理由 |
|------|---------|------|
| 纯Java微服务 | Dubbo + Nacos | 服务治理能力强 |
| 跨语言微服务 | gRPC | 跨语言支持好 |
| Spring生态 | Spring Cloud Alibaba | 无缝集成 |
