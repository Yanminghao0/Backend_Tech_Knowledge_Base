## 1. 多模态AI概述

### 什么是多模态

```
多模态 = 处理多种类型的数据（文本+图片+音频+视频）

模态组合：
  文本+图片 → 图片描述/图文问答/OCR
  文本+音频 → 语音助手/会议纪要
  文本→图片 → AI绘画/设计生成
  文本→音频 → 语音合成/有声书
  图片+音频 → 视频生成

主流多模态模型：
  GPT-5：文本+图片+音频（输入）→文本
  Claude 4：文本+图片→文本
  LLaVA：开源图文理解
  CLIP：图文匹配（Embedding）
  Whisper：语音识别（OpenAI开源）
  DALL-E 3/Midjourney/Stable Diffusion：图片生成
```

### GPT-5/Claude视觉API

```java
// Spring AI + OpenAI Vision
@Service
public class ImageUnderstandingService {
    
    @Autowired
    private ChatClient chatClient;
    
    // 图片描述
    public String describeImage(byte[] imageBytes, String mimeType) {
        String base64Image = Base64.encodeBase64String(imageBytes);
        
        return chatClient.prompt()
            .user(u -> u.text("请详细描述这张图片的内容")
                .media(MimeTypeUtils.parseMimeType(mimeType), 
                    new org.springframework.core.io.ByteArrayResource(imageBytes)))
            .call()
            .content();
    }
    
    // 图文问答
    public String askAboutImage(byte[] imageBytes, String question) {
        return chatClient.prompt()
            .user(u -> u.text(question)
                .media(MediaType.IMAGE_JPEG, 
                    new ByteArrayResource(imageBytes)))
            .call()
            .content();
    }
    
    // OCR + 结构化提取
    public InvoiceInfo extractInvoice(byte[] imageBytes) {
        String prompt = """
            请从这张发票图片中提取以下信息，输出JSON：
            {"invoiceNo": "发票号", "date": "日期", "amount": "金额", "taxId": "税号"}
            """;
        
        return chatClient.prompt()
            .user(u -> u.text(prompt)
                .media(MediaType.IMAGE_JPEG, new ByteArrayResource(imageBytes)))
            .call()
            .entity(InvoiceInfo.class);
    }
}
```

### CLIP图文匹配

```java
// CLIP：计算图片和文本的相似度
@Service
public class ClipService {
    
    // 调用本地CLIP模型（通过Python微服务或ONNX Runtime）
    public double calculateSimilarity(byte[] image, String text) {
        // 1. 图片Embedding
        float[] imageEmbedding = clipClient.imageEmbed(image);
        
        // 2. 文本Embedding
        float[] textEmbedding = clipClient.textEmbed(text);
        
        // 3. 余弦相似度
        return cosineSimilarity(imageEmbedding, textEmbedding);
    }
    
    // 应用：图片搜索（用文本搜索图片库）
    public List<String> searchImages(String query, List<byte[]> imageLibrary) {
        return imageLibrary.stream()
            .map(img -> Map.of("image", img, "score", calculateSimilarity(img, query)))
            .sorted((a, b) -> Double.compare(b.get("score"), a.get("score")))
            .limit(10)
            .map(m -> encodeBase64(m.get("image")))
            .collect(Collectors.toList());
    }
}
```

## 3. 语音识别（ASR）

### Whisper（OpenAI开源）

```java
@Service
public class SpeechToTextService {
    
    private final WebClient whisperClient;
    
    // 方式1：调用OpenAI Whisper API
    public String transcribe(byte[] audioBytes, String format) {
        MultipartBodyBuilder builder = new MultipartBodyBuilder();
        builder.part("file", new ByteArrayResource(audioBytes))
               .filename("audio." + format);
        builder.part("model", "whisper-1");
        builder.part("language", "zh");
        
        return webClient.post()
            .uri("https://api.openai.com/v1/audio/transcriptions")
            .contentType(MediaType.MULTIPART_FORM_DATA)
            .body(builder.build())
            .retrieve()
            .bodyToMono(WhisperResponse.class)
            .map(WhisperResponse::getText)
            .block();
    }
    
    // 方式2：本地Whisper（faster-whisper）
    // 通过Python微服务调用
    public String transcribeLocal(byte[] audioBytes) {
        return localWhisperClient.transcribe(audioBytes);
    }
}
```

```
4. 客服质检：通话录音→文字→情感分析
```

```java
@Service
public class TextToSpeechService {
    
    // OpenAI TTS
    public byte[] synthesize(String text) {
        return webClient.post()
            .uri("https://api.openai.com/v1/audio/speech")
            .bodyValue(Map.of(
                "model", "tts-1",
                "input", text,
                "voice", "alloy",  // alloy/echo/fable/onyx/nova/shimmer
                "response_format", "mp3"
            ))
            .retrieve()
            .bodyToMono(byte[].class)
            .block();
    }
    
    // 流式TTS（边生成边播放）
    public Flux<byte[]> streamSynthesize(String text) {
        return webClient.post()
            .uri("https://api.openai.com/v1/audio/speech")
            .bodyValue(Map.of(
                "model", "tts-1",
                "input", text,
                "voice", "nova",
                "response_format", "opus",
                "stream", true
            ))
            .retrieve()
            .bodyToFlux(byte[].class);
    }
}
```

### 端到端语音助手

```java
@RestController
public class VoiceAssistantController {
    
    @PostMapping("/api/voice/ask")
    public byte[] voiceAsk(@RequestParam MultipartFile audio) {
        // 1. 语音→文字（ASR）
        byte[] audioBytes = audio.getBytes();
        String text = sttService.transcribe(audioBytes, "wav");
        
        // 2. LLM生成回答
        String answer = chatClient.prompt()
            .user(text)
            .call()
            .content();
        
        // 3. 文字→语音（TTS）
        return ttsService.synthesize(answer);
    }
    
    // 流式语音助手（低延迟）
    @GetMapping(value = "/api/voice/stream", 
                produces = MediaType.APPLICATION_OCTET_STREAM_VALUE)
    public Flux<byte[]> voiceStream(@RequestParam MultipartFile audio) {
        // 1. ASR
        String text = sttService.transcribe(audio.getBytes(), "wav");
        
        // 2. LLM流式生成
        Flux<String> textStream = chatClient.prompt()
            .user(text)
            .stream()
            .content();
        
        // 3. 按句子分割 → 逐句TTS
        return textStream
            .bufferUntil(c -> c.equals("。") || c.equals("！") || c.equals("？"))
            .map(chars -> String.join("", chars))
            .flatMap(sentence -> 
                Mono.fromCallable(() -> ttsService.synthesize(sentence))
                    .subscribeOn(Schedulers.boundedElastic())
            );
    }
}
```

## 5. 图片生成

```java
@Service
public class ImageGenerationService {
    
    // DALL-E 3
    public String generateImage(String prompt, String size) {
        DallEResponse response = webClient.post()
            .uri("https://api.openai.com/v1/images/generations")
            .bodyValue(Map.of(
                "model", "dall-e-3",
                "prompt", prompt,
                "size", size != null ? size : "1024x1024",
                "quality", "hd",
                "n", 1
            ))
            .retrieve()
            .bodyToMono(DallEResponse.class)
            .block();
        
        return response.getData().get(0).getUrl();
    }
    
    // Stable Diffusion（本地部署）
    public byte[] generateImageLocal(String prompt) {
        return sdClient.post()
            .uri("/sdapi/v1/txt2img")
            .bodyValue(Map.of(
                "prompt", prompt,
                "steps", 30,
                "width", 1024,
                "height", 1024,
                "cfg_scale", 7.5
            ))
            .retrieve()
            .bodyToMono(SDResponse.class)
            .map(r -> Base64.decodeBase64(r.getImages().get(0)))
            .block();
    }
}
```

## 6. 面试要点

### Q1: 多模态AI有哪些应用场景？

```
1. 图片理解：图片描述/OCR/图文问答/质检
2. 语音识别：会议纪要/字幕/语音搜索/客服质检
3. 语音合成：有声书/语音助手/通知播报
4. 图片生成：AI绘画/设计素材/广告素材
5. 视频生成：营销视频/教学视频
6. 端到端：语音助手（ASR→LLM→TTS）
```

### Q2: 如何实现语音助手？

```
端到端流程：
  1. 用户语音输入 → Whisper ASR → 文字
  2. 文字 → LLM → 回答文字
  3. 回答文字 → TTS → 语音输出

优化：
  - 流式：LLM流式生成→按句TTS→减少延迟
  - VAD：语音活动检测，自动开始/结束录音
  - 打断检测：用户说话时停止TTS播放
```

### Q3: Java调用多模态模型的方式？

```
1. 云API：OpenAI/Claude/Qwen的Vision API
   - Spring AI的media()方法
   - 简单但需联网

2. 本地部署：
   - CLIP（图文匹配）→ ONNX Runtime / Python微服务
   - Whisper（ASR）→ faster-whisper Python微服务
   - Stable Diffusion（图片生成）→ Automatic1111 API

3. Java直接调用：
   - DJL（Deep Java Library）支持部分多模态模型
   - ONNX Runtime Java SDK
```
