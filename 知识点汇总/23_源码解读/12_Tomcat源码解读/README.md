## 📖 Tomcat架构概览

```
┌─────────────────────────────────────────────────────────────┐
│                      Tomcat架构                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                     Server                           │   │
│  │  ┌─────────────────────────────────────────────┐    │   │
│  │  │                  Service                     │    │   │
│  │  │  ┌───────────────┐  ┌───────────────────┐   │    │   │
│  │  │  │   Connector   │  │     Container      │   │    │   │
│  │  │  │  (HTTP/AJP)   │  │  ┌─────────────┐  │   │    │   │
│  │  │  │               │  │  │   Engine    │  │   │    │   │
│  │  │  │  ┌─────────┐  │  │  │ ┌─────────┐ │  │   │    │   │
│  │  │  │  │Endpoint │  │  │  │ │  Host   │ │  │   │    │   │
│  │  │  │  │         │  │──▶│ │ ┌───────┐│ │  │   │    │   │
│  │  │  │  │Processor│  │  │  │ │Context││ │  │   │    │   │
│  │  │  │  └─────────┘  │  │  │ │Wrapper││ │  │   │    │   │
│  │  │  └───────────────┘  │  │ └───────┘│ │  │   │    │   │
│  │  │                     │  └─────────────┘  │   │    │   │
│  │  └─────────────────────────────────────────────┘    │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘

核心组件：
- Server: 整个Tomcat服务器
- Service: 包含Connector和Container
- Connector: 处理网络连接
- Container: 处理请求（Engine→Host→Context→Wrapper）
```
