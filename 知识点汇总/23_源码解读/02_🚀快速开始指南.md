### 3. 调试技巧
```java
// 示例：调试HashMap的put方法
public class HashMapDebug {
    public static void main(String[] args) {
        Map<String, String> map = new HashMap<>(4);
        
        // 设置断点，观察扩容过程
        map.put("key1", "value1");
        map.put("key2", "value2");
        map.put("key3", "value3");
        map.put("key4", "value4");
        map.put("key5", "value5"); // 触发扩容
        
        System.out.println("容量: " + getCapacity(map));
    }
    
    // 反射获取HashMap容量
    private static int getCapacity(Map<?, ?> map) {
        try {
            Field field = HashMap.class.getDeclaredField("table");
            field.setAccessible(true);
            Object[] table = (Object[]) field.get(map);
            return table == null ? 0 : table.length;
        } catch (Exception e) {
            return -1;
        }
    }
}
```

## 🔥 面试突击

### 必背知识点
```
1. HashMap原理（必问）
   - put/get流程
   - 哈希冲突解决
   - 红黑树转换
   - 扩容机制

2. Spring Bean生命周期（必问）
   - 实例化 → 属性填充 → 初始化 → 销毁
   - BeanPostProcessor作用时机
   - Aware接口回调

3. 循环依赖解决（必问）
   - 三级缓存作用
   - 为什么需要三级缓存
   - 构造器循环依赖无法解决

4. AOP实现原理（高频）
   - JDK代理 vs CGLIB代理
   - 代理创建时机
   - 同类方法调用失效原因

5. 事务传播机制（高频）
   - REQUIRED vs REQUIRES_NEW
   - NESTED嵌套事务
   - 事务失效场景
```
