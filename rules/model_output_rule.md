# 模型输出规则

## 规则说明
每次AI助手输出回复时，必须在**对话响应**中标注当前使用的模型型号。

**重要**：模型型号信息只出现在对话返回的结果中，**不要**放在生成的文件内容中。

## 规则要求

### 1. 输出位置
- 在每次**对话回复**的末尾添加模型型号信息
- 使用统一的格式标记
- **仅限对话响应**，不包含在创建或编辑的文件内容中

### 2. 输出格式
```
---
🤖 当前模型：[模型名称] [完整版本号]
```

### 3. 版本信息要求
- 必须包含**完整的模型名称**
- 必须包含**详细的版本号**（如有子版本或实验版本标识，也需包含）
- 格式示例：`模型系列名称 版本号`

### 4. 示例格式
```markdown
---
🤖 当前模型：GPT-4 Turbo 2024-04-09
```

或

```markdown
---
🤖 当前模型：Claude 3.5 Sonnet (claude-3-5-sonnet-20241022)
```

或

```markdown
---
🤖 当前模型：Gemini 2.0 Flash Thinking Experimental 1219
```

## 适用场景
- ✅ 对话回复消息
- ✅ 问题解答响应
- ❌ **不适用**：生成的 md 文件内容
- ❌ **不适用**：生成的代码文件内容
- ❌ **不适用**：任何写入文件系统的文件

## 目的
- 便于追踪不同模型的输出质量
- 方便进行模型效果对比
- 有助于优化提示词设计
- 提高工作透明度

## 备注
此规则应作为系统级规则，在AI助手的系统提示词中配置。
