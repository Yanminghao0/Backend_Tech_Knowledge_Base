# 分析类提示词模板

## 模板结构
```
你是一位[专业领域]专家。请帮我分析[分析对象]，要求如下：

**分析背景：**
- 分析目的：[为什么要分析]
- 分析范围：[分析的范围和边界]
- 分析深度：[分析的详细程度]
- 时间要求：[分析的时间限制]

**分析要求：**
1. 数据收集：[需要收集什么数据]
2. 分析方法：[使用什么分析方法]
3. 分析维度：[从哪些角度分析]
4. 结论要求：[需要得出什么结论]
5. 建议要求：[需要提供什么建议]

**输出格式：**
请按以下结构输出：
## 分析概述
[分析的整体概述]

## 详细分析
### 维度1：[分析维度名称]
[分析内容]

### 维度2：[分析维度名称]
[分析内容]

### 维度3：[分析维度名称]
[分析内容]

## 结论与建议
### 主要结论
[核心结论]

### 具体建议
[可操作的建议]

### 风险提示
[潜在风险和注意事项]
```

## 使用说明
1. **专业定位**：明确分析的专业领域
2. **分析对象**：清楚说明要分析什么
3. **背景信息**：提供分析的必要背景
4. **分析要求**：详细说明分析标准
5. **输出格式**：结构化输出便于理解

## 适用场景
- 市场分析
- 财务分析
- 竞品分析
- 用户分析
- 风险评估
- 趋势预测

## 使用技巧
- 分析目的要明确
- 分析维度要全面
- 结论要基于数据
- 建议要可操作
- 风险要提前识别
