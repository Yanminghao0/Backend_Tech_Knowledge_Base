# 基础提示词模板

## 模板结构
```
你是一位[角色定位]。请帮我[具体任务]，要求如下：

**基本信息：**
- [关键信息1]：[具体内容]
- [关键信息2]：[具体内容]
- [关键信息3]：[具体内容]

**具体要求：**
1. [要求1]
2. [要求2]
3. [要求3]
4. [要求4]
5. [要求5]

**输出格式：**
请按以下格式输出：
[格式要求]

**注意事项：**
- [注意事项1]
- [注意事项2]
- [注意事项3]
```

## 使用说明
1. **角色定位**：明确AI的身份和专长
2. **具体任务**：清楚说明要完成什么
3. **基本信息**：提供必要的背景信息
4. **具体要求**：列出详细的任务要求
5. **输出格式**：指定期望的输出格式
6. **注意事项**：补充重要的约束条件

## 适用场景
- 通用任务处理
- 内容创作
- 问题解决
- 信息整理
- 分析报告

## 使用技巧
- 角色定位要具体，不要过于宽泛
- 任务描述要清晰，避免歧义
- 要求要具体，避免模糊表达
- 格式要明确，便于后续处理
- 注意事项要全面，避免遗漏
