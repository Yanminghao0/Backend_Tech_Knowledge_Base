# 创意类提示词模板

## 模板结构
```
你是一位富有创意的[创作领域]专家。请帮我创作[创作内容]，要求如下：

**创作背景：**
- 创作目的：[为什么创作]
- 目标受众：[面向谁]
- 使用场景：[在哪里使用]
- 风格要求：[什么风格]

**创作要求：**
1. 主题表达：[如何表达主题]
2. 情感色彩：[什么情感]
3. 语言风格：[什么语言风格]
4. 结构要求：[什么结构]
5. 创新要求：[什么创新点]

**输出格式：**
请按以下结构输出：
## 创作大纲
[创作的整体思路]

## 完整作品
[完整的创作内容]

## 创作说明
- 主题表达：[如何表达主题]
- 创作技巧：[使用的技巧]
- 预期效果：[希望达到什么效果]
```

## 使用说明
1. **创作领域**：明确创作的专业领域
2. **创作内容**：清楚说明要创作什么
3. **背景信息**：提供创作的必要背景
4. **创作要求**：详细说明创作标准
5. **输出格式**：结构化输出便于理解

## 适用场景
- 文案创作
- 故事写作
- 广告创意
- 品牌策划
- 内容营销
- 创意设计

## 使用技巧
- 目标受众要明确
- 情感色彩要突出
- 语言风格要一致
- 结构要清晰
- 创新点要突出
