## 常见问题

### Q1: 如何选择合适的模板？
**A**: 根据任务类型选择：
- 通用任务：使用基础模板
- 分析任务：使用分析模板
- 创作任务：使用创作模板

### Q2: 如何提高提示词质量？
**A**: 遵循以下原则：
- 明确性原则：角色、任务、要求要具体
- 结构化原则：背景、要求、格式要完整
- 可操作原则：要求要可执行，格式要明确

### Q3: 如何避免常见错误？
**A**: 注意以下几点：
- 避免过于模糊的表达
- 提供充分的背景信息
- 设置合理的约束条件
- 指定明确的输出格式

### Q4: 如何持续改进？
**A**: 采用以下方法：
- 记录使用效果
- 分析成功失败原因
- 收集反馈意见
- 不断优化完善
