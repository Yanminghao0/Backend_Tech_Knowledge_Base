# 样例 #03 - 创意写作提示词

## 提示词内容
```
你是一位富有创意的写作导师。请帮我创作一个故事，要求如下：

**创作要求：**
- 类型：[故事类型，如悬疑、科幻、爱情等]
- 长度：[字数要求]
- 主题：[核心主题]
- 风格：[写作风格，如幽默、严肃、诗意等]

**角色设定：**
- 主角：[主角基本信息]
- 配角：[重要配角]
- 反派：[如果有反派角色]

**情节要求：**
1. 开头要有吸引力
2. 中间要有冲突和转折
3. 结尾要有意义或启发
4. 对话要生动自然
5. 描写要生动具体

**输出格式：**
请按以下结构输出：
## 故事大纲
[简要大纲]

## 完整故事
[完整故事内容]

## 创作说明
- 主题表达：[如何表达主题]
- 写作技巧：[使用的技巧]
- 读者感受：[希望读者感受到什么]
```

## 使用场景
- 小说创作
- 剧本写作
- 广告文案
- 品牌故事
- 个人博客

## 效果展示
**输入示例：**
- 类型：悬疑推理
- 长度：2000字
- 主题：真相与正义
- 风格：紧张刺激

**输出效果：**
- 开头设置悬念，吸引读者
- 情节跌宕起伏，逻辑严密
- 结尾揭示真相，发人深省
- 人物对话生动，描写细腻

## 优点分析
1. **结构清晰**：包含大纲和完整内容
2. **要求具体**：明确了创作标准
3. **格式规范**：便于阅读和理解
4. **有说明**：解释了创作思路
