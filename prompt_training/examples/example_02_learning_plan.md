# 样例 #02 - 学习计划提示词

## 提示词内容
```
你是一位经验丰富的学习顾问。请帮我制定一个学习计划，具体要求如下：

**背景信息：**
- 学习目标：[具体目标]
- 当前水平：[现有基础]
- 可用时间：[每天/每周可用时间]
- 学习偏好：[喜欢的学习方式]

**计划要求：**
1. 制定3个月的学习路径
2. 每周安排具体学习任务
3. 包含阶段性检测点
4. 提供学习资源推荐
5. 考虑实际执行难度

**输出格式：**
请按以下结构输出：
## 学习目标
[目标描述]

## 学习路径
### 第1个月：基础阶段
- 周1-2：[具体任务]
- 周3-4：[具体任务]

### 第2个月：进阶阶段
- 周5-6：[具体任务]
- 周7-8：[具体任务]

### 第3个月：应用阶段
- 周9-10：[具体任务]
- 周11-12：[具体任务]

## 检测点
- 第1个月末：[检测内容]
- 第2个月末：[检测内容]
- 第3个月末：[检测内容]

## 推荐资源
- 书籍：[推荐书籍]
- 课程：[推荐课程]
- 实践：[实践建议]
```

## 使用场景
- 学习新技能
- 准备考试
- 职业发展
- 兴趣爱好培养

## 效果展示
**输入示例：**
- 学习目标：掌握Python数据分析
- 当前水平：编程零基础
- 可用时间：每天2小时
- 学习偏好：视频教程+实践

**输出效果：**
- 制定了详细的3个月学习路径
- 每周任务具体可执行
- 包含阶段性检测和资源推荐
- 考虑了零基础的学习曲线

## 优点分析
1. **个性化**：根据个人情况定制
2. **可执行**：任务具体明确
3. **有检测**：包含进度验证
4. **资源丰富**：提供学习资源
