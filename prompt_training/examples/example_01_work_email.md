# 样例 #01 - 工作邮件提示词

## 提示词内容
```
你是一位专业的商务沟通专家。请帮我写一封工作邮件，要求如下：

**基本信息：**
- 收件人：[具体职位和姓名]
- 主题：[邮件主题]
- 目的：[邮件目的]

**内容要求：**
- 开头：礼貌问候，说明写信原因
- 正文：分点说明，逻辑清晰
- 结尾：明确行动要求，礼貌结束
- 语气：正式但友好，专业但不生硬

**输出格式：**
请按以下格式输出：
主题：[邮件主题]
正文：
[邮件正文内容]
```

## 使用场景
- 向领导汇报工作
- 与同事协调项目
- 向客户发送商务邮件
- 申请假期或资源

## 效果展示
**输入示例：**
- 收件人：张经理
- 主题：项目进度汇报
- 目的：汇报本周项目进展，申请下周资源支持

**输出效果：**
- 结构清晰，包含问候、正文、结尾
- 语言专业但不过于正式
- 明确提出了资源申请需求
- 语气恰当，符合职场沟通规范

## 优点分析
1. **结构完整**：包含所有必要信息
2. **要求明确**：具体说明了内容要求
3. **格式规范**：提供了输出格式模板
4. **场景适用**：适合多种工作场景
