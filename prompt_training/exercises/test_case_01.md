# 提示词训练测试用例 #01

## 任务描述
请帮我写一个关于"如何提高工作效率"的提示词，要求：
1. 结构清晰，包含背景、任务、要求、输出格式
2. 能够引导AI给出实用的建议
3. 适合职场人士使用

## 评分标准
- **结构完整性** (25分): 是否包含所有必要部分
- **清晰度** (25分): 语言表达是否清晰易懂
- **实用性** (25分): 能否产生有用的建议
- **创新性** (25分): 是否有独特的思路或方法

## 示例提示词模板
```
背景：[描述使用场景]
任务：[具体要完成的任务]
要求：[详细的要求和约束]
输出格式：[期望的输出格式]
```

## 测试要求
请根据以上要求，写一个完整的提示词，并说明：
1. 这个提示词适用于什么场景？
2. 预期会得到什么样的回答？
3. 如何验证提示词的效果？
