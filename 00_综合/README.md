## 🎓 学习方法论

### 费曼学习法
```
1. 选择一个概念
2. 用简单的语言解释给别人听
3. 发现不理解的地方，重新学习
4. 简化和类比，直到能让小白听懂
```

### 刻意练习
```
1. 明确目标（我要掌握什么）
2. 拆解技能（分解成小的知识点）
3. 集中练习（高强度专注学习）
4. 持续反馈（验证是否掌握）
```

### 项目驱动学习
```
1. 找一个实际问题/项目
2. 尝试用新技术解决
3. 遇到问题查文档/源码
4. 总结经验，形成自己的理解
```
